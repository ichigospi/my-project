- 運命の恋（問題認知用：「運命の恋を叶えたい」「運命の恋が終わってしまう前に」）

### フェーズ3: 予告・販売（Day 7〜9 / 4/17〜4/19）

**目的:** オファーを出して売る。「待ちきれない状態」で受け取らせる。

**1日あたり3投稿 × 3日 = 9投稿**

| Day | 投稿内容 | 温度 |
|---|---|---|
| **Day 7 / 4/17(金)** | 正式告知 + 商品説明 + 理想の未来 | 2〜3 |
| **Day 8 / 4/18(土)** | 欲求喚起 + FAQ + 反論処理 + 「迷ってる人へ」 | 2〜3 |
| **Day 9 / 4/19(日)** | **販売日** カウントダウン + 販売開始 + リアルタイム報告 + 感謝 | 2〜3 |

**フェーズ3のテクニック:**
- スクリーニング: 「本気の人にだけ届けたい」「覚悟のある方のみ」→カリギュラ効果
- 簡単そうに見せる: 「申し込むだけ。あとは任せて」
- 理想の未来だけ語る: ロジックより「受けたらどうなるか」
- 限定感: 部数限定 + 予定数非公開で「いつ締まるかわからない」緊急性
- リアルタイム報告: 「今朝だけで○件！」でライブ感
- **完売→追加枠パターン（本音鑑定で実証済み）**: 完売宣言→「守護天使様に相談」→追加枠→再完売→キャンセル枠で最終回収
- **購入資格の付与（本音鑑定で実証済み）**: LINE内の診断で「選ばれた方のみ」に資格を付与→特別感で購買抵抗を下げる
- **権威を天使に帰属**: 「守護天使様に相談した結果、追加が可能に」など、販売判断を人間でなくスピリチュアルな存在に帰属させる
- **天体イベント連動**: 販売日を新月等に合わせ「この条件では二度と受けられない」を自然に演出

### なぜこの流れなのか

本音鑑定（2月）で即完売を達成した導線には、実証済みの心理構造がある。

```
無料ギフト → 行動要求 → 教育 → 診断 → 結果+資格付与 → 予告 → 販売
```

この「段階的コミットメント」の骨格は崩さない。ただし同じ人がLINEで見ているため、以下3点だけテーマを変える：

1. **無料ギフトの形式** — 祝福アート → 運命の天使カード（カードリーディング形式で個別感UP）
2. **診断の切り口** — 相性診断（%スコア）→ あの人との運命の絆リーディング（「糸の状態」で返信）
3. **再販の武器を活用** — 前回即完売の実績＋受けた方の成功事例を教育〜告知の全段階で使う

それ以外の心理構造（行動要求→選別、診断→資格付与、完売→追加枠）はそのまま踏襲。変える必要のないものは変えない。

```
SNS投稿で教育（14日間）
    ↓
コラム配布企画でLINE登録を促進
    ↓
LINE配信（4/11〜4/19 / 以下のシナリオ）
    ↓
4/19(日) 21:00 販売開始（LP経由）
    ↓
完売 → 追加枠 → キャンセル枠回収 → 感謝＋関係維持
```

**LINE登録のきっかけ:** コラム無料配布企画（「いいね＋フォローでコラムプレゼント✨」）→ LINE登録を促す。加えて、フェーズ3の告知投稿から直接LINE登録リンクへ誘導

### 設計判断の根拠まとめ

| 判断 | 根拠 |
|---|---|
| **心理構造（段階的コミットメント）はそのまま** | 本音鑑定で即完売の実績あり。壊す理由がない |
