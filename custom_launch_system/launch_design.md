## 成果物2: 刷り込みキーワード（全期間で繰り返す）

2週間で何十回も自然に使い、フォロワーの脳に定着させる。

### メインキーワード5つ

| キーワード | 役割 | 使い方 |
|---|---|---|
| 運命の人 | 最重要。この概念自体が商品の存在理由 | 「あなたの運命の人は…」「運命の人と結ばれるために」など、恋愛の究極のゴールとして使う |
| 恋の試練 | 商品コンセプトの一言化。欲求喚起ワード | 「今の辛さは"恋の試練"かもしれません」「試練を乗り越えた先に…」など、現状の苦しみに名前をつける |
| 守護天使様 | 独自メソッドの刷り込み | 「守護天使様を通して視ると…」「天使様が教えてくれたのは…」など、アリサ独自の鑑定手法として |
| サイレント期間 | 問題認知を促すネガティブワード | 「音信不通＝サイレント期間かも」「サイレント期間の正体は…」など、不安な現状に意味を与える |
| 溺愛 | 世界観に合った温かい表現 | 「運命の相手から溺愛される未来」「溺愛されすぎて困るかも…」など、理想の未来を描く |

### サブキーワード

- 運命の恋（問題認知用：「運命の恋を叶えたい」「運命の恋が終わってしまう前に」）
- 最幸の未来（理想の未来用：「最幸の未来へ一歩踏み出しませんか？」）
- 霊視タロット（権威性＋独自性：「守護天使様とのチャネリングによる霊視タロット」）

### 使用ルール

- 1投稿にメインキーワードを最低1つ、自然に含める
- 同じキーワードを3投稿連続で使わない（ローテーション）
- サブキーワードは味付けとして添える
- 「売り感」が出ないよう、教育・ストーリーの文脈に溶け込ませる

### フェーズ3: 予告・販売（Day 11〜14）

**目的:** オファーを出して売る。「待ちきれない状態」で受け取らせる。

**1日あたり3投稿 × 4日 = 12投稿**

| Day | 投稿内容 | 温度 |
|---|---|---|
| **Day 11** | 正式告知 + 商品説明 + 理想の未来 + 短文テイザー | 2〜3 |
| **Day 12** | 欲求喚起 + FAQ + 反論処理 + 「迷ってる人へ」 | 2〜3 |
| **Day 13** | **販売開始**「LINE登録で申込」+ リアルタイム報告 + 購入者の声 | 2〜3 |
| **Day 14** | 残り枠告知 + 最後の欲求喚起 + 締め切り + 感謝投稿 | 1〜3 |

**フェーズ3のテクニック:**
- スクリーニング: 「本気の人にだけ届けたい」→カリギュラ効果
- 簡単そうに見せる: 「申し込むだけ。あとは任せて」
- 理想の未来だけ語る: ロジックより「受けたらどうなるか」
- 限定感: 「○名まで」で緊急性
- リアルタイム報告: 「今朝だけで○件！」でライブ感

## ファネル設計

```
SNS投稿で教育（14日間）
    ↓
コラム配布企画でLINE登録を促進
    ↓
LINE内でコラム3本＋教育メッセージ配信
    ↓
セールスレター配信（Day 11-12）
    ↓
購入（LINE経由）→ Day 14締め切り
```

**LINE登録のきっかけ:** コラム無料配布企画（「いいね＋フォローでコラムプレゼント✨」）→ LINE登録を促す。加えて、フェーズ3の告知投稿から直接LINE登録リンクへ誘導

## 口調チューニングルール

| 一般的なマーケ表現 | あなたの口調への変換 |

## 投稿生成時のチェックリスト

- [ ] 課題解決ロジックの7段階のどこに位置する投稿か明確か
- [ ] 刷り込みキーワードが自然に含まれているか（メイン1つ以上）
- [ ] 該当フェーズの目的に沿っているか
- [ ] 口調・温度感が character_profile.json と一致しているか（口癖1つ以上使用）
- [ ] 温度レベルが適切か（3投稿連続同じ温度にしない）
- [ ] 短文投稿が5投稿に1回は入っているか
- [ ] 書き出し・締め方にバリエーションがあるか
- [ ] 仮想敵の批判が自分の温かさの範囲内か（攻撃的になっていないか）
- [ ] セールス感が出すぎていないか（教育・ストーリーに溶け込んでいるか）
- [ ] Phase 1-2で商品を直接売っていないか
- [ ] コラムの連鎖構造（末尾フック）が機能しているか

## 各Day別 投稿ガイドライン

### Day 1: 概念の種まき
- 豆知識で興味を引く（教育×エンタメ）
- メインキーワードを初めて出す
- 日常投稿も混ぜてナチュラルに

### Day 2: 問題の提示
- 「頑張ってるのに変わらない人」の問題提起
- 自分の原体験ストーリー
- 短文で問いかけ

### Day 3: 知識の深掘り
- 教育を具体例で説明
- ドライテスト（いいねして）で反応測定

### Day 4: コラム①配布
- **コラム①企画投稿**（いいね＋フォローで配布）
- 属人性ストーリー
- コラム補強の問題提起

### Day 5: コラム①の余韻
- 反響に触れる教育投稿
- 属人性ストーリー
- 感想を聞くCTA

### Day 6: 仮想敵の登場
- 間違った方法を否定
- 再現性（結果が出た話）
- 属人性ストーリー（失敗談）

### Day 7: 社会的証明の強化
- 別の実績エピソード
- 差別化（なぜ私の方法か）
- 短文でKW刷り込み

### Day 8: コラム②配布
- **コラム②企画投稿**
- 仮想敵批判（補強）
- 感想を聞くCTA

### Day 9: 差別化を深める
- 他との具体的な違い
- 数字で語る再現性
- テイザー「すごいこと思いついた」

### Day 10: コラム③配布
- **コラム③企画投稿**
- 属人性ストーリー（覚悟）
- 短文で期待感

### Day 11: 正式告知
- メイン告知投稿（商品説明 + 限定感）
- 理想の未来投稿
- 短文テイザー

### Day 12: 欲求喚起＋反論処理
- 「受けたらどうなるか」を描写
- FAQ・反論処理
- 「迷ってる人へ」の背中押し

### Day 13: 販売開始
- 販売開始告知
- リアルタイム報告「○件申込！」
- 反論処理（最後の迷い）

### Day 14: クロージング＋感謝
- 残り枠の最終告知
- 最後の欲求喚起
- 締め切り告知
- 感謝投稿「ここまで読んでくれたあなたへ」
