### フェーズ3: 予告・販売（Day 11〜14）

**目的:** オファーを出して売る。「待ちきれない状態」で受け取らせる。

**1日あたり3投稿 × 4日 = 12投稿**

| Day | 投稿内容 | 温度 |
|---|---|---|
| **Day 11** | 正式告知 + 商品説明 + 理想の未来 + 短文テイザー | 2〜3 |
| **Day 12** | 欲求喚起 + FAQ + 反論処理 + 「迷ってる人へ」 | 2〜3 |
| **Day 13** | **販売開始**「LINE登録で申込」+ リアルタイム報告 + 購入者の声 | 2〜3 |
| **Day 14** | 残り枠告知 + 最後の欲求喚起 + 締め切り + 感謝投稿 | 1〜3 |

**フェーズ3のテクニック:**
- スクリーニング: 「本気の人にだけ届けたい」→カリギュラ効果
- 簡単そうに見せる: 「申し込むだけ。あとは任せて」
- 理想の未来だけ語る: ロジックより「受けたらどうなるか」
- 限定感: 「○名まで」で緊急性
- リアルタイム報告: 「今朝だけで○件！」でライブ感

## 投稿生成時のチェックリスト

- [ ] 課題解決ロジックの7段階のどこに位置する投稿か明確か
- [ ] 刷り込みキーワードが自然に含まれているか（メイン1つ以上）

- [ ] 口調・温度感が character_profile.json と一致しているか（口癖1つ以上使用）
- [ ] 温度レベルが適切か（3投稿連続同じ温度にしない）

- [ ] 書き出し・締め方にバリエーションがあるか
- [ ] 仮想敵の批判が自分の温かさの範囲内か（攻撃的になっていないか）

- [ ] Phase 1-2で商品を直接売っていないか
- [ ] コラムの連鎖構造（末尾フック）が機能しているか
