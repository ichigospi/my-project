# LINE配信メッセージ 実例集（詳細版・テクニック解説付き）

複数のローンチパターンのLINE配信ログを匿名化・コンプライアンス修正の上で収録。パターンA（物販型・OC＋個別LINE 2チャネル運用）とパターンB（鑑定サービス型・OC単独運用）を収録。チャネル運用・配信タイミング・テクニックの参照用。

---

## ⚠ コンプライアンスガイド（生成時に必ず守ること）

### 景品表示法
- 「お金の問題が突然消える」「人生が好転する」等の断定表現を使わない
- 「たった○週間で全てを体験した」等の極端な成功事例を一般的効果として書かない
- 通常価格との比較は、実際に販売実績のある価格のみ使用する
- 「売り切れ」「残りわずか」は事実に基づくこと

### 薬機法
- 精神的・身体的な改善を断定しない
- 石・場所に治療・治癒効果があるような表現をしない

### 一般的な表現ルール
- 「波動」「ヒーリング」等は独自の用語であることを注記する
- 「世界に1つの魔法の石」等の誇大表現は避ける
- 他社の商品・サービスを否定・批判しない
- 成功事例は「きっかけ」であり、本人の努力・行動変容が主要因であることを示す

# パターンA: 物販型（OC＋個別LINE 2チャネル運用）

## 2チャネル構成の概要

| | チャネルA: オープンチャット | チャネルB: 個別LINE |
|---|---|---|
| **特徴** | 全員が見える公開空間 | 1対1のプライベート空間 |
| **役割** | 盛り上がりの可視化・社会的証明 | コラム/レター配布・直接的な行動促進 |
| **トーン** | コミュニティ運営者として | 個人的な関係性として |
| **配信頻度** | 高い（1日に複数回） | 要所のみ |

## 全体の配信構造マップ

```
【Day 1】 ティザー → コラム①配布 → 反響シェア
【Day 2】 コラム②配布 → 反響シェア → 動画予告
【Day 5】 メッセージ紹介＋動画予告
【Day 6】 動画公開＋無料鑑定
【Day 6-8】 無料鑑定リマインド
【Day 8】 無料鑑定締め切り
【Day 9】 緊急予告 → 販売告知
【Day 10】 販売カウントダウン → 販売開始 → リマインド → 完売報告
【Day 13】 再販告知 → 再販開始 → 完売 → 感謝
```

## 配信テクニック

### タイミング戦略
- 3時間前ティザー: 配信時刻を予告し待機させる
- 1時間前リマインド: 忘れていた人を呼び戻す
- 10分前カウントダウン: 緊急性を最大化
- 50分後リマインド: 販売直後の勢いを加速

### 社会的証明の使い方
- 具体的な数字（「300名突破」等）のリアルタイム共有
- 読者の声の引用
- 「返信が追いついていない」で間接的に大量申込を示す
- 完売報告で次回への最大の社会的証明

### オープンループの連鎖
1. 「夜21時にお届けします」→ コラム配布
2. 「続きを書かないとダメですね」→ 次のコラム配布
3. 「動画の公開、受け取りたい方はリアクションを」→ 動画公開
4. 「各方面と調整しています」→ 販売告知
5. 「明日○時にご案内します」→ 販売開始
6. 「間に合わなかった方、覚えています」→ 再販告知

### 再販の設計パターン
1. 完売→声の蓄積（3日間）→直前告知（3時間前）→30分限定再販
2. 初回より短い販売時間で緊急性UP
3. 「業者との交渉ストーリー」で再販の正当化
4. 「間に合わなかった方の声」で需要の証明

# パターンB: 鑑定サービス型（OC単独運用）

## 全体の配信構造マップ

```
【Day 1】 ティザー
【Day 2】 コラム①予告 → 配布 → 反響シェア
【Day 3】 反響シェア＋コラム②執筆中
【Day 4】 コラム②完成報告 → 配布
【Day 5】 フォロー＋コラム③予告
【Day 6】 リマインド → コラム③配布
【Day 7】 動画完成報告 → 動画公開＋無料鑑定募集
【Day 8】 無料鑑定進捗 → エピソードシェア
【Day 9】 無料鑑定締め切り
【Day 10】 有料鑑定への声紹介 → 緊急予告 → 音声＋販売告知
【Day 11】 カウントダウン → 販売開始 → リマインド
【Day 12】 朝リマインド → 残枠告知 → ラスト → 完売報告
```

## パターンB特有のテクニック

| テクニック | 説明 |
|---|---|
| 執筆リアルタイム中継 | 「今書いてます」「間に合いました！」でライブ感 |
| キーワード送信CTA | コラム末尾のキーワードを公式LINEに送らせてリスト化 |
| リアクション依頼 | 「チャットを長押ししてリアクション」で参加意識 |
| コラムURL再掲 | 複数日にわたって未読者向けにURL再掲 |
| 音声メッセージ活用 | テキストだけでなく音声ファイルで親密さ |
| PDFメッセージ | 「大切なメッセージ」をPDFで特別感演出 |
| プラン人気情報 | FAQ形式で上位プランの人気を伝え誘導 |

---

## パターンA vs B 比較

| 項目 | パターンA（物販型） | パターンB（鑑定サービス型） |
|---|---|---|
| **チャネル** | OC＋個別LINE 2チャネル | OC単独 |
| **期間** | 13日間 | 12日間 |
| **コラム配布** | 個別LINEで配布、OCで告知 | OCでURL直接配布 |
| **無料→有料の導線** | 無料鑑定→動画→物販 | 無料鑑定→音声→鑑定サービス |
| **再販** | あり（30分限定） | なし（1日で完売） |
| **価格帯** | 単一価格 | 松竹梅3プラン |
| **トーン** | 権威型 | 親しみ型 |
| **コラム教育期間** | Day 1-2（2日間集中） | Day 2-6（5日間じっくり） |

## ⚠ 生成時のコンプライアンスチェックリスト

- [ ] 「お金の問題が消える」「人生が好転する」等の断定表現がないか
- [ ] 体験談の近くに「個人の感想です」の免責表示があるか
- [ ] 「波動」「ヒーリング」等の独自用語に定義注釈があるか
- [ ] 他社の商品・業界を攻撃的に批判していないか
- [ ] 「売り切れ」「残りわずか」が事実に基づいているか
- [ ] 宝くじ当選・臨時収入等の極端な成功事例がないか
- [ ] 精神的・身体的な改善を断定していないか（薬機法）
- [ ] 通常価格は実際の販売実績に基づいているか
- [ ] 「世界に1つの魔法の石」等の誇大表現がないか
- [ ] 成功事例は「きっかけ」であり本人の努力が主要因であることが示されているか
- [ ] 「お金がないなら買うべき」等の不安を煽る論法がないか
