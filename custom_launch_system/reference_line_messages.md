# LINE配信メッセージ 実例集（詳細版・テクニック解説付き）

複数のローンチパターンのLINE配信ログを匿名化・コンプライアンス修正の上で収録。パターンA（物販型・OC＋個別LINE 2チャネル運用）とパターンB（鑑定サービス型・OC単独運用）を収録。

---

## ⚠ コンプライアンスガイド（生成時に必ず守ること）

### 景品表示法
- 「お金の問題が突然消える」「人生が好転する」等の断定表現を使わない
- 極端な成功事例を一般的効果として書かない
- 通常価格との比較は販売実績のある価格のみ使用
- 「売り切れ」「残りわずか」は事実に基づくこと

### 薬機法
- 精神的・身体的な改善を断定しない
- 石・場所に治療・治癒効果があるような表現をしない

### 一般的な表現ルール
- 「波動」「ヒーリング」等は独自の用語であることを注記する
- 他社の商品・サービスを否定・批判しない
- 成功事例は「きっかけ」であり、本人の努力・行動変容が主要因であることを示す

| | チャネルA: オープンチャット | チャネルB: 個別LINE |
|---|---|---|
| **特徴** | 全員が見える公開空間 | 1対1のプライベート空間 |
| **役割** | 盛り上がりの可視化・社会的証明 | コラム/レター配布・行動促進 |
| **トーン** | コミュニティ運営者として | 個人的な関係性として |
| **配信頻度** | 高い（1日に複数回） | 要所のみ |

# 配信テクニック まとめ

## タイミング戦略

| テクニック | 説明 |
|---|---|
| **3時間前ティザー** | 具体的な配信時刻を予告し待機させる |
| **1時間前リマインド** | 忘れていた人を呼び戻す |
| **10分前カウントダウン** | 緊急性を最大化 |
| **50分後リマインド** | 販売直後の勢いを加速 |
| **15分前ラストコール** | 締め切り直前の最後の後押し |

## 社会的証明の使い方

- **具体的な数字**: 「300名突破」「数百名が反応」等のリアルタイム数値
- **読者の声の引用**: メッセージの一部を引用
- **返信が追いつかない**: 間接的に大量の申込があることを示す
- **完売報告**: 次回販売への最大の社会的証明

## オープンループの連鎖

| オープン | クローズ |
|---|---|
| 「夜21時にお届け」 | → コラム①配布 |
| 「続きを書かないとダメ」 | → コラム②配布 |
| 「特別動画を受け取りたい方はリアクションを」 | → 動画公開 |
| 「各方面と調整」 | → 販売告知 |
| 「明日[X]時にご案内」 | → 販売開始 |
| 「覚えています」 | → 再販告知 |

## 再販の設計パターン

1. 完売 → 声の蓄積（3日間）→ 直前告知（3時間前）→ 30分限定再販
2. 初回より短い販売時間で緊急性UP
3. 「業者との交渉ストーリー」で再販の正当化
4. 「間に合わなかった方の声」で需要の証明

---

## チャネル使い分けの原則（パターンA）

1. **OC = 盛り上がりの舞台**: リアクション・数値・教育で「みんなが注目している」を可視化
2. **個別LINE = 確実な配布**: コラム・レター・動画のURLを直接届ける＋見逃し防止
3. **個別LINE → OC誘導**: ティザーやコラム配布のP.S.でOCへ誘導
4. **両チャネル同時配信**: 販売開始・完売報告・再販等は両方で同時に送る
5. **個別LINE特有の工夫**: 3連投（通知複数回）、動画直接添付、コラムURL再掲

- [ ] 「お金の問題が消える」「人生が好転する」等の断定表現がないか
- [ ] 体験談の近くに「個人の感想です」の免責表示があるか
- [ ] 「波動」「ヒーリング」等の独自用語に定義注釈があるか

- [ ] 「売り切れ」「残りわずか」が事実に基づいているか

- [ ] 精神的・身体的な改善を断定していないか（薬機法）
- [ ] 通常価格は実際の販売実績に基づいているか

- [ ] 成功事例が「きっかけ」であり本人の努力が主要因であることが示されているか
- [ ] 「お金がないなら買うべき」等の不安を煽る論法がないか
