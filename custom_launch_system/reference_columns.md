# ローンチコラム 実例集（詳細版）

2つの異なるローンチパターンの実例。匿名化済み。構造・テクニック・ファネル設計を学ぶための参照資料。

**パターンA**: 段階的フック型（物販系 — 3段階のオープンループでLINE→動画→無料鑑定へ誘導）
**パターンB**: 3本連鎖コラム型（サービス系 — 教育→信頼→欲求喚起の王道3本構造）

---

## ⚠ コンプライアンス・ガイドライン（コンテンツ生成時に必ず遵守）

この実例集のコンテンツを参考にローンチコンテンツを生成する際、以下のルールを必ず守ること。

### 景品表示法（有利誤認・優良誤認の防止）
- **個人の体験談には必ず免責表示を付記する**：「※個人の感想です。効果には個人差があります」「※特定の結果を保証するものではありません」
- **極端な成功事例（宝くじ当選・臨時ボーナス等）を一般的な効果のように書かない**：レアケースは有利誤認表示に該当する可能性が極めて高い
- **「全員」「必ず」「確実に」等の断定表現を避ける**：「多くの方が」「～という可能性があります」「～と感じる方がいらっしゃいます」に留める

### 薬機法（精神的・身体的効果の表現）
- **「不安が消える」「体が軽くなる」「疲れが取れる」等の精神的・身体的改善を断定しない**
- **感情面の変化も「個人の感想」として扱い、免責表示を必ず付記する**

### 他社比較（不当な比較広告の防止）
- **「他社の石はダメ」「市販の石は空っぽ」等の他社否定をしない**：「私たちは一人ひとりに寄り添うことを大切にしています」等、自社の姿勢の強調に留める
- **失敗事例と成功事例の不当な対比を避ける**：比較するなら公正な条件で
- **「波動」「ヒーリング」等の独自用語には定義注釈を入れる**：「※ここでの『波動』とは、当方独自の考え方で〇〇を指します。科学的に実証された用語ではありません」

### 表現の基本方針
- **「石＝幸せになる」ではなく「石＝きっかけ、本人の努力が前提」**
- **断定（～はず、～です）→ 意思表明（～と考えています、～という想いで制作しています）**
- **「確実性」を排除し「可能性の追求」として表現する**

# パターンA: 段階的フック型ローンチ

**商材ジャンル:** スピリチュアル物販（波動鑑定＋オーダーメイド制作）
**ファネル構造:**
1. コラム①（テイザー） → LINE登録CTA
2. コラム②（真相公開＋暴露予告） → 動画視聴CTA
3. 動画台本（フル教育＋証明＋恐怖＋オファー） → 無料鑑定CTA

**このパターンの特徴:**
- 「あること」の正体を段階的に明かす**オープンループ構造**
- 読者の反応（ボタン押下数）をリアルタイムで次コンテンツに反映する**ライブ感**
- 業界の闇を暴露する**「敵を作る覚悟」ポジション**
- テキスト→動画の切り替えで**メディアの信頼度を段階的に上げる**
- 最終CTAが「無料鑑定」で**ハードルを極限まで下げる**

# 構造パターン比較まとめ

| | パターンA（段階的フック型） | パターンB（3本連鎖コラム型） |
|---|---|---|
| **商材** | 物販（オーダーメイド制作物） | サービス（個別鑑定） |
| **コンテンツ数** | テキスト2本＋動画1本 | テキスト3本 |
| **メディア切替** | テキスト→動画 | テキストのみ |
| **オープンループ** | 「あること」の正体を2段階で明かす | 各コラム末尾のフックで次回へ繋ぐ |
| **CTA構造** | LINE登録→動画視聴→無料鑑定（3段階） | LINE登録→LINE登録→告知予告（3段階） |
| **社会的証明** | 体験談2名 | 4名の詳細エピソード＋120件の数字 |
| **最終オファー** | 無料鑑定（ハードル最低） | 有料鑑定の案内予告（期待値最大） |

## 共通する重要テクニック

1. **オープンループ**: 正体を伏せて次のコンテンツへ引っ張る
2. **二分割**: 「知っている人 vs まだ知らない人」で興味を引く
3. **自社の姿勢強調**: 他社を否定するのではなく、自分たちのこだわりを伝える
4. **スクリーニング**: 「本気の方だけ」でカリギュラ効果
5. **比喩の活用**: 「バケツの穴」「血液型の輸血」等で概念を可視化
6. **リアルタイム感**: 前回の反響数を次のコンテンツ冒頭で使う
7. **離脱防止**: 「ここで読むのをやめないで」「あと3分だけ」
8. **きっかけの提示**: 「石＝幸せ」ではなく「石＝きっかけ、あなたの努力が主役」
9. **属人性ストーリー**: 発信者自身の原体験で信頼を構築
10. **段階的限定性**: 期間限定→人数限定→ボタンを押した人限定の多重レイヤー
