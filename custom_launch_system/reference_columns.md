## ⚠ コンプライアンス・ガイドライン（コンテンツ生成時に必ず遵守）

この実例集のコンテンツを参考にローンチコンテンツを生成する際、以下のルールを必ず守ること。

### 景品表示法（有利誤認・優良誤認の防止）
- **個人の体験談には必ず免責表示を付記する**：「※個人の感想です。効果には個人差があります」「※特定の結果を保証するものではありません」
- **極端な成功事例（宝くじ当選・臨時ボーナス等）を一般的な効果のように書かない**：レアケースは有利誤認表示に該当する可能性が極めて高い
- **「全員」「必ず」「確実に」等の断定表現を避ける**：「多くの方が」「～という可能性があります」「～と感じる方がいらっしゃいます」に留める

### 薬機法（精神的・身体的効果の表現）
- **「不安が消える」「体が軽くなる」「疲れが取れる」等の精神的・身体的改善を断定しない**
- **感情面の変化も「個人の感想」として扱い、免責表示を必ず付記する**

### 他社比較（不当な比較広告の防止）
- **「他社の石はダメ」「市販の石は空っぽ」等の他社否定をしない**
- **失敗事例と成功事例の不当な対比を避ける**
- **「波動」「ヒーリング」等の独自用語には定義注釈を入れる**

### 表現の基本方針
- **「石＝幸せになる」ではなく「石＝きっかけ、本人の努力が前提」**
- **断定（～はず、～です）→ 意思表明（～と考えています、～という想いで制作しています）**
- **「確実性」を排除し「可能性の追求」として表現する**

# パターンA: 段階的フック型ローンチ

**商材ジャンル:** スピリチュアル物販（波動鑑定＋オーダーメイド制作）
**ファネル構造:**
1. コラム①（テイザー） → LINE登録CTA
2. コラム②（真相公開＋暴露予告） → 動画視聴CTA
3. 動画台本（フル教育＋証明＋恐怖＋オファー） → 無料鑑定CTA

**このパターンの特徴:**
- 「あること」の正体を段階的に明かすオープンループ構造
- 読者の反応をリアルタイムで次コンテンツに反映するライブ感
- 業界の闘を暴露する「敵を作る覚悟」ポジション
- テキスト→動画の切り替えでメディアの信頼度を段階的に上げる
- 最終CTAが「無料鑑定」でハードルを極限まで下げる

## ⚠ 生成時のコンプライアンスチェックリスト

- [ ] 個人の体験談・メッセージ引用の近くに「個人の感想です」「結果を保証するものではありません」の免責表示があるか
- [ ] 宝くじ当選・臨時収入等の極端な成功事例を一般的な効果のように書いていないか
- [ ] 「全員」「必ず」「確実に」等の断定表現がないか
- [ ] 精神的・身体的な改善を断定していないか（薬機法）
- [ ] 他社の商品・サービスを否定・批判していないか
- [ ] 「波動」「ヒーリング」等の独自用語に定義注釈があるか
- [ ] 「石＝幸せ」ではなく「石＝きっかけ、本人の努力が前提」の構図になっているか
- [ ] 断定（～です、～はず）ではなく意思表明（～と考えています、～という想いで）になっているか
- [ ] 参拝と生活変化の間に直接的な因果関係を断定していないか
- [ ] 「鬱」「体が軽くなった」等の医療的表現を使っていないか
- [ ] ランキングやSNS等を「ダメ」「間違い」と否定せず、穏やかに自社の強みを伝えているか
