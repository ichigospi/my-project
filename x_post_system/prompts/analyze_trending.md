# 伸びてるポスト分析プロンプト

> アプリ `/x-post/analysis` および手動分析時に使うプロンプト。
> ノウハウ取り込み完了後に最終調整する。

## 入力

- 競合ポスト群（本文 + 指標）
- ジャンル（business / spiritual）

## 出力

各ポストについて以下を抽出:

1. **構造タイプ**（フック型 / リスト型 / ストーリー型 / 質問型 / 対比型 等）
2. **フック分析**（冒頭1-2行で何をしているか）
3. **展開の型**（根拠提示 / 体験談 / 数字提示 / 比喩 等）
4. **締めの型**（行動喚起 / 余韻 / 質問返し 等）
5. **使われたパワーワード**
6. **伸びた仮説**（なぜこの数字が出たかの推定）
7. **自アカ転用ヒント**（該当ジャンルの自アカで使うとしたら）

## 参照ファイル

- `../knowledge_common/structure_patterns.md`
- ジャンルに応じて `../knowledge_business/` または `../knowledge_spiritual/`

> 詳細は取り込み完了後に追記。
