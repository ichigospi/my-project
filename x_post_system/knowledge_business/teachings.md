#### ビジ系特有のフック例

ビジ系で実際にハマるフックの具体例。生成時のサンプルとして使う。

- 「日給2万円いったんやが…」
- 「月2000万稼ぐ友達に聞いたんだけど…」
- 「20垢運用してわかった」
- 「メルカリで1億売り上げてる主婦から聞いた」
- 「noteで月2000万円売った友達に教えてもらった」
- 「万垢30個分析してわかったんだけど」
- 「23歳のフリーターの弟に教えたら速攻で83万円稼いじゃった」
- 「今裏で仕込んでるジャンルだけど」
- 「ワンチャン月500万円くらいいけそうw こっそりやっちゃうよーん！」
- 「俺が〇〇円稼いだ」
- 「昨日仕入れに行って爆利益品みっけたんだけど」

#### ビジ系の実例（要素分解付き）

##### 例1: ぶんじろう垢のTips販売継続報告
```
本文:
  この垢でTips販売開始して3ヶ月目とかやけど地味に売れ続けるコツみたいなんが
  若干掴めつつある。商品数がゴミやから売上しょぼいけどこの容量でガンガン
  増やしていくか。ぶんじろう垢と違って一撃の爆発力はないけど属人生を
  限りなく落としてるから正味誰でも運用できるからかなり楽。
  （売上画像）

3要素:
  - フック: 売上画像 / 商品増やしたら儲かる（雰囲気）
  - 興味対象: 売れ続けるコツ / 属人性のない運用方法
  - 強化: 属人性落としてる / 誰でも運用できる / かなり楽
```

##### 例2: にきち垢の即日現金化せどり
```
本文:
  今日友達にとある即日現金化できる㊙せどり手法教えたんだけど、
  速攻で利益29047円ゲットしてて草
  内訳: 仕入れ61,000 / 売上90,047 / 利益29,047
  もうこの手法知っちゃったら他の手法なんてめんどくさすぎてやってられないね

3要素:
  - フック: 友達が速攻で利益2.9万円ゲット（他人の行動＋成果）
  - 興味対象: 即日現金化できる㊙せどり手法
  - 強化: 速攻（即効性）/ ㊙（秘密性）/ めんどくさすぎてやってられない（優越感・手軽さ）
```

##### 例3: 新商材2日目ポスト（作成例）
```
本文:
  新商材を仕入れ始めて2日目でこれ。仕入れ値やすいからダメージ少ないし
  回転率もいい。こういうのは情報持ってたら誰でもできるから、
  やっぱ情報ゲーやね。（実績画像）

3要素:
  - フック: 新商材を仕入れ始めて2日目でこれ
  - 興味対象: 仕入れ商材の情報
  - 強化: 仕入れ値安い / 回転率いい / 誰でもできる
```

#### ビジ垢の階段モデル（X運用月100万まで）

ビジ垢で「次の段に進ませる」ためのターゲット階段。例: X運用で月100万円稼ぐ方法を発信している場合。

```
4段目: 月50万円くらい稼げてるけど不安定
  ↑
3段目: 月10〜20万円くらいをコンテンツ販売で稼げている
  ↑
2段目: XでTipsアフィや安いnoteで5万円くらい稼げたけど停滞
  ↑
1段目: Xを動かしてるけど伸びてないし稼げてない  ← ⭕ ターゲット
  ↑
0段目: 情報発信やSNS運用に興味があるけど特にやったことはない人 ← ⭕ ターゲット
```

#### 階段ごとのポスト方針

##### 0段目向け
- 「Xアツいじゃん！」「Xってそんな稼げんの？？」と思わせる
- X運用をスタートしたくなる興味付け
- 例: 「俺がXで月100万稼いだ方法、知識ゼロでも余裕でいけたわ」

##### 1段目向け
- 「こうやったら伸びるよ」「X伸ばすとこんな良いことあるよ」
- X運用の伸ばすノウハウ・稼ぐノウハウへの興味付け
- 例: 「X運用で停滞してるやつに共通する致命傷ミス3つ」

#### NGポスト実例（教材3より）

##### NG例: ローンチ稼げる人向けポスト

```
売り上げる方法がローンチしかない人は今後どんどんキツくなる。
実際コンテンツ作ってローンチって流れに市場も飽き始めてるしな。
逆に今稼ぎまくってるのは裏でこっそり売ってる奴ら。
フロントは継続的な集客に特化して、後ろの仕組みでガッツリ売るパターン。
2ヶ月目で月3ケタ万超えてる別垢もこれ。
```

**ボツ理由**:
- ターゲットが狭すぎる
- そもそもローンチできる人は100人いたら1〜2人くらいしかいない
- フォロワーの90%は0〜1段目にいるので、コアすぎる知識は「興味の対象外」
- 「何発信していいかわからない」「アカウントをまだ持ってない」のレベル感が0〜1段目

→ 教訓: **コアな中上級者向けの考察は書かない**。完全ゼロイチの初心者に向けてわかりやすさ重視。

#### 冒頭一文目の良い例（教材3より）

##### 例A: AI系
```
AIが【知らなきゃヤバい話】今すぐ読め。絶対。
```
- 「AIが」 → トレンドテーマ
- 「知らなきゃヤバい話」 → 損失回避
- 「今すぐ読め。絶対。」 → 強調

##### 例B: X運用ビジネス系
```
Xで売上でないから萎えた。マジでビジネス理解してなさすぎ。そりゃ稼げない。
```
- 「Xで売上でないから萎えた」 → 1段目のターゲット狙い
- 「マジでビジネス理解してなさすぎ。そりゃ稼げない。」 → 損失回避

#### ビジ垢の市場トレンドテーマ（観察対象）

論争・話題で触れるとインプが伸びるテーマ（教材3が出てきたタイミングの市場感）:
- 奢る・奢られ論争
- ローンチ vs 継続販売論争
- AIで稼ぐ vs 王道で稼ぐ論争

→ 自分の界隈で論争が起きていないか日々観察する。触れる時は **中立ではなく見解を明確に** 出す。

#### ビジ系の6つの教育の具体例

##### ① 目的の教育（ビジ系）

- 「複数のマネタイズポイントがある」
- 「月収がかなり安定する」
- 「月商1000万円超えれた」
- 「旅をしながら自由に稼いでます」（広告系の定番）
- 「カフェで好きな場所で仕事してます」
- 「鮨食ってたら10万円稼げちゃってて黒字！」（リアルタイム性）

##### ② 信用の教育（ビジ系）

定番ストーリーパターン（田中noteの構造）:
```
タイトル: 工場勤務14年、SNS未経験へっぽこ男が、
         X開始2ヶ月で月870万円稼げちゃった裏側がやばすぎたww

[流れ]
スマホ1台1日20分で月100万円稼げるなんて怪しい…
  ↓
【田中のへぼさを知る】（過去のダメダメさ）
  ↓
【田中が成功したストーリー】（紆余曲折）
  ↓
【成功までの具体的なノウハウ】（価値提供）
  ↓
こんなへぼいやつがいけたなら俺でもいけそうな気がする！
信頼できそうな人やな！
```

ポイント:
- 顔出しも信用の教育になる（信憑性を高める）
- 価値提供と同時に信用構築をする

##### ③ 問題点の教育（ビジ系）

- 「『ポスト』ばっか頑張ってもXは伸びません」
- 「初心者が『低単価』で戦うと詰みます」
- 「物販で稼げない人は『利益率低すぎ』w」
- 「コンテンツ販売って商品作成やらレター作成やらLINE構築、教育記事やらやることが多すぎるから稼げないんですよ」

仮想敵の例:
- 「noteで稼ごう」みたいな流れ
- X運用下手な人がXを教えてる現状
- 利益率の低いビジネス推し

##### ④ 手段の教育（ビジ系）

問題点との一貫性例（アパレル物販を売る場合）:
- 問題点: 「初心者は仕入れで金がかかる商材を選ぶな」「利益率が低い商材はクソ」
- 手段: 「資金10万円以下ならアパレル一択」「仕入れ値の数倍で売れるから利益率最強」「服好きな人は超イージー」「僕が学生から稼ぐならアパレルから始めるね」

##### ⑤ 行動の教育（ビジ系）

- 「フットワーク重いやつは稼げない」
- 「チャンスは即動くべき」
- 「今が〇〇はチャンスです！」
- 「これ逃すと〇〇万円の損失やね」

##### ⑥ 投資の教育（ビジ系）

- 「自分も〇〇万円のコンサル受けて結果出しました」
- 「金がないやつほど投資しろ」
- 「良い情報は無料で出回らない」
- 「まずは無料でとか言ってるやつは稼げない」

#### noteで稼ぐ流派否定の問題点教育例

```
ぶっちゃけnoteとかって1記事売れても数千円にしかなりません。
初心者に謎に人気ですが、商品作成が大変なわりにへぼい集客力で
低単価売ったところでほぼインパクトがないので問題点教育で使った。
↓ 解決策誘導
高単価アフィなら…
```

→ 構造: ❶ 流派の問題点指摘 ❷ 仮想敵化 ❸ 自分の推す手段（高単価アフィ）に誘導

#### X運用商材を売る際の想定グダ要素

- 実績がないと稼げない
- 商品作るの大変そう
- 高額商品が作れないから無理
- 大手にゴマスリするの嫌だ
- 毎日ポストが思いつかない
- 後発だと稼げるのか不安
- 顔出し・声出しとか必要なの？
- 尖った発信とかできない

**グダ消しポストの方向**:
| グダ | 消すポストの方向 |
|---|---|
| 実績がないと稼げない | 実績なくてもいける（具体例提示） |
| 商品作るの大変そう | 他人の商品を売って稼ごう（アフィ提案） |
| 高額商品が作れないから無理 | 高単価アフィなら高単価いける |
| 大手にゴマスリするの嫌だ | 一人でも伸ばせる |
| 時間がかかるんでしょ？ | 初期から伸ばせる裏ワザ |
| 毎日ポストが思いつかない | ポストネタ一覧があってパクればOK |
| 後発だと稼げるのか不安 | 今がバブル！チャンスすぎる |
| 顔出し・声出しとか必要なの？ | 顔出し・声出し不要 |
| 尖った発信とかできない | 主婦が尖らず稼いだ事例があります |

### 業界事例: アダルト系商材（裏垢ジャンル）ローンチ

> 競合との協業で、3週間の準備で **7,000万円を売り上げた** ローンチ事例。
> グダ消しの実践例として価値が高いので原本のまま転記。

#### 商材

- ジャンル: 裏垢マネタイズ（女性裏垢のようなアカウントを作り、おじさんのファンをつけてnoteなどを販売する手法）
- 売上ハードル: 「おはよう〜❤️」くらいポストが雑でも、おっさんの性欲で伸びるので運用が超簡単で稼ぐハードルが低い
- 実施時期: 2024年12月
- 売上: **7,000万円**（3週間準備）

#### 洗い出されたグダ要素 25項目

太字＝特に重要なグダ要素。

1. **人に言えない（社会的地位）/ 旦那の目が気になる / バレるの恥ずい / 子供の目**
2. 承認欲求が満たされない / 自己重要感を満たせない
3. 単純作業すぎて飽きる / 楽しくない
4. 裏アカのスキルが横展開できない / スキルの市場価値が上がらない
5. ネカマでおじを騙している感覚がある（感謝されづらい）
6. **おじとのやり取りがしんどい（気持ち悪い内容 / 会いたい / セクハラ対応）**
7. リプとかDMでの精神的負担とリプ返しの労力的負担がある
8. **性的な気持ち悪さ / 男の裏側がしっかりと見えるのでしんどい**
9. **体の画像やエロ画像を出さないといけないのか？**
10. シャドバン多いイメージで新規アカ作らないといけない / 端末たくさん買う
11. センスのあるストーリーとか切り口が必要なんじゃないか（バズセンス）
12. **犯罪リスクがある / 会社クビになる / 子どもや家族に迷惑をかける**
13. 恋愛経験がないからおじに魅力的に見てもらう方法わからん
14. 競合が多くなるのでは？（再現性ある + 今から遅いのか？）
15. 資産性がない（シャドバン / 競合過多）→ ずっと自転車操業
16. 0→1の突破は簡単だが収益性が低い（一発で稼げない）
17. 短命で終わる
18. **男性が運用する場合は女性の演者必要 / 仲良い女の子がいないから協力者いない**
19. 毎月新作を作り続けてローンチをしないといけない
20. 全体の売るための流れがわからない / マネタイズまでのイメージがつかない
21. 自動化ができず大変そう
22. 外注の探し方がわからない（外注コストだるい…）
23. アカウントを大量に運用しないといけないのだるい
24. （他、洗い出し可）
25. （他、洗い出し可）

#### 実際に投下したグダ消しポストの方向

- 「クリーンで稼げないよりも、まずは稼げた方がいい」（社会的地位グダ → 価値観で塗り替え）
- 「ちゃんとやれば法的リスクは低い」（犯罪リスクグダ）
- 「エロ画像いらない。女性演者も不要。」（エロ忌避 + 男性参入グダ）
- 「おぢとのやりとりが嫌なら安く外注して自動化できる」（やり取り負担グダ）

#### この事例から学べるグダ消しの設計手順

1. **競合と一緒に商材を分析** → 想定グダを20〜30個書き出す
2. 太字でマーキング（**5〜10個に絞る**）
3. 各グダに対応する **逆論破ポスト** をローンチ前から仕込む
4. 同じグダに対しても **角度を変えた複数本** で攻める（1ポストでは消えない）
5. ローンチ直前で「全グダ消えたか」セルフチェック

#### 事例集: ぶんじろう垢のナンパ系ストーリー3パターン

1. **実績ゼロ・小さな成功でもファン化は起こる**

2. **失敗オチも有効**
   - 3パターンのうち2つは失敗オチ
   - 失敗だからこそ拡散性・共感性が高まることもある

3. **強者ブランディング不要**
   - 普通のX運用は実績ゴリ押しで強者ぶる
   - ストーリー型は **弱者・挑戦者ポジション** で機能する

#### 事例: 田中の高単価アフィ実践記

> ストーリー型運用 + 教育の典型成功例。「変化率」と「再現性の教育」の好例。

| 要素 | 田中事例の中身 |
|---|---|
| **目新しさ** | 「日本語も絶望的な超へっぽこ部下が、高単価アフィという新しいスキームに挑戦」 |

> へっぽこが870万円を稼ぐ → **再現性の教育**（「俺でもいけそう」）

→ ただのバズ狙いではなく、最終的に商材（高単価アフィ系）への誘導に繋がる教育設計。

→ ポイント: **通知オン勢に名前を付ける（「なまチル勢」）** ことで承認型インセンティブも追加。

##### 実例3
```
ポストを1つ消した
悪いポストは即削除する
悪いポストがわかるのが重要だから
見れた人はラッキー
```
→ ポイント: **「悪いポストがわかるのが重要」** という付加価値を匂わせて、見たかった度を上げる。

##### 機能した結果のリプ実例
- 「通知オンで見れました！！」（追従誘導）
- 「通知オンしてたけど見れませんでした…」（更なる後悔煽り）
- 「通知オンにしました！」（行動変容の報告）

→ こうしたリプを **そのまま残す** ことで、他の読者にも追従効果が広がる。

#### のんびりくん裏垢ローンチの伏線設計（教材6/7事例の再分析）

> 教材6/7では「ストーリー型運用」として保存した事例を、教材8の **インセンティブ設計視点** で再分析。

##### 全体フロー
```
[1] なまいきくんが裏垢に挑戦する（ストーリー起）
       ↓
[2] エロ画像を撮りまくる暴走（ストーリー承）
       ↓
[3] 「事件」ジャブポスト ← インセンティブ設計の伏線
       ↓
[4] 「実はエロ画像不要」教育ポスト ← 急所ヒット
       ↓
[5] のんびりくんセミナーへ誘導 ← セールス
```

##### きあいだめ理論の機能

```
ジャブポスト「最悪な出来事…」「これまでの努力が無駄に…」← きあいだめ
  ↓
読者「え、何があったの？！」← 注目度MAX
  ↓
教育ポスト「実はエロ画像は要らなかった」← 急所ヒット（グダ消し教育）
  ↓

1. **ストーリー型 × インセンティブ設計** は重ね合わせると最強
2. ジャブポストは **教育もセールスもしていない**（単体では無価値）

#### 事例3: 弁護士事件→特大ブチギレポストチェーン（掌握ローンチ前）

> 出典: 教材9（2026-04-27）
> 「ためてためて放出」連鎖の典型例。

##### 5ポスト構成

| ポスト | 内容（要旨） | インプ | 役割 |
|---|---|---|---|
| 1 | 弁護士案件をほのめかす | （小） | 伏線・捨てジャブ |
| 2 | さらに事件性を煽る | （小） | 伏線・捨てジャブ |
| 3 | ジャブ的反応 | 5万 | 部分発火 |
| 4 | 「ブチギレポスト書いてたら数千文字消えた」 | （中） | さらなる事件性 |
| 5 | **特大ブチギレポスト（本命）** | **18.6万** / 143RT / 165リプ / 538いいね | 本命爆発 |

##### 思考プロセスの裏側

教材執筆者の自己分析:
```
ポスト1の段階では、何も計画していなかった。
弁護士案件が発生したとき、「あぁ面倒ごとが起こった、ポストでネタにしてやろう」程度。
ローンチ前だから何かに繋げようかな、くらい。

ポスト1後に、どう教育に繋げるかを考えてブチギレポストを思いつく。
そこでポスト3を打つ。

ブチギレポストを書いてたら吹っ飛んだので、それもネタにしてポスト4。

そして、満を持して本命のポスト5。
```

→ **完全に計画していたわけではない**。事件が起こったらすぐ「教育に繋げられないか？」と考える思考が重要。

##### 機能した要素

- 事件性・ニュース性
- 感情の動き（「ブチギレ」）
- リアルタイムで起きた出来事を時系列で見せる（ストーリー型運用）
- 各ポストが次のポストの威力を上げる（捨てジャブ理論）

##### この事例から学べる原則

1. **完璧な事前計画は不要**
2. 起きた出来事を **「教育に繋げられないか？」** と考えるクセ
3. 事件・トラブル・感情の動きは **全部ジャブの素材**
4. 5本連鎖でも100倍の威力が出る

#### ビジ垢「まだまだ客→今すぐ客」価値観遷移マップ

ビジ垢（占いマネタイズコンサル）のターゲット = 占いマネタイズしたい人。
彼らがどう価値観遷移してコンサル購入に至るか:

```
[まだまだ客 / 階段0段目]
会社員しんどい / 副業興味あるけど何やればいいかわからない
       ↓ 価値観操作
[階段1段目]
副業しないとやばい / Xマネタイズって稼げるらしい
       ↓ 価値観操作
[階段2段目]
占いマネタイズって聞いたことあるけど自分にもできるかな
占い師じゃないけどスピ系興味あるからやってみたい
       ↓ 価値観操作
[階段3段目]
占いマネタイズは在庫不要・高単価・労働切り売りじゃない最強市場
自己投資して知識つけるのが稼ぐ近道
       ↓ ロジック構築
[今すぐ客 / 階段4段目]
占いマネタイズで稼ぐにはコンセプト設計が9割
ノウハウ独学だと遠回り、コンサル受けるのが最短
〇〇さんから学ぶのが一択
       ↓
コンサル予約します！
```

#### ビジ垢のロジック構築テンプレ

ビジ垢で「占いマネタイズコンサル」が欲しくなるロジックを4ステップで組む:

```
[① 共感]
占い系で稼ぎたいって思ってる人、めちゃ増えてるんですよね。
でも大体の人がフォロワー伸びない・収益化できないで止まる。

[② 真因]
で、伸びない人のアカウント見ると、ほぼ全員「コンセプト設計」でつまづいてる。
タロット教えますとかスピ系発信ですみたいなアカウントは、
誰に何を売りたいかが曖昧で、ファンが付かない。

[③ 解決策]
コンセプトを「特定の悩み × 特定の顧客 × 自分の強み」で組めば、
1ヶ月で1,000フォロワー超える人もザラにいる。
コンセプト設計が9割って言っても過言じゃない。

[④ 商品導線]
俺のコンサルでは、最初の1週間でコンセプトを叩き直すところから入る。
ここを直すだけで、あとは普通に運用してれば伸びる。
迷ってる人は1度話聞きにきてください。
```

#### ビジ垢のロジック構築バリエーション

商品/フェーズによってロジックは変わる。テンプレ集:

##### コンセプト設計の重要性ロジック
```
真因: コンセプトの曖昧さ
解決: 「悩み×顧客×強み」の3軸設計
導線: コンセプト設計コンサル
```

##### 集客導線の重要性ロジック
```
真因: フォロワーいてもLINE登録ない
解決: 集客導線の整備（教育コンテンツ→LINE→LP）
導線: 導線設計コンサル / テンプレ販売
```

##### ローンチノウハウの重要性ロジック
```
真因: 商品作っても売れない（ローンチノウハウない）
解決: 教材4・8の運用 + ローンチ設計
導線: ローンチコンサル / 0→1突破コース
```

#### 重要な数値メモ

> ロジック形成あり vs なし → 売上が **2倍以上** 変わる

→ ビジ垢のローンチ前は必ずロジック構築ポストを **複数本** 仕込む。

ビジ垢（占いマネタイズコンサル）のターゲット = 占いマネタイズ志望者。
彼らの **絶望客 → 理想客** の価値観遷移を詳細化:

□ なまいきくん（自垢）は強そうだけど怪しい

□ 自己投資にお金は使いたくない
□ ノウハウ独学でなんとかしたい
□ 失敗するの怖いから現状のままでいい
□ 自分には占い系で稼ぐ才能ないと思う
□ 他の人は稼げるけど自分は無理
□ 占い界隈はオワコンだと聞いた
□ 行動するの面倒、もうちょい考えたい
□ コンサルとか高額商品はぼったくり
```

##### ビジ垢の理想客の価値観
```
□ 〇〇さんから占いマネタイズを学びたい！
□ 在宅で占いマネタイズで稼いで自由に生きたい
□ 〇〇さんの発信は信頼できる、全部見てる
□ 自分が伸びない原因はコンセプト設計だと気づいてる
□ 解決策はコンサル受けることだと分かっている
□ 〇〇さんの手法は再現性ある、自分に合ってる
□ 知識・環境にお金使うのは当たり前
□ ある程度のリスク取らないと稼げない
□ 自責で行動するのが起業家マインド
□ チャンスは即動かないと逃げる
```

#### ビジ垢用 6つの教育の追加tips実例

##### ① 目的の教育（小実績見せ方の応用）

ビジ垢で実績がそこまで大きくない時期の見せ方:
```
❌ 「占いマネタイズで月50万稼ぎました」
⭕ 「占いマネタイズって寝てる間に売上来るのバグすぎる…
    今月50万円超えたけど作業時間1日30分くらい笑」
```

→ 「労力低そう」「当たり前そう」を演出。

##### ② 信用の教育（共通点作り）

ビジ垢用の共通点パターン:
- 「俺もスピ系1ミリも信じてなかった」
- 「占いマネタイズ知ったとき胡散臭いと思った」
- 「最初の3ヶ月はフォロワー50人だった」
- 「コンサル受けるまで月3万も稼げなかった」

→ 占いマネタイズ志望者と同じスタート地点を演出。

##### ③ 問題点の教育（ダイエット例の応用）

```
あなたが占いマネタイズで稼げないのは「コンセプト設計」が原因なので、
ここを直さない限り一生稼げません。

それどころか間違ったコンセプトでアカウント運用してると
逆にブランドが固まって修正が効かなくなるので
まず最初にコンセプトを定めないと結構ヤバいです。
```

##### ④ 手段の教育（他選択肢消し）

```
「物販は仕入れリスクと低利益率で初心者には厳しい」
「動画編集はスキル習得に時間かかるし単価も低い」
「アフィは商品リサーチで時間溶かすし安定しない」
→ だから占いマネタイズが資金不要・高単価・スピリチュアルブーム的に最強
```

##### ⑥ 投資の教育（学費例の応用）

```
普通に大学院・専門学校に300万円払う人は多いけど、
占いマネタイズコンサルに30万円払う人は少ない。

ただ大学の知識より占いマネタイズの方が100倍収入に直結するので、
ここに投資しないのは機会損失でしかない。
```

#### 教育を経ない面談NG（ビジ垢実例）

ビジ垢のローンチでよくある失敗パターン:

```
❌ NG例
1. ローンチ告知ポスト
2. いきなり面談予約LINE
3. 面談で売り込み
   → 絶望客のまま面談 → 「ちょっと考えます」で終了

⭕ 正しい流れ
1. 普段から6つの教育で理想客を増やす（教材4）
2. ローンチ前に伏線・インセンティブ設計（教材8/9）
3. ロジック構築教育で「コンサル必要」と思わせる（教材10）
4. ストーリー型運用で注目集める（教材6/7）
5. 面談予約 → ほぼセールス不要で成約
```
