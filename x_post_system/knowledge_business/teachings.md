#### ビジ系特有のフック例

ビジ系で実際にハマるフックの具体例。生成時のサンプルとして使う。

- 「日給2万円いったんやが…」
- 「月2000万稼ぐ友達に聞いたんだけど…」
- 「20垢運用してわかった」
- 「メルカリで1億売り上げてる主婦から聞いた」
- 「noteで月2000万円売った友達に教えてもらった」
- 「万垢30個分析してわかったんだけど」
- 「23歳のフリーターの弟に教えたら速攻で83万円稼いじゃった」
- 「今裏で仕込んでるジャンルだけど」
- 「ワンチャン月500万円くらいいけそうw こっそりやっちゃうよーん！」
- 「俺が〇〇円稼いだ」
- 「昨日仕入れに行って爆利益品みっけたんだけど」

#### ビジ系の言い換え（目新しさ作り）

「よくある表現」を避けて目新しく見せるためのビジ系特化の言い換え集。

| よくある表現 | 目新しい表現 |
|---|---|
| メルカリ | とあるプラットフォーム |
| noteのタイトルを改善した | noteに1分だけテコ入れした |
| 裏垢で稼ぎました | ちょいグレーなジャンル攻めてますw |
| 月収10万円超えました | 今月114,983円来たね |

#### ビジ系の実績画像バリエーション

物販・Tips販売・アフィ系で使える画像の見せ方:
- 売上画面（ダッシュボード）
- 銀行振込された画面
- 別のプラットフォームの画面
- 発送前の商品画像（物販）
- ※ 同じ実績でも見せ方を変えて反応を試す

#### ビジ系の実例（要素分解付き）

##### 例1: ぶんじろう垢のTips販売継続報告
```
本文:
  この垢でTips販売開始して3ヶ月目とかやけど地味に売れ続けるコツみたいなんが
  若干掴めつつある。商品数がゴミやから売上しょぼいけどこの容量でガンガン
  増やしていくか。ぶんじろう垢と違って一撃の爆発力はないけど属人生を
  限りなく落としてるから正味誰でも運用できるからかなり楽。
  （売上画像）

3要素:
  - フック: 売上画像 / 商品増やしたら儲かる（雰囲気）
  - 興味対象: 売れ続けるコツ / 属人性のない運用方法
  - 強化: 属人性落としてる / 誰でも運用できる / かなり楽
```

##### 例2: にきち垢の即日現金化せどり
```
本文:
  今日友達にとある即日現金化できる㊙せどり手法教えたんだけど、
  速攻で利益29047円ゲットしてて草
  内訳: 仕入れ61,000 / 売上90,047 / 利益29,047
  もうこの手法知っちゃったら他の手法なんてめんどくさすぎてやってられないね

3要素:
  - フック: 友達が速攻で利益2.9万円ゲット（他人の行動＋成果）
  - 興味対象: 即日現金化できる㊙せどり手法
  - 強化: 速攻（即効性）/ ㊙（秘密性）/ めんどくさすぎてやってられない（優越感・手軽さ）
```

##### 例3: 新商材2日目ポスト（作成例）
```
本文:
  新商材を仕入れ始めて2日目でこれ。仕入れ値やすいからダメージ少ないし
  回転率もいい。こういうのは情報持ってたら誰でもできるから、
  やっぱ情報ゲーやね。（実績画像）

3要素:
  - フック: 新商材を仕入れ始めて2日目でこれ
  - 興味対象: 仕入れ商材の情報
  - 強化: 仕入れ値安い / 回転率いい / 誰でもできる
```

#### ビジ系の参考データ（インプ実績メモ）

> ⚠️ 画像が貼れない都合で本文不明。インプ数のみ記録。後日本文を入手したら追記。

- 初期の興味付けポスト3点（あまり当たらず）: 7,900 / 2,600 / 4,700 インプ
- 「300万円の売上画像」のポスト: 当時のビジ系では数字大きいが反応イマイチ
- アフィ系の当たりポスト: **45,000 インプ**
- 学び: 当たるまで角度を変えて打ち続ける必要がある（→ 共通の「待ちのマインド」原則）

#### カタログ概要

10カテゴリ × 5前後のサブカテゴリ × 特徴ワード × 例文の大規模カタログ。
ビジ垢（占いマネタイズコンサル販売）の文脈に直撃する。

1. 他テーマと比較した自テーマの魅力
2. 自分の手法の魅力（同業者比較）
3. ロードマップ
4. 得られた変化（Before/After）
5. 憧れのライフスタイル
6. 実績報告
7. 初心者の第一歩
8. 誤解・先入観の否定
9. ノウハウ・考え方
10. 実践の様子（舞台裏）
+ その他便利な切り口（理念/小さな成果/読者肯定/周りとの違い/顧客の声/他選択肢のデメリット）

#### ビジ垢生成時の運用

- カタログから切り口を1つ選ぶ
- 教材1の3要素公式（フック × 切り口 × 強化要素）に乗せる
- 例文の「アパレル物販」「メルカリ」「仕入れ」を「占いマネタイズ」「X/note/LINE」「鑑定/コンテンツ作成」等に置換

#### 占い垢では使わない理由

- 占い垢はBtoC消費者向け（恋愛/金運の悩み解決）
- カタログは情報商材／コンサル販売文脈に最適化されている
- 金額訴求・同業比較は消費者には逆効果
