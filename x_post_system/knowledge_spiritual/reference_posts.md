# 参考ポスト集（占いスピ系）

伸びた投稿の実例集。各ポストに「構造分析」を併記する。

> 自アカ・他アカ問わず、参考になる投稿を蓄積する。

## フォーマット

```
### #001 [投稿タイトル/冒頭5文字]

- 投稿者: @xxx
- 取得日: YYYY-MM-DD
- 指標: いいねXXX / RT XX / インプ XX,XXX
- ジャンル: spiritual
- 本文:
  > （本文をそのまま）
- 構造タイプ: フック型 / リスト型 / ストーリー型 / etc
- 伸びた要素分析:
  - フック:
  - 展開:
  - 締め:
  - 使われたワード:
- 自アカ転用ヒント:
```
