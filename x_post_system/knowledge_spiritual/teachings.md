- ただし以下のメタ原則は流用可:

## 占い垢の階段モデル（仮）

> 出典: 教材3「反応がとれる良いポストの5つのポイント」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。

ビジ垢が「収益額」で階段を切るのに対し、占い垢は **悩みの深さと占いとの距離** で階段を切る。

```
4段目: 鑑定で人生変わった経験あり、リピーター・ファン
  ↑
3段目: 占いを信じて鑑定を受けたことがある（単発）
  ↑
2段目: 占いに興味あり、無料コンテンツやSNSをフォローして見ている
  ↑
1段目: 恋愛/金運の悩みを抱えてSNSを眺めている  ← ⭕ ターゲット
  ↑
0段目: 漠然と人生に不安がある（占いを意識してない）  ← ⭕ ターゲット
```

### 階段ごとのポスト方針（仮）

#### 0段目向け
- 「占いってこんなことまで分かるんだ」と思わせる
- 占いそのものへの興味付け
- 漠然とした人生不安にフックをかける

#### 1段目向け
- 恋愛/金運の悩みに具体的に寄り添う
- 「自分の悩み、占いで読み解けそう」と思わせる
- 鑑定を受けてみたくなる興味付け

### ⚠️ ビジ垢の階段思考をそのまま流用しない理由

- ビジ垢は「収益化」の階段（金額訴求）
- 占い垢は「悩み解決」の階段（感情訴求）
- 構造は同じでも語彙・訴求軸が全く違う

## 6つの教育の占い垢翻訳ガイド（仮）

> 出典: 教材4「ポストネタの考え方」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。
> フレームワーク本体は `../knowledge_common/education_framework.md` 参照。

### 各教育タイプの占い垢例

#### ① 目的の教育（占い垢）

「鑑定で人生変わった理想状態」を見せて欲求喚起。

- 「鑑定後に運命の人と出会えて結婚できた」（恋愛の理想）
- 「金運上昇のメソッドを実践したらボーナスが入った」（金運の理想）
- 「悩みが消えて毎日が穏やかになった」（精神の理想）
- 「お客様が『人生変わりました』と泣いてくださった」（鑑定者側の喜び）
- 「満月の鑑定でこんなに変わるとは思わなかった」（リアルタイム性）

> ⚠️ ビジ垢のような「月収」「稼げた」訴求はNG。あくまで「人生・運気・恋愛・金運の好転」で訴求。

#### ② 信用の教育（占い垢）

- 「20年の鑑定歴」「のべ1万人鑑定」（実績）
- 「神社の宮司から授かった流派」（権威）
- 「最初は自分の人生に絶望していた→占いに救われた→今は人を救う側に」（紆余曲折ストーリー）
- 「占いを軽蔑していた私が信じるようになった理由」（共感パターン）
- お客様の声・体験談の引用
- 鑑定中の写真・神社参拝の写真（視覚的信用）

#### ③ 問題点の教育（占い垢）

「悩みを放置した時の地獄」を煽る。

- 「自己流の開運は逆効果になることがある」
- 「悩みを放置すると運気がどんどん下がる」
- 「『占いは信じない』と言ってる人ほど、人生詰みやすい」
- 「タイミングを逃すと運命が固まって変えられなくなる」
- 「金運が悪いまま放置すると、5年後に大損害を被るパターンが多い」

仮想敵候補:
- 「無料アプリの占い」
- 「自己流のスピリチュアル」
- 「占い否定派の常識」

#### ④ 手段の教育（占い垢）

- 「タロット鑑定で根本原因が分かる」
- 「あなたの星回りに合ったパワーストーンが必要」
- 「神社選定鑑定で運気が変わる」
- 「九星気学に基づく開運メソッド」
- 「他のジャンル占いでは見えない部分を当流派なら見抜ける」

#### ⑤ 行動の教育（占い垢）

- 「タイミングを逃すと運気が変わる」
- 「満月の夜に動け」「新月のうちに願いを書け」
- 「動かない人に占いの効果はない」
- 「迷ってる時間が一番運気を下げる」
- 「『考えます』と言ってる人ほど運命を逃している」

#### ⑥ 投資の教育（占い垢）

- 「自分の人生に投資する＝鑑定を受ける」
- 「お金をかけずに本気で人生変えるのは無理」
- 「無料の占いと本物の鑑定は別物」
- 「鑑定料金を惜しむ人ほど、損失の方が大きい」
- 「私自身も〇〇万円の鑑定を受けて人生が変わった」

## 占い垢の想定グダ要素（仮）

占い垢の商品（鑑定・パワーストーン）を売る際に、お客さんが抱きそうなグダ要素のリスト。

> ⚠️ 仮リスト。占い系教材で正式版に更新する。

### 占い・鑑定への疑念系グダ

- 「占いって当たらないんでしょ？」
- 「占い師って胡散臭い」
- 「結果が悪かったら知るのが怖い」
- 「自分の悩み程度で鑑定受けていいの？」
- 「占いに頼るのは情けない」

### 金額・コスト系グダ

- 「鑑定料金が高い」
- 「パワーストーンって高いだけで効果なさそう」
- 「お金をかけずに自分で何とかしたい」
- 「無料の占いアプリで十分」

### プライバシー系グダ

- 「個人情報を教えるのが怖い」
- 「家族や友達に占いに頼ってると言えない」
- 「彼氏・夫に内緒で受けたい」

### 効果への疑念系グダ

- 「鑑定受けても本当に変わるの？」
- 「過去にも占い受けたけど何も変わらなかった」
- 「自分は特殊なケースだから当たらないのでは」

### 行動への抵抗系グダ

- 「予約取るのが面倒」
- 「対面は緊張する」
- 「いつ受ければいいタイミングか分からない」

### グダ消しポストの方向（仮）

| グダ | 消すポストの方向 |
|---|---|
| 占い当たらない | 当たった具体事例・お客様の声で実証 |
| 鑑定料金が高い | 「自分への投資」フレーミング + 損失計算 |
| 結果が怖い | 「悪い結果も対処法がある」「知らない方が怖い」 |
| 自己流でいい | 「自己流の落とし穴」事例 |
| 占いに頼るのは情けない | 「成功者ほど占いを使っている」「迷う時間こそ無駄」 |

## 占い垢のストーリー型運用応用案（仮）

> 出典: 教材6「ストーリー型運用」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。
> フレームワーク本体は `../knowledge_common/story_operation.md` 参照。

### ⚠️ 占い垢ストーリー型の重要原則

ビジ垢（占いマネタイズコンサル）で機能するナンパ系のような **過激なストーリー** は、**占い垢（BtoC消費者向け）ではNG**。

| 観点 | ビジ垢のストーリー | 占い垢のストーリー（仮） |
|---|---|---|
| **温度感** | 過激・笑い・失敗ネタOK | 共感・優しさ・神秘性 |
| **主人公** | 弱者・挑戦者・童貞 | 悩める女性・人生迷子・占い師自身 |
| **クライマックス** | 派手な成功/失敗 | 鑑定後の人生変化・運命的な出会い |
| **オチ** | 笑える失敗オチもアリ | 心温まる結末 / 神秘的な啓示 |

→ 占い垢は **共感性 > 過激さ**。占いの神秘性を損なわない範囲で組む。

### 占い垢で作れるストーリーパターン例（仮）

#### パターン1: 自分自身の人生変化ストーリー

```
[Post 1] 「20代の私は人生どん底、毎日泣いてた」
  → 弱者ポジションスタート

[Post 2] 「占いに出会ったきっかけはこんな偶然だった…」
  → 出会いの物語

[Post 3] 「半信半疑で受けた鑑定で言われた一言が衝撃すぎた」
  → クライマックス前の引き

[Post 4] 「あの一言通りに動いたら、本当に人生が変わった【オチ】」
  → 信用の教育を埋め込んだストーリー完結
```

→ 信用の教育（紆余曲折ストーリー）+ ストーリー型のハイブリッド。

#### パターン2: お客様の悩み解決ストーリー（匿名・許可前提）

```
[Post 1] 「30代独身のAさん、結婚相手が見つからず鑑定にきた」
[Post 2] 「彼女のホロスコープを見たら、想像を超える結果が…」
[Post 3] 「過去の恋愛が全てある一つの傾向だった」
[Post 4] 「3ヶ月後、運命の人と出会えたという報告が【オチ】」
```

→ 第三者目線で語ることで、自分への自慢にならず信用が積み上がる。

#### パターン3: 開運メソッド実験企画

```
[Post 1] 「100日後に運命の人と出会う」企画スタート
[Post 2] 「Day 30: 神社参拝を続けたら金運が動き出した気がする」
[Post 3] 「Day 60: 想定外の出会いがあった！運命？」
[Post 4] 「Day 100: 結果報告【オチ】」
```

→ YouTube・TikTokで人気の「100日後に〇〇する」をXに持ってくる戦略。

#### パターン4: 自身の鑑定師としての成長ドキュメント

```
[Post 1] 「OLだった私が占い師になることを決めた日」
[Post 2] 「最初の鑑定、震える手でカードを引いた」
[Post 3] 「3ヶ月で100人鑑定。気づいたある共通点」
[Post 4] 「鑑定で人生が変わったお客様からの手紙【オチ】」
```

→ 占い師としてのバックストーリーを連投で見せる、信用構築型ストーリー。

### 占い垢ストーリー型のNG

- ❌ 過激な失敗オチ（占いの信頼性を損なう）
- ❌ ナンパ・性的内容（消費者層に合わない）
- ❌ 鑑定結果を断定的に保証する内容（特商法・占い倫理）
- ❌ 個人を特定できる事例（プライバシー）
- ❌ 不安を過剰に煽るストーリー（消費者の信頼を失う）

### クライマックス前で切るときの占い系言い回し例

```
✅ 「次の満月でついに動き出す。何が起きるか…」
✅ 「カードが示した結果が衝撃的すぎて言葉が出ない」
✅ 「この鑑定の続き、明日の夜に話します」
✅ 「彼女に伝えた一言、その後の展開が…」
```

## 占い垢のネタ3要素設計（仮）

> 出典: 教材7「ストーリー型運用の組み方とポイント」を占い垢に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。

### 目新しさの作り方（占い垢用）

ビジ垢の「副業マネタイズ」ではなく **占い・スピ系の文脈** で目新しさを作る:

⭕ 機能する目新しさ:
- 「20年無宗教だった私が占い師になるまで」（背景の意外性）
- 「占い師の私が30日間ノー鑑定で運気を上げる実験」
- 「神社100社巡りで人生がどう変わるか全公開」
- 「自分のホロスコープ通りに行動したらどうなるか実験」
- 「〇〇という超マイナー占術で人生変えてみる」

❌ 弱い目新しさ:
- 「タロットの基本を発信します」（普通すぎ）
- 「毎日のラッキーアイテム」（無数の人がやってる）

### 変化率の作り方（占い垢用）

ビジ垢の「収益の差」ではなく **人生・運気・恋愛/金運の差** で変化率を作る。

#### スタート地点を下げる例

| ❌ 弱い | ⭕ 強い |
|---|---|
| 普通の30代女性が占いを学ぶ | 「占い大嫌いだった私」が占い師に |
| 適度に幸せな人が運気アップ | 「人生どん底・うつ寸前」だった私が |
| 軽い恋愛悩みの人 | 「10年片想い・諦めかけてた」女性 |
| 普通の主婦が金運アップ | 「借金300万・自己破産寸前」だった主婦 |

#### ゴールの示し方

- 「半年で運命の人と結婚」
- 「3ヶ月で金運が劇変、ボーナス想定外の額」
- 「人生180度変わった」
- 「鑑定で見えた未来通りに人生が動いた」

### 事件性の作り方（占い垢用）

⚠️ ビジ垢のような **過激な事件性（警察・借金・性病等）はNG**。
占いの神秘性・信頼性を損なわない範囲で組む。

⭕ 機能する事件性:
- 「鑑定中に予言した出来事が翌日リアルに起こった」
- 「神社参拝中に偶然〇〇が起きた」
- 「カードが警告した通りの出来事が…」
- 「お客様が鑑定の3日後に運命の人と出会った」
- 「私が予言した通りに〇〇さんに連絡が来た」

→ **シンクロニシティ**・**予言的中**・**神秘的偶然** を事件性として使う。

❌ 占い垢で使えない事件性:
- 過激なトラブル系（警察沙汰・性的事件）
- 占いそのものの否定的事件（「鑑定が外れた」「効かなかった」）

## 占い垢のピエロ戦略の限界と代替案（仮）

> 出典: 教材7「ピエロ戦略」を占い垢用に翻訳（2026-04-27）

### ⚠️ ピエロ戦略は占い垢では機能しにくい

ビジ垢（コンサル）では「俺アホやけど稼げてるよ」が共感を生むが、占い垢では:

| 観点 | ビジ垢 | 占い垢 |
|---|---|---|
| アホ演出 | ⭕ 再現性訴求になる | ❌ 占い師の権威・神秘性を損なう |
| 弱者ポジ | ⭕ 「俺でもいける」共感 | △ 鑑定能力への信頼を毀損 |
| 笑える失敗 | ⭕ 親近感 | ❌ 鑑定への信頼を損なう |

### 代替戦略: 「等身大の親近感」

占い垢では **権威性は鑑定スキル側に置いたまま**、人間味は別軸で出す:

#### ⭕ 機能する親近感の出し方

- 占い師としての修行時代の苦労話（権威の前段階）
- 自分自身も占いに救われた一人としての姿勢
- お客様への感謝・喜びの素直な表現
- 占いとは関係ない普通の生活感（カフェ・ペット・趣味）
- 鑑定中の悩み・葛藤（「この鑑定結果伝えるのが辛い」等）

#### 言い回しの例

```
✅「私も昔は占いを信じていなかったんです」（過去の弱さ→今の権威への流れ）
✅「鑑定をしていて、たまに自分が見えすぎて怖くなる時があります」（権威+人間味）
✅「お客様が泣いて喜んでくれた時、私も一緒に泣いてしまった」（感情の振れ幅）
```

### 占い垢で使える「感情の振れ幅」

ピエロ戦略は使えないが **感情の振れ幅は使える**:

- 鑑定結果に感動して泣く
- お客様の人生変化を聞いて号泣する
- 神社の不思議体験で鳥肌が止まらない
- 怖い結果が出た時の驚き・葛藤

→ ビジ垢の「ブチギレ・テンション爆上げ」より、**深い感動・驚き・畏怖** の方向で振れ幅を作る。

## 占い垢のインセンティブ設計案（仮）

> 出典: 教材8「インセンティブ設計」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。
> フレームワーク本体は `../knowledge_common/incentive_design.md` 参照。

### ⚠️ 占い垢のインセンティブ設計の特徴

ビジ垢が **金銭的特典・実績誇示** をインセンティブにするのに対し、占い垢は **神秘性・体験ベース** のインセンティブが機能しやすい:

| 観点 | ビジ垢 | 占い垢 |
|---|---|---|
| 主軸 | 金銭・実績 | 神秘・運気・縁 |
| 損失回避 | 「稼げない」恐怖 | 「運気が下がる」恐怖 |
| 特典 | 限定ノウハウ | 限定鑑定・祈祷・お守り |
| 承認 | 実績の認知 | 「選ばれた」「縁がある」感 |
| 限定性 | 時間限定（先着等） | 占い的タイミング限定（満月・新月・大安） |

### 行動別のインセンティブ案（仮）

#### 通知オン化のインセンティブ
```
仮案:
「特定日時にしか言わない開運のヒント」
「満月の夜だけ流す浄化のメッセージ」
「ポスト即消し戦略 → 『見れた人はその時点で運気が動いてます』」
「通知オン勢を『〇〇月会』のような限定コミュニティ呼称で承認」
```

→ ビジ垢の即消し戦略は占い垢でも使えるが、**「見逃した＝運気が下がる」** に演出することで損失回避型インセンティブを強化できる。

#### いいねのインセンティブ
```
仮案:
「いいね数だけ運気が上がる企画」
「祈祷を込めて1いいね1祈り」
「いいね100超えたら全員に守護メッセージ送ります」
「特定時刻にいいねくれた方は今月の運気アップ確定」
```

→ 占い垢では **「いいねが行動として運気に影響する」** という神秘性を持たせるのが効く。

#### 引用RT・拡散のインセンティブ
```
仮案:
「シェア10で1名鑑定無料」
「拡散すると徳が積まれる」（仏教的フレーミング）
「拡散者の中から運命の人にアドバイス」
```

#### 感想記入のインセンティブ
```
仮案:
「感想書いてくれた方に守護メッセージを返信」
「感想いただいた方限定で個別メッセージ」
「感想を読ませてもらうと、自然にあなたの運気を見させてもらってます」
```

→ 占い垢の感想特典は **金銭的特典より個別の鑑定的サービス** が機能する。

#### スペース参加のインセンティブ
```
仮案:
「リアルタイム参加者全員に集団祈祷」
「スペースで言うのはここだけの極秘鑑定情報」
「参加者にしか教えない満月の儀式」
```

#### LINE登録のインセンティブ
```
仮案:
「LINE限定で個別の運気カレンダー配布」
「登録者だけに今月の運気アップ波動を送る」
「LINEだけでやる無料簡易鑑定」
```

### 占い垢のジャブポスト（伏線）案

```
仮案:
「今夜、衝撃的なメッセージが降りてきました…明日の朝にお話しします」
「鑑定中に予言した出来事が起こってしまいました…」
「カードが警告した内容が、現実になりつつあります」
「神社で頂いたメッセージが想定外でした、明日詳しく」
```

→ 占い垢のジャブは **神秘性・予言性** を活用する。「事件」より「啓示」「兆し」のフレーミング。

### 占い垢のローンチ伏線回収プラン（仮）

```
[Pre-launch -3M] 即消し戦略 → 通知オン勢「〇〇会」を作る
[Pre-launch -2M] 過去鑑定の感想を「運命変化エピソード」として集める
[Pre-launch -1M] スペースで「ここだけの鑑定情報」文化を作る
[Pre-launch -2W] 「啓示」系ジャブを散りばめる（神秘的予言）
[Pre-launch -1W] 「重要なメッセージが降りてきた」系で注目を集める
[Launch Day] 全伏線が回収され、通知オン勢が即反応 → 大爆発
```

### ⚠️ 占い垢でのNG（インセンティブ設計）

- ❌ あからさまに金銭的・現世利益的なインセンティブ（占いの神秘性を損なう）
- ❌ 過剰な煽り（「これ知らないと不幸になる」レベルの恐怖煽り）
- ❌ 鑑定結果を断定的に保証する特典（特商法・占い倫理）
- ❌ 個人を特定できる情報を扱うインセンティブ（プライバシー）

→ 占い垢は **神秘性・倫理・読者への敬意** のバランスを保ちながらインセンティブ設計する。

### 占い垢の売上8変数（仮）

```
□ ① 占い・鑑定・スピリチュアルに興味を持つ人の数
```

### 占い垢の事例応用案（仮）

#### 当事者意識醸成（The.X応用）

#### アフィ参戦のインセンティブ教育（占い垢用）

占い垢の場合、アフィの直接訴求より **「占い師仲間を応援する」** 文化を作るのが効果的（仮）:
```
「私のメソッドで救われた占い師仲間が、自分のお客様にも紹介してくれている」
「この鑑定法は伝統だから、広める使命がある」
「占い業界を一緒に盛り上げてくれる方を募集」
```

→ ビジ垢の「儲かる」訴求の代わりに **「使命感・連帯感」** で動かす。

#### 「シコいメンエス嬢選手権」応用案（占い垢用）

⚠️ 元事例はナンパ垢の事例。占い垢への応用:
```
「自分の人生で起こった奇跡エピソード選手権」
  → 読者が自分の体験を振り返る → 占いの神秘性を再認識 → 鑑定したくなる
  → 上位エピソードに無料鑑定プレゼント

「私の守護霊・守護神の絵を描いてくれた人選手権」
  → 読者がスピリチュアル世界に没入 → 占いマインドが醸成 → 鑑定や守護神鑑定したくなる
  → 上位の方にスピリチュアルなプレゼント
```

→ **「読者の頭を占い・スピリチュアルでいっぱいにする企画」** という構造で応用。

### ⚠️ 占い垢でのNG（教材9関連）

- ❌ 過激な事件性ジャブ（「警察沙汰」「借金」等は占い師の信頼性を損なう）

→ 占い垢のインセンティブは **神秘性・体験ベース・連帯感** で組む。
