- ただし以下のメタ原則は流用可:

## 占い垢の階段モデル（仮）

> 出典: 教材3「反応がとれる良いポストの5つのポイント」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。

ビジ垢が「収益額」で階段を切るのに対し、占い垢は **悩みの深さと占いとの距離** で階段を切る。

```
4段目: 鑑定で人生変わった経験あり、リピーター・ファン
  ↑
3段目: 占いを信じて鑑定を受けたことがある（単発）
  ↑
2段目: 占いに興味あり、無料コンテンツやSNSをフォローして見ている
  ↑
1段目: 恋愛/金運の悩みを抱えてSNSを眺めている  ← ⭕ ターゲット
  ↑
0段目: 漠然と人生に不安がある（占いを意識してない）  ← ⭕ ターゲット
```

### 階段ごとのポスト方針（仮）

#### 0段目向け
- 「占いってこんなことまで分かるんだ」と思わせる
- 占いそのものへの興味付け
- 漠然とした人生不安にフックをかける

#### 1段目向け
- 恋愛/金運の悩みに具体的に寄り添う
- 「自分の悩み、占いで読み解けそう」と思わせる
- 鑑定を受けてみたくなる興味付け

### ⚠️ ビジ垢の階段思考をそのまま流用しない理由

- ビジ垢は「収益化」の階段（金額訴求）
- 占い垢は「悩み解決」の階段（感情訴求）
- 構造は同じでも語彙・訴求軸が全く違う

## 6つの教育の占い垢翻訳ガイド（仮）

> 出典: 教材4「ポストネタの考え方」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。
> フレームワーク本体は `../knowledge_common/education_framework.md` 参照。

### 各教育タイプの占い垢例

#### ① 目的の教育（占い垢）

「鑑定で人生変わった理想状態」を見せて欲求喚起。

- 「鑑定後に運命の人と出会えて結婚できた」（恋愛の理想）
- 「金運上昇のメソッドを実践したらボーナスが入った」（金運の理想）
- 「悩みが消えて毎日が穏やかになった」（精神の理想）
- 「お客様が『人生変わりました』と泣いてくださった」（鑑定者側の喜び）
- 「満月の鑑定でこんなに変わるとは思わなかった」（リアルタイム性）

> ⚠️ ビジ垢のような「月収」「稼げた」訴求はNG。あくまで「人生・運気・恋愛・金運の好転」で訴求。

#### ② 信用の教育（占い垢）

- 「20年の鑑定歴」「のべ1万人鑑定」（実績）
- 「神社の宮司から授かった流派」（権威）
- 「最初は自分の人生に絶望していた→占いに救われた→今は人を救う側に」（紆余曲折ストーリー）
- 「占いを軽蔑していた私が信じるようになった理由」（共感パターン）
- お客様の声・体験談の引用
- 鑑定中の写真・神社参拝の写真（視覚的信用）

#### ③ 問題点の教育（占い垢）

「悩みを放置した時の地獄」を煽る。

- 「自己流の開運は逆効果になることがある」
- 「悩みを放置すると運気がどんどん下がる」
- 「『占いは信じない』と言ってる人ほど、人生詰みやすい」
- 「タイミングを逃すと運命が固まって変えられなくなる」
- 「金運が悪いまま放置すると、5年後に大損害を被るパターンが多い」

仮想敵候補:
- 「無料アプリの占い」
- 「自己流のスピリチュアル」
- 「占い否定派の常識」

#### ④ 手段の教育（占い垢）

- 「タロット鑑定で根本原因が分かる」
- 「あなたの星回りに合ったパワーストーンが必要」
- 「神社選定鑑定で運気が変わる」
- 「九星気学に基づく開運メソッド」
- 「他のジャンル占いでは見えない部分を当流派なら見抜ける」

#### ⑤ 行動の教育（占い垢）

- 「タイミングを逃すと運気が変わる」
- 「満月の夜に動け」「新月のうちに願いを書け」
- 「動かない人に占いの効果はない」
- 「迷ってる時間が一番運気を下げる」
- 「『考えます』と言ってる人ほど運命を逃している」

#### ⑥ 投資の教育（占い垢）

- 「自分の人生に投資する＝鑑定を受ける」
- 「お金をかけずに本気で人生変えるのは無理」
- 「無料の占いと本物の鑑定は別物」
- 「鑑定料金を惜しむ人ほど、損失の方が大きい」
- 「私自身も〇〇万円の鑑定を受けて人生が変わった」

## 占い垢の想定グダ要素（仮）

占い垢の商品（鑑定・パワーストーン）を売る際に、お客さんが抱きそうなグダ要素のリスト。

> ⚠️ 仮リスト。占い系教材で正式版に更新する。

### 占い・鑑定への疑念系グダ

- 「占いって当たらないんでしょ？」
- 「占い師って胡散臭い」
- 「結果が悪かったら知るのが怖い」
- 「自分の悩み程度で鑑定受けていいの？」
- 「占いに頼るのは情けない」

### 金額・コスト系グダ

- 「鑑定料金が高い」
- 「パワーストーンって高いだけで効果なさそう」
- 「お金をかけずに自分で何とかしたい」
- 「無料の占いアプリで十分」

### プライバシー系グダ

- 「個人情報を教えるのが怖い」
- 「家族や友達に占いに頼ってると言えない」
- 「彼氏・夫に内緒で受けたい」

### 効果への疑念系グダ

- 「鑑定受けても本当に変わるの？」
- 「過去にも占い受けたけど何も変わらなかった」
- 「自分は特殊なケースだから当たらないのでは」

### 行動への抵抗系グダ

- 「予約取るのが面倒」
- 「対面は緊張する」
- 「いつ受ければいいタイミングか分からない」

### グダ消しポストの方向（仮）

| グダ | 消すポストの方向 |
|---|---|
| 占い当たらない | 当たった具体事例・お客様の声で実証 |
| 鑑定料金が高い | 「自分への投資」フレーミング + 損失計算 |
| 結果が怖い | 「悪い結果も対処法がある」「知らない方が怖い」 |
| 自己流でいい | 「自己流の落とし穴」事例 |
| 占いに頼るのは情けない | 「成功者ほど占いを使っている」「迷う時間こそ無駄」 |

## 占い垢のストーリー型運用応用案（仮）

> 出典: 教材6「ストーリー型運用」を占い垢用に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。
> フレームワーク本体は `../knowledge_common/story_operation.md` 参照。

### ⚠️ 占い垢ストーリー型の重要原則

ビジ垢（占いマネタイズコンサル）で機能するナンパ系のような **過激なストーリー** は、**占い垢（BtoC消費者向け）ではNG**。

| 観点 | ビジ垢のストーリー | 占い垢のストーリー（仮） |
|---|---|---|
| **温度感** | 過激・笑い・失敗ネタOK | 共感・優しさ・神秘性 |
| **主人公** | 弱者・挑戦者・童貞 | 悩める女性・人生迷子・占い師自身 |
| **クライマックス** | 派手な成功/失敗 | 鑑定後の人生変化・運命的な出会い |
| **オチ** | 笑える失敗オチもアリ | 心温まる結末 / 神秘的な啓示 |

→ 占い垢は **共感性 > 過激さ**。占いの神秘性を損なわない範囲で組む。

### 占い垢で作れるストーリーパターン例（仮）

#### パターン1: 自分自身の人生変化ストーリー

```
[Post 1] 「20代の私は人生どん底、毎日泣いてた」
  → 弱者ポジションスタート

[Post 2] 「占いに出会ったきっかけはこんな偶然だった…」
  → 出会いの物語

[Post 3] 「半信半疑で受けた鑑定で言われた一言が衝撃すぎた」
  → クライマックス前の引き

[Post 4] 「あの一言通りに動いたら、本当に人生が変わった【オチ】」
  → 信用の教育を埋め込んだストーリー完結
```

→ 信用の教育（紆余曲折ストーリー）+ ストーリー型のハイブリッド。

#### パターン2: お客様の悩み解決ストーリー（匿名・許可前提）

```
[Post 1] 「30代独身のAさん、結婚相手が見つからず鑑定にきた」
[Post 2] 「彼女のホロスコープを見たら、想像を超える結果が…」
[Post 3] 「過去の恋愛が全てある一つの傾向だった」
[Post 4] 「3ヶ月後、運命の人と出会えたという報告が【オチ】」
```

→ 第三者目線で語ることで、自分への自慢にならず信用が積み上がる。

#### パターン3: 開運メソッド実験企画

```
[Post 1] 「100日後に運命の人と出会う」企画スタート
[Post 2] 「Day 30: 神社参拝を続けたら金運が動き出した気がする」
[Post 3] 「Day 60: 想定外の出会いがあった！運命？」
[Post 4] 「Day 100: 結果報告【オチ】」
```

→ YouTube・TikTokで人気の「100日後に〇〇する」をXに持ってくる戦略。

#### パターン4: 自身の鑑定師としての成長ドキュメント

```
[Post 1] 「OLだった私が占い師になることを決めた日」
[Post 2] 「最初の鑑定、震える手でカードを引いた」
[Post 3] 「3ヶ月で100人鑑定。気づいたある共通点」
[Post 4] 「鑑定で人生が変わったお客様からの手紙【オチ】」
```

→ 占い師としてのバックストーリーを連投で見せる、信用構築型ストーリー。

### 占い垢ストーリー型のNG

- ❌ 過激な失敗オチ（占いの信頼性を損なう）
- ❌ ナンパ・性的内容（消費者層に合わない）
- ❌ 鑑定結果を断定的に保証する内容（特商法・占い倫理）
- ❌ 個人を特定できる事例（プライバシー）
- ❌ 不安を過剰に煽るストーリー（消費者の信頼を失う）

### クライマックス前で切るときの占い系言い回し例

```
✅ 「次の満月でついに動き出す。何が起きるか…」
✅ 「カードが示した結果が衝撃的すぎて言葉が出ない」
✅ 「この鑑定の続き、明日の夜に話します」
✅ 「彼女に伝えた一言、その後の展開が…」
```

## 占い垢のネタ3要素設計（仮）

> 出典: 教材7「ストーリー型運用の組み方とポイント」を占い垢に翻訳（2026-04-27）
> ⚠️ **仮置き**。占い系教材PDFが取り込まれた時点で正式版に更新する。

### 目新しさの作り方（占い垢用）

ビジ垢の「副業マネタイズ」ではなく **占い・スピ系の文脈** で目新しさを作る:

⭕ 機能する目新しさ:
- 「20年無宗教だった私が占い師になるまで」（背景の意外性）
- 「占い師の私が30日間ノー鑑定で運気を上げる実験」
- 「神社100社巡りで人生がどう変わるか全公開」
- 「自分のホロスコープ通りに行動したらどうなるか実験」
- 「〇〇という超マイナー占術で人生変えてみる」

❌ 弱い目新しさ:
- 「タロットの基本を発信します」（普通すぎ）
- 「毎日のラッキーアイテム」（無数の人がやってる）

### 変化率の作り方（占い垢用）

ビジ垢の「収益の差」ではなく **人生・運気・恋愛/金運の差** で変化率を作る。

#### スタート地点を下げる例

| ❌ 弱い | ⭕ 強い |
|---|---|
| 普通の30代女性が占いを学ぶ | 「占い大嫌いだった私」が占い師に |
| 適度に幸せな人が運気アップ | 「人生どん底・うつ寸前」だった私が |
| 軽い恋愛悩みの人 | 「10年片想い・諦めかけてた」女性 |
| 普通の主婦が金運アップ | 「借金300万・自己破産寸前」だった主婦 |

#### ゴールの示し方

- 「半年で運命の人と結婚」
- 「3ヶ月で金運が劇変、ボーナス想定外の額」
- 「人生180度変わった」
- 「鑑定で見えた未来通りに人生が動いた」

### 事件性の作り方（占い垢用）

⚠️ ビジ垢のような **過激な事件性（警察・借金・性病等）はNG**。
占いの神秘性・信頼性を損なわない範囲で組む。

⭕ 機能する事件性:
- 「鑑定中に予言した出来事が翌日リアルに起こった」
- 「神社参拝中に偶然〇〇が起きた」
- 「カードが警告した通りの出来事が…」
- 「お客様が鑑定の3日後に運命の人と出会った」
- 「私が予言した通りに〇〇さんに連絡が来た」

→ **シンクロニシティ**・**予言的中**・**神秘的偶然** を事件性として使う。

❌ 占い垢で使えない事件性:
- 過激なトラブル系（警察沙汰・性的事件）
- 占いそのものの否定的事件（「鑑定が外れた」「効かなかった」）

## 占い垢のピエロ戦略の限界と代替案（仮）

> 出典: 教材7「ピエロ戦略」を占い垢用に翻訳（2026-04-27）

### ⚠️ ピエロ戦略は占い垢では機能しにくい

ビジ垢（コンサル）では「俺アホやけど稼げてるよ」が共感を生むが、占い垢では:

| 観点 | ビジ垢 | 占い垢 |
|---|---|---|
| アホ演出 | ⭕ 再現性訴求になる | ❌ 占い師の権威・神秘性を損なう |
| 弱者ポジ | ⭕ 「俺でもいける」共感 | △ 鑑定能力への信頼を毀損 |
| 笑える失敗 | ⭕ 親近感 | ❌ 鑑定への信頼を損なう |

### 代替戦略: 「等身大の親近感」

占い垢では **権威性は鑑定スキル側に置いたまま**、人間味は別軸で出す:

#### ⭕ 機能する親近感の出し方

- 占い師としての修行時代の苦労話（権威の前段階）
- 自分自身も占いに救われた一人としての姿勢
- お客様への感謝・喜びの素直な表現
- 占いとは関係ない普通の生活感（カフェ・ペット・趣味）
- 鑑定中の悩み・葛藤（「この鑑定結果伝えるのが辛い」等）

#### 言い回しの例

```
✅「私も昔は占いを信じていなかったんです」（過去の弱さ→今の権威への流れ）
✅「鑑定をしていて、たまに自分が見えすぎて怖くなる時があります」（権威+人間味）
✅「お客様が泣いて喜んでくれた時、私も一緒に泣いてしまった」（感情の振れ幅）
```

### 占い垢で使える「感情の振れ幅」

ピエロ戦略は使えないが **感情の振れ幅は使える**:

- 鑑定結果に感動して泣く
- お客様の人生変化を聞いて号泣する
- 神社の不思議体験で鳥肌が止まらない
- 怖い結果が出た時の驚き・葛藤

→ ビジ垢の「ブチギレ・テンション爆上げ」より、**深い感動・驚き・畏怖** の方向で振れ幅を作る。

## 占い垢のロジック構築テンプレ（仮）

占い垢で「鑑定」や「パワーストーン」が欲しくなるロジックを4ステップで組む:

### 鑑定への誘導ロジック

```
[① 共感]
彼が連絡くれない、なんでこんなに辛い思いを…って悩んでる方、
めちゃくちゃ多いんですよね。

[② 真因]
で、こういう悩みって「彼の気持ち」を見ても解決しないんです。
本当の原因は「あなたと彼の魂の相性パターン」にあって、
表面の言動だけ見てたら一生苦しむ。

[③ 解決策]
タロットや西洋占星術で「魂の相性」を読み解けば、
彼の本音と、あなたが今取るべき行動が見えてくる。
これが分かれば、3ヶ月で関係性が劇的に変わる方も多い。

[④ 商品導線]
私の鑑定では、最初に魂の相性パターンを見極めるところから入ります。
今、苦しい時期にいる方、よかったら一度ご相談ください。
```

### パワーストーンへの誘導ロジック

```
[① 共感]
頑張ってるのに金運が上がらない、お給料増えない、貯金できない…
こういう悩み、本当に多いんです。

[② 真因]
実はこれ、「金運の波動」が乱れてるから。
頑張ること自体は悪くないけど、波動が乱れてると
何やっても結果が出にくい状態になってしまいます。

[③ 解決策]
波動を整えるには、自分の星回りに合ったパワーストーンが必要。
人によって合う石が違うので、適当に選ぶと逆効果なことも。

[④ 商品導線]
私のところでは、生年月日・お悩み・今の運気を見て
お一人お一人に合ったストーンを選定しています。
気になる方はDMください。
```

### ⚠️ 占い垢ロジック構築のNG

- ❌ 「絶対叶う」「必ず変わる」等の断定（特商法・占い倫理）
- ❌ 不安を過剰に煽るロジック（「今やらないと一生不幸」等）
- ❌ 医療行為と混同される表現（「病気が治る」等、薬機法）
- ❌ ビジ垢の「収益・稼ぐ」フレームをそのまま流用

→ 占い垢のロジックは **共感性・神秘性・倫理性** のバランスを保つ。

### 占い垢の重要数値メモ

> 教材10「ロジック形成あり vs なし → 売上2倍以上」
> 占い垢でも同じ法則が働くと想定（仮）。

→ 鑑定/パワーストーン販売前は、必ずロジック構築ポストを **複数本** 仕込む。

## 占い垢用 2型ロジック実例（教材13）

> 出典: 教材13「ロジック構築」を占い垢に応用（2026-04-27）
> ⚠️ 教材13はユーザーから「占いでも共通」と明言された教材。
> 本体は `../knowledge_common/education_framework.md` + `../knowledge_common/structure_patterns.md` 参照。

教材13で導入された **課題解決型 + 欲求喚起型** の2型ロジックは、占い垢でもそのまま使える。
ここでは占い垢の主力商品（鑑定 / パワーストーン）販売に応用した実例を保存。

### 課題解決型ロジック（鑑定誘導）

```
[1. 都合のいい問題提起]
彼から連絡が来ない、関係がうまくいかない、
こんな辛い恋愛の悩みありませんか？

[2. 都合のいい原因指摘]
実はその苦しみ、「魂の相性パターン」が原因です。
表面の言動を見てても解決しないし、自己流の対処法じゃ
むしろ運気を下げて状況が悪化することも。

[3. 都合のいい解決策]
タロット鑑定を受けるだけで、根本原因と
今あなたが取るべき行動がはっきり見えてきます。

[4. 理想の未来 - 3層]
直接: 彼の本音と、自分が動くべきタイミングが分かる
その先: 3ヶ月で関係が劇的に変わる、運命の人と出会える
日常: 毎日穏やかな気持ちで過ごせる、自分を好きになれる、
     未来への不安が消える、誰かに依存しない強さが身につく
```

### 課題解決型ロジック（パワーストーン誘導）

```
[1. 都合のいい問題提起]
頑張ってるのにお金が貯まらない、給料も増えない…
こんな金運の悩み、めちゃ多いです。

[2. 都合のいい原因指摘]
実は「金運の波動」が乱れてるのが原因です。
頑張ること自体は悪くないけど、波動が乱れてると何やっても結果が出にくい。
適当に選んだストーンや自己流の開運は逆効果になることも。

[3. 都合のいい解決策]
あなたの星回りに合ったパワーストーンを身につけるだけ。
人によって合う石が違うので、専門家に選定してもらうのが必須。

[4. 理想の未来 - 3層]
直接: 金運の波動が整って、お金の流れがスムーズに
その先: 想定外の収入、ボーナス増、貯金が貯まる
日常: 値段を気にせず買い物できる、家族旅行に行ける、
     老後の不安が消える、お金のストレスから解放される
```

### 欲求喚起型ロジック（鑑定誘導）

```
[1. 理想の未来で欲求喚起]
30代独身で「もう運命の人なんていない」と諦めてた私のお客様。
鑑定の3ヶ月後に運命の人と出会って、半年後に結婚しました。

[2. 要因提示]
これも全部、タロット鑑定で「運命の流れ」を読み解いて
タイミング通りに動いたおかげ。

[3. 都合のいい根拠]
過去100名以上の恋愛鑑定をしてきて、結果を分析すると
「鑑定後にタイミング通り動いた人」の8割が3ヶ月以内に変化を実感。
逆に自己流で動いた人は、何年経っても同じ場所で止まってる。

[4. 結論+アクション]
迷ってる時間は運気を止めます。
鑑定で運命を動かして、自分の人生を取り戻しませんか？
```

### 欲求喚起型ロジック（パワーストーン誘導）

```
[1. 理想の未来で欲求喚起]
お客様の話ですが、半年前まで貯金ゼロで悩んでた40代主婦。
パワーストーンを身につけ始めて3ヶ月で、
ご主人の昇給と臨時収入で50万円以上の好転がありました。

[2. 要因提示]
彼女の星回りに合ったストーンを身につけたことで、
金運の波動が整ったのが大きい。

[3. 都合のいい根拠]
私が選定したパワーストーンを身につけた方の体感では、
3ヶ月以内に8割の方が「お金の流れが変わった」と報告してます。
特に星回りに合ったストーンを選ぶと変化が早い。

[4. 結論+アクション]
金運に悩んでる方こそ、自分に合ったパワーストーンを身につけよう。
波動を整えるだけで、お金の流れが劇的に変わる可能性がある。
```

### 占い垢のロジック構築のNG確認

教材13の共通NG（`../knowledge_common/ng_patterns.md` 参照）に加えて、占い垢特有のNG:

- ❌ 鑑定/パワーストーンの効果を **断定的に保証** する解決策（特商法・占い倫理）
- ❌ 「絶対叶う」「必ず変わる」表現
- ❌ 不安を過剰に煽る問題提起
- ❌ 医療・薬機法に踏み込む表現

→ 占い垢は **共感的な問題提起** + **可能性として語る解決策** + **人生の豊かさレベルの未来** で組む。
