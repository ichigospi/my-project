## 興味付けポスト構造

> 出典: 教材1「興味づけポストの基礎」（2026-04-27 取り込み）

```
[フック] → [興味の対象] → [強化要素 ×2〜3]
```

### フックのカテゴリと例

| カテゴリ | 例 |
|---|---|
| 自分の実績/成果報告 | 「俺が〇〇円稼いだ」「昨日仕入れに行って爆利益品みっけたんだけど」 |
| 経歴/背景 | 「20垢運用してわかった」 |
| 他人の権威 | 「メルカリで1億売り上げてる主婦から聞いた」「noteで月2000万売った友達に教えてもらった」「万垢30個分析してわかったんだけど」 |
| 他人の行動＋成果 | 「23歳のフリーターの弟に教えたら速攻で83万円稼いじゃった」 |
| 自身の行動＋期待感 | 「今裏で仕込んでるジャンルだけど」「ワンチャン月500万くらいいけそうw」 |

> ⚠️ フックは **必ずしも成果が出たものでなくていい**。「うまくいきそう」という期待感だけでも人間は興味を持つ。

### 興味の対象カテゴリ

#### 情報系
- 今アツいボロ儲け市場 / 激アツジャンル / バブルなプラットフォーム
- アルゴリズム変更 / 極秘仕入れ先情報
- 知らないとヤバい落とし穴 / 裏情報 / これから来るトレンド

#### 手法系
- コツ・秘訣がある / ずるい戦略 / 独自のスキーム
- とある施策が… / ノウハウ / 着眼点

#### その他（ツール・資産）
- ツール / 専用AI / 型・テンプレ / シート / サイト

> 💡 ジャンルによって何を欲しがるかは違う。「他の人が何を配ってるか」「何系の情報を欲してるか」を常に観察する。

## 強化要素 14カテゴリ語彙集

> 出典: 教材1（2026-04-27）
> ポスト作成時、ここから2〜3個ピックして組み込む。

### ① シンプルで真似しやすい
- やったのはこの2つだけ / 簡単すぎる / コピペでいける

### ② 他の人が知らない
- 誰もやってない / ライバルがいない / 知ってる人マジでいない
- どの教材にも書いてない / 僕が思いついた / 超ニッチ / ブルーオーシャン

### ③ 一部の人だけが独占してる
- ガチで極秘コミュニティだけで言ったけど / 本当は言いたくないけど

### ④ 再現性がある
- 誰でもできる / 高卒ニートの弟でもできたw / 中学生でもいける
- 猿でもいける / 偏差値30でもいける

### ⑤ 労力がかからない
- 15分〇〇しただけ / ガチで労力かからん

### ⑥ 即効性がある
- 2日でこれ / 1週間もあれば

### ⑦ リスクがない
- 仕入れ4円だけ / スマホだけでいける / 元手ゼロでいける
- ローリスク / 顔出し・声出しもいらない

### ⑧ 優越感
- ライバルが消える / 〇〇に困ることがなくなる
- 〇〇できてしかたがない / 無限に〇〇できる / 〇〇を量産できる

### ⑨ 限定性（情報を入手しにくい）
- 極秘ルートで仕入れた情報 / 1億円稼ぐ大学生から聞いた
- 一部のコミュニティだけで共有されてる / 年5億稼ぐ会社のマニュアルに書いてあった

### ⑩ トレンド性
- AIで〜 / XTEP使って〜

### ⑪ 感情
- エグい / ヤバい / 神じゃね？？

### ⑫ 恐怖心（知らなきゃ損）
- これ知らないとヤバい / 知らないと反応が落ちる

### ⑬ 不明性（何をして？どうやって？）
- とある手法で / あるツールを使うことで / あるサイトを見れば
- ちょっとあることをするだけで / 某アカウントで / 某界隈で

### ⑭ 常識破壊
- エロ・恋愛・ビジネスじゃない（SNSで） / 1ポストもしてない・ローンチしてない
- フォロワーいらない / 仕入れ不要 / 鑑定不要

### ⑮ 新規性（情報が新しい）
- 最新〇〇 / 緊急で / ガチで稼いでる人の中で話題になってる / 今トレンドがきてる

### ⑯ 長期的な利益
- 1年以上ずっと入ってくる / 1回やったら300万円確定

### その他
- 得られる金額がデカい / 成果出るのが異常に早い

## 実例分解テンプレ

> 出典: 教材1（2026-04-27）
> 既存ポストを分析するときの3列分解パターン。

```
本文:
　（ポスト本文をそのまま）

フック:
　・（何が「おっ？」を作っているか）

興味対象:
　・（読者は何が知りたくなったか）

強化要素:
　・（どの心理レバーが効いているか × 2〜3）
```

### 分解例1: ぶんじろう垢のTips販売ポスト

```
本文:
  この垢でTips販売開始して3ヶ月目とかやけど地味に売れ続けるコツみたいなんが
  若干掴めつつある。商品数がゴミやから売上しょぼいけどこの容量でガンガン
  増やしていくか。ぶんじろう垢と違って一撃の爆発力はないけど属人生を
  限りなく落としてるから正味誰でも運用できるからかなり楽。
  （売上画像）

フック:
  ・売上画像
  ・商品増やしたら儲かる（雰囲気）

興味対象:
  ・売れ続けるコツ掴めつつある
  ・属人性のない運用方法

強化要素:
  ・属人性落としてる → 再現できそう
  ・誰でも運用できる → 再現できそう
  ・かなり楽
```

### 分解例2: にきち垢の即日現金化せどりポスト

```
本文:
  今日友達にとある即日現金化できる㊙せどり手法教えたんだけど、
  速攻で利益29047円ゲットしてて草
  内訳: 仕入れ61,000 / 売上90,047 / 利益29,047
  もうこの手法知っちゃったら他の手法なんてめんどくさすぎてやってられないね

フック:
  ・他人の行動＋成果（友達が速攻で利益2.9万円ゲット）

興味対象:
  ・即日現金化できる㊙せどり手法

強化要素:
  ・速攻で（即効性）
  ・㊙（秘密性）
  ・めんどくさすぎてやってられない（優越感・手軽さ）
```

### 分解例3: 新商材2日目ポスト（作成例）

```
本文:
  新商材を仕入れ始めて2日目でこれ。仕入れ値やすいからダメージ少ないし
  回転率もいい。こういうのは情報持ってたら誰でもできるから、
  やっぱ情報ゲーやね。（実績画像）

フック:
  ・新商材を仕入れ始めて2日目でこれ（自身の行動＋成果）

興味対象:
  ・仕入れ商材の情報

強化要素:
  ・仕入れ値が安い（リスク低）
  ・ダメージ少ない・回転率いい
  ・誰でもできる（再現性）
```

## 冒頭一文目の興味付け14手法

> 出典: 教材3「反応がとれる良いポストの5つのポイント」 + 教材5「反応が取れるポストのコツ」（2026-04-27 取り込み）

ポストが伸びる流れ:

```
ポストが目に止まる
  ↓
最後まで読む
  ↓
アクション
```

冒頭一文目で「これ気になる」「自分に関係ある」と思わせないとインプは伸びない。
**影響力が小さいアカウントほど冒頭にこだわる**。
冒頭で惹きつけられるかで **ポストの伸びが7割決まる**（教材5）。

> ⭐ = 教材3で特に重要とされた4つ。最初はこの4つから使う。

### ① 不完全情報

何が起きているか曖昧にして「続きを知りたい」と思わせる。

例:
- 「とある案件で儲かっててヤバい…」
- 「ちょっと聞いてほしい話があるんですが…」

### ② 重要性

「これ、超大事です」と前置きで重みを乗せる。

例:
- 「これ結構ヤバい情報ですが...」
- 「超有益なんで言いたくないんですが…」

### ③ 希少性

「めったに公開しない情報」感を出す。

例:
- 「ガチで誰も知らんノウハウなんですが、実は〜」
- 「教材にも書いてないことを言いますが…」

### ④ 権威性

実績ある人物・組織の名前で信頼を借りる。

例:
- 「アフィで月300万円売ってる友人から聞いたんですが...」
- 「年商10億の経営者がこっそり教えてくれたんやが」

### ⑤ 恐怖・損失回避 ⭐

知らないと損する／このままだとヤバい、というフレーミング。
人は「得する話」より「損を避ける話」に強く反応する。

例:
- 「これ知らないと、ガチで損すると思う」
- 「知らなきゃヤバい話」
- 「マジでビジネス理解してなさすぎ。そりゃ稼げない。」

### ⑥ ターゲット刺し ⭐

階段の特定段（特に0〜1段目）の読者を直接呼びかける。

例:
- 「主婦こそ〇〇すべき理由について話します。」
- 「Xで売上でないから萎えた。」（1段目を狙い撃ち）
- 「サラリーマンの副業はこれ一択。」

### ⑦ 強烈な感情

感情を冒頭に置いて引き込む。

例:
- 「エグすぎるwwww」
- 「ちょっとこの前、おもろいことがあったんですが、」
- 「衝撃すぎて寝れなくなった件について」

### ⑧ パワーワード

普通使わない強烈な単語を冒頭に。

例:
- 「放送事故やらかしました...」
- 「情弱狩りが最高な話聞きます？」
- 「稼ぎたければ大◯吸え」
- 「爆益報告させてください」

### ⑨ 簡易性

「こんな簡単に？」感を出す。

例:
- 「たった2分で〇〇できちゃってビビってます...」
- 「コピペ1回するだけで反応3倍になった話」

### ⑩ 矛盾

直感に反する主張で「えっ？」と立ち止まらせる。

例:
- 「自己投資すればするほど稼げなくなる理由」
- 「真面目な人ほどXで稼げない」

### ⑪ ニュース性 ⭐

「最新」「速報」「トレンド」感を出す。

例:
- 「流石にやらかしたかもしれません...」
- 「【速報】〇〇を発見しちゃいました！」
- 「AIが…」（トレンドテーマ）

### ⑫ 暴露・報告

「こっそり教える」感で読者を共犯者にする。

例:
- 「ちょっと深夜なので暴露しちゃうんですが、」
- 「大事な報告をさせてください」
- 「身バレ覚悟で言うんですが、」

### ⑬ 限定性

「今しか読めない」感で焦らせる。

例:
- 「後ですぐ消す」
- 「24時間で消すので保存推奨」

### ⑭ 反社会性

界隈のタブー・グレーに踏み込んで注目を集める。

例:
- 「批判覚悟で言うのですが」
- 「正味、かなりグレーなんだけど」
- 「禁断のノウハウですが…」

### 14手法の組み合わせ

複数手法を1ポストの冒頭で組み合わせると強化される。

```
例: 「【AIが】今すぐ読め。絶対。【知らなきゃヤバい話】」
  - ⑪ ニュース性（AIが）
  - ⑦ 強烈な感情 + ⑧ パワーワード（今すぐ読め。絶対。）
  - ⑤ 損失回避（知らなきゃヤバい話）
```

```
例: 「ちょっと深夜なので暴露しちゃうんですが、ガチで誰も知らんノウハウあって…」
  - ⑫ 暴露・報告
  - ③ 希少性
  - ① 不完全情報
```

### 14手法と強化要素14分類の関係

冒頭14手法は強化要素14分類と一部重なる（同じ心理レバーを冒頭で使うか本文で使うか）:

| 冒頭手法 | 強化要素分類 |
|---|---|
| ③ 希少性 | ② 他の人が知らない / ③ 一部の人だけが独占 |
| ④ 権威性 | ⑨ 限定性（情報を入手しにくい） |
| ⑤ 損失回避 | ⑫ 恐怖心（知らなきゃ損） |
| ⑦ 強烈な感情 | ⑪ 感情 |
| ⑨ 簡易性 | ① シンプルで真似しやすい / ⑤ 労力かからない |
| ⑪ ニュース性 | ⑩ トレンド性 / ⑮ 新規性 |
| ⑫ 暴露・報告 | ⑬ 不明性（とある〜） |
| ⑭ 反社会性 | ⑭ 常識破壊 |

→ 強化要素14分類は **本文の体力**、冒頭14手法は **入口の引っかかり**。役割が違うので両方使う。

### コツ

- 伸びてる人の冒頭をひたすら **真似てストック**
- 「言いたいこと」をそのまま書かず、必ず冒頭にフックを噛ませる
- 小手先と思わずちゃんとやる（伸びてない人は冒頭意識がない）

## Xポストの構成原則: 結論はラスト

> 出典: 教材5「反応が取れるポストのコツ」（2026-04-27）

### PREP法はXでは最悪

ライティング教科書で定番のPREP法（結論→理由→具体例→結論）は **Xでは最悪**。

```
PREP法:
  Point   ← 主張（結論）
  Reason  ← 理由
  Example ← 具体例
  Point   ← 再度主張（まとめ）
```

**理由**: 最初に結論を言うと **続きが気にならなくなって満足される** → 滞在時間が伸びない → アルゴで弱い。

### Xは結論ラスト構造

```
冒頭フック（14手法）
  ↓
背景・興味付け・ストーリー
  ↓
モチベ上げ・自己啓発
  ↓
理由・具体例（最大まで膨らませる）
  ↓
ある程度引き延ばしてから
  ↓
満を持して結論
```

→ 結論を引き延ばすことで滞在時間を稼ぐ。

## 滞在時間を伸ばす（長文化テクニック）

> 出典: 教材5（2026-04-27）

滞在時間が長いほどアルゴ優遇。**たまに長文を挟む**（全部だとウザい）。

### 長文化のアイデア10種

1. **冒頭に強い興味付けを入れる**（14手法）
2. **ストーリーを入れる**（実話・コンサル生事例 等）
3. **感情をリアルに具体的に書く**（「マジでビビった」「鳥肌立った」）
4. **情報の具体度を上げる**（数字・固有名詞・場面描写）
5. **理由を掘り下げる**（なぜそれがダメか / なぜ効くか）
6. **モチベ上げを入れる**（自己啓発要素 — 後述30要素）
7. **メリットを語る**（手段の教育の魅力）
8. **恐怖を煽る**（問題点の教育の放置リスク）
9. **対比を入れる**（よくある人 vs 賢い人）
10. **読者の知能に合わせて噛み砕く**（無駄に親切に書く）

### 例: 短文 → 膨らませる

```
❌ 短文（薄い）:
「ノウハウポストは無意味です」

⭕ 膨らませた版:
「『ノウハウをポストしましょう！🤪』という発信者はパチモンです。
 そもそもノウハウだけ垂れ流しても誰も読まないし、
 教育要素のないポストは反応も来ないから完全に時間の無駄。
 むしろ目的の教育や信用の教育を混ぜないと永遠に伸びません。」
```

→ 倍以上の長さになり、反応したくなる表現に変わる。

### 重要原則: レベル高い人ほど端的に書きがち

レベルが高い人は **本質だけ端的に書く** 癖がある。
→ 初心者には届かない。
→ あえて **無駄にわかりやすく**、読者の知能に合わせて掘り下げる。

## モチベ上げ30要素リスト（自己啓発フレーズ集）

> 出典: 教材5（2026-04-27）
> 文中に2〜3個入れるだけで反応率が大幅に上がる。
> 思いつかない時はChatGPTに「このポストをモチベが上がるように膨らませて書き換えて」と依頼するのも有効。

### 1. 可能性を肯定する
- 読者の成長余地や才能を認める
- 「まだ始めてないだけ」「やればできる」と伝える

### 2. 行動の価値を強調する
- 行動した瞬間に未来が変わる
- 小さな一歩の意味を大きくする

### 3. 継続の力を伝える
- 才能より習慣、継続が勝つ
- 毎日の積み重ねが未来を作る

### 4. 未熟さの肯定
- 最初は誰でも下手
- 失敗は成長の証

### 5. 比較優位を示す
- 行動してるだけで上位層
- 悩んでる時点で進んでる

### 6. 今すぐ始める理由を与える
- 今日の行動が未来を決める
- 後回しにすると損をする

### 7. 成果のタイムラグを意識させる
- 半年後に振り返ってよかったと思える
- 今の努力は未来で花開く

### 8. 常識を逆転させる
- 「やる気が出たらやる」ではなく「やるからやる気が出る」
- リスクは行動しない方が大きい

### 9. 未来の後悔を想像させる
- 挑戦しなかった後悔は一生残る
- 「あの時やっておけば」とならないために動く

### 10. 自分ごと化させる
- 読んでるあなたはすでに特別
- 行動できる人だけがここにいる

### 11. 小さな成功体験を意識させる
- 小さな一歩で十分
- 成功は最初の行動から始まる

### 12. 成長をプロセスとして捉えさせる
- 完璧より未完成の行動
- 過程が実力を作る

### 13. 自己肯定感を高める
- 悩めるのは前進の証拠
- 迷うことは可能性を探してるサイン

### 14. 不安を逆手に取る
- 不安がある＝挑戦してる証
- ドキドキは伸びてるサイン

### 15. 読者を「挑戦者」扱いする
- 行動してるだけで他より前にいる
- あなたはすでに勝負の舞台に立っている

### 16. シンプルな真理を突く
- 動けば変わる
- 習慣が人生を作る

### 17. 背中を押す問いかけ
- 半年後にどうなっていたい？
- 今日の一歩を未来の自分は喜ぶ？

### 18. 勇気の定義を与える
- 恐怖があっても踏み出すのが勇気
- 小さな挑戦こそ勇気の証

### 19. 可能性の拡大を見せる
- 今の一歩で可能性が何倍にも広がる
- 今日動けば未来の選択肢が増える

### 20. 周囲との差を意識させる
- 動いた人だけが景色を変えられる
- 大多数は動かない。だから差がつく

### 21. 成果のシンプル法則
- 習慣＞才能
- 継続＞瞬発力

### 22. 一歩先の未来像を描かせる
- 来月には見える景色が変わる
- 数ヶ月後のあなたは今を感謝する

### 23. 自分の選択を正当化させる
- 行動したあなたは正しい
- 悩んでも動いた人だけが進める

### 24. やめないことの強さ
- 諦めない人が最終的に勝つ
- やめないだけで才能になる

### 25. シンプルな行動促進
- 今すぐやれ
- 後でじゃなく、今

### 26. 逆説的に肯定する
- 「できない」と思う時点で伸びしろがある
- 不安がある＝挑戦してる証拠

### 27. 成功のハードルを下げる
- まずは小さな行動でOK
- 完璧は必要ない

### 28. 長期目線の安心感
- 1年続ければ誰でも変わる
- 時間は味方になる

### 29. 未来の自分に語りかける
- 今日の自分を未来の自分が誇る
- 未来の自分が感謝する行動をしよう

### 30. 読者を選ばれた存在にする
- このポストに出会った時点で特別

### 自分ジャンルの自己啓発を見つけるコツ

- **書籍で発見**: 自分のジャンルの書籍を本屋で買って読む。書籍は内容を引き延ばすために自己啓発を散りばめている。
- **GPTに依頼**: 「このポストをモチベが上がるように膨らませて書き換えて」

## 起承転結型連投構成（ストーリー型運用）

> 出典: 教材6「ストーリー型運用」 + 教材7「ストーリー型運用の組み方とポイント」（2026-04-27 取り込み）
> 全体像は `story_operation.md` 参照。

ストーリー型運用では **複数のポストを連投で繋げて1つの物語** を作る。
基本骨格は **起承転結**:

```
①起：宣言「〇〇をします！」
   ↓
②承：途中経過・期待感を高める・死亡フラグ
   ↓
③転：ピンチ or チャンス（メイン事件）
   ↓
④結：失敗 or 成功 + オチ
```

### 各段階の詳細

#### ①起 — 物語の始まり / 宣言

エピソードの開始を宣言する。

```
✅ 例: 「浪人界隈の囲われ姫とエンカ（リアルで会う）してホテルに行きます」
✅ 例: 「ネカフェ連れ込み成功！これから童貞卒業するぞ！」
```

→ 「これから物語が進む」感を出す。

#### ②承 — 途中経過 / 死亡フラグ

途中経過をポストすると、読者が自然と応援したくなる。

```
✅ 例: 「ホテル到着、緊張してる」
✅ 例: 「マジでこれから童貞卒業します」
✅ 例: 「絶対いける！俺の勝ちや！」
```

##### 死亡フラグの活用

確信に満ちたポストをあえて入れることで、読者に「あ、こいつヤバいかもな…」と期待感が生まれる（バトルアニメで「俺を残して先に行け！」と言うやつはだいたい死ぬ）。

#### ③転 — ピンチ or チャンス（メイン事件）

物語のメイン事件。状況の上振れ／下振れ。

```
✅ 失敗系: 「シャワー浴びてる間に逃げられた…」
✅ 成功系: 「俺の勝ち！！！！童貞卒業！！！きたぞ！！！！」
```

#### ④結 — オチ

成功談 / 失敗談 / 後日談でオチをつける。

```
✅ 失敗 + 二段落ち: 「ブロックされた」
✅ 成功 + 派手なオチ: 「性病感染の可能性…」
```

##### ハッピーエンドのまま終わらせない

```
❌ ハッピーエンドだけで終わる
⭕ どこかで落としてネタ要素を入れる
⭕ ただ伸ばしたい場合はバッドエンドも有効
```

ただし **商材を売る場合** は成功体験（理想状態）に繋げる必要があるので、最低限の実績にはたどり着かせる。

### 各ポストの切り方の鉄則

ドラマ・アニメと同じく **クライマックス直前で毎回終わる** ように設計する。

```
✅ 「ホテル来たけどやり方わかんねぇ…教えてくれ！これ逃げられたら埋めてくれ…」
   → 続きが死ぬほど気になる

❌ 「ナンパしてホテル行ったけど逃げられました。来週も頑張ります。」
   → 完結してしまって読者は離れる
```

### ストーリーの5要素（構成要素チェックリスト）

連投シリーズを設計する時に必ず含める:

- ✅ **挑戦の対象**（何を達成しようとしているか明確）
- ✅ **障害・困難**（うまくいかない要素）
- ✅ **感情の起伏**（喜び・絶望・焦りをリアル描写）
- ✅ **クライマックス**（成功 / 派手な失敗 / 意外なオチ）
- ✅ **連投で引っ張る切り方**（毎回続きが気になる地点で終える）

### ネタの強さ（事前設計）

連投構成の前に、ネタの強さを確認する3要素（詳細は `story_operation.md`）:

- ✅ **目新しさ** — 他人がやってない / やれない / やりたくない
- ✅ **変化率** — スタートとゴールの差（実績小ならスタートを下げる）
- ✅ **事件性** — 予期しないトラブル

### 実例: ぶんじろう垢の起承転結分析

教材6の3パターンを起承転結に当てはめると:

```
[パターン1] 浪人界隈時代（失敗オチ）
  起: 浪人界隈の姫とエンカでホテル
  承: ホテル到着、緊張・期待
  転: シャワー浴びたら逃げられてる
  結: ブロックされる（二段落ち）

[パターン2] 駆け出しナンパ師時代（失敗オチ）
  起: 女の子と宅飲み！童貞卒業の大チャンス
  承: 期待感ポスト
  転: 途中までいったのに追い出される
  結: 鬼ディスられる

[パターン3] ナンパ成功 → 派手オチ
  起: ネカフェ連れ込み成功
  承: 大チャンス！応援よろしく！
  転: 童貞卒業！俺の勝ち！
  結: 性病感染の可能性…
```

→ 失敗・派手なオチでも十分バズる。

### ⚠️ テンプレ通りに動かそうとしない

教材執筆者自身も「細かいシナリオは事前に決めず、その場その場でやる」と明言。
**起承転結はあくまで事後分析の骨格** であって、リアルタイムは出来事に応じて柔軟に。

100%テンプレ通りのストーリー型運用は **そもそも不可能**。

## ジャブ投稿パターン

> 出典: 教材7「ストーリー型運用の組み方とポイント」（2026-04-27 取り込み）
> ストーリー型運用のリアルタイム感を強化するための補助ポスト。

### ジャブ投稿とは

ストーリーの本流ポスト（起承転結）の **間に挟む短い投稿**。
単体では意味がないが、リアルタイム感を強化し、次のポストへの **布石** になる。

### 例

```
「ちょ、、、ヤバいこと起こった...」
「ショックすぎて気絶してました...夜にまた話します。」
「今から楽しいことしちゃうよーん♪」
「田中まじでこいつ終わりだろ」
「明日大事なポストするので寝ずに待っといてください」
```

### 機能する理由

- **「え、何があったの？！」** と反射的に気にさせる → マインドシェア
- 本流ポストへの **期待感** を高める
- **リアルタイムで何かが進行している感** を演出

### 配置タイミング

```
[起] → [ジャブ] → [承] → [ジャブ] → [転] → [結]
       ↑                ↑
       期待を煽る        次の事件を予感させる
```

### 注意

ジャブを入れるか入れないかで反応が **全然変わる**。
ストーリー型運用時は **意識的に入れる**。

## インセンティブ系ポストパターン

> 出典: 教材8「インセンティブ設計」（2026-04-27 取り込み）
> 全体像は `incentive_design.md` 参照。

### ポスト即消しパターン（通知オン誘導）

レアな情報をポストして即削除することで、通知オン勢以外に **後悔と好奇心** を植え付け、通知オン化を促進する。

#### テンプレ
```
「ポスト消した。通知オンしてポスト見れた人はラッキー」
「久々にポスト消した。通知オン勢は見れたよね」
「悪いポストを1つ消した。見れた人はラッキー」
```

#### 機能の流れ
```
ポスト即消し
  ↓
通知オン勢「見れた！」（特典型インセンティブ獲得）
通知オフ勢「えー！見れなかった！」（損失回避型インセンティブの不発）
  ↓
通知オフ勢が後悔・好奇心を感じる
  ↓
「次は見逃さないように」通知オンする
```

#### コツ
- **直接「通知オンしてください」と言わない**
- 通知オン勢のリプを残して **追従効果** を最大化
- 定期的に繰り返す（1回では行動変容しない）

### 感想特典ポストパターン

```
「コンテンツの感想書いてくれた方に〇〇プレゼント」
```

→ 「感想書いて！」と直接お願いするより圧倒的に感想が回る。
→ 表面パクリで終わらせず、原理（インセンティブ設計）を理解して他のアクションにも応用する。

### 伏線ジャブポストパターン（きあいだめ理論）

ストーリー型運用やローンチ前に、**単体では意味がないが次のポストの威力を上げる** ジャブを入れる。

#### テンプレ
```
「最悪な出来事が起こりました、、、」
「これまでの努力が無駄になったんやが、、、」
「ヤバいこと起こった...」
「ショックすぎて気絶してました...夜にまた話します」
「明日大事なポストするので寝ずに待っといてください」
```

#### 機能の流れ
```
ジャブポスト「最悪な出来事…」← きあいだめ
  ↓
読者「え、何があったの？！」← 注目度MAX
  ↓
本命ポスト（教育・セールス・新展開）← 急所ヒット
```

#### ジャブポスト vs ストーリー型運用のジャブ投稿
- どちらも本質は同じ（次の威力を上げる）
- 構造的位置づけが違う:
  - **ストーリー型運用のジャブ**: 起承転結の **間** に挟む（リアルタイム感を強化）
  - **インセンティブ設計のジャブ**: 単一の本命ポストの **直前** に置く（注目を集めて急所ヒット）

→ 両方とも「次の威力を上げる」ためのインセンティブ設計の一形態。

### ローンチ伏線回収パターン

ローンチを意識して **何ヶ月も前から** 各アクションのインセンティブを仕込む:

```
[Pre-launch -3M] 通知オン化（即消し戦略を定期実施）
[Pre-launch -2M] 感想特典で感想を集めて事例化
[Pre-launch -1M] スペースで参加者限定情報の文化を作る
[Launch Day] 全インセンティブが同時発火 → 最大火力
```

詳細は `incentive_design.md` の「商品ローンチへの伏線回収設計」参照。

## 「ためてためて放出」連鎖パターン

> 出典: 教材9「インセンティブ設計の具体的方法」（2026-04-27 取り込み）
> 教材8の「きあいだめ理論」の発展形。複数のジャブを連鎖させて本命の威力を最大化する。

### 5ポスト連鎖構造

```
[ポスト1] 捨てジャブ ← 伏線（教育意味なし）
   ↓
[ポスト2] 捨てジャブ ← さらに事件性を煽る
   ↓
[ポスト3] ジャブ的反応 ← 部分的な伸び
   ↓
[ポスト4] 補助技 ← さらなる事件性
   ↓
[ポスト5] 本命ポスト ← 大爆発
```

### 実例: 弁護士事件→特大ブチギレポスト（掌握ローンチ前）

| ポスト | 内容 | 役割 |
|---|---|---|
| 1 | 弁護士案件をほのめかす | 伏線 |
| 2 | さらに事件性を煽る | 伏線 |
| 3 | ジャブ的反応 → 5万インプ | 部分発火 |
| 4 | 「ブチギレポスト書いてたら数千文字消えた」 | さらなる事件性 |
| 5 | 特大ブチギレポスト → 18.6万インプ / 143RT / 165リプ / 538いいね | 本命爆発 |

詳細・本文は `../knowledge_business/teachings.md` の「弁護士事件→ブチギレチェーン」参照。

### コツ

- 各ポストは **教育もセールスもしていない**
- でも全体として **本命ポストの威力を最大化** する
- 事件性・ニュース性・感情の動き があるほど注目を引きやすい
- リアルタイムで起きた出来事を **時系列でポストする** とストーリー型にもなる

## 誘導ポストパターン集

> 出典: 教材9（2026-04-27）

特定の行動を引き出すための短いポストテンプレ集。

### プロフクリック誘導

```
「プロフ地味に変えたけど、何が変わったでしょうか？わかる人えぐいw」
「フォロワー数の壁を越えました（具体数字を言わない）」
「さっきのポストでえぐい情報書いちゃったけど…」
```
→ **興味性** で誘発。

### ポスト遡及誘導

```
「俺の直近のポスト見てくれたら、Xのアルゴ全部わかるw」（情報興味）
「最近ポスト遡りまくって全部いいねくれる人いるけど、こういう人が伸びるよなw」（承認欲求）
```
→ **情報興味 + 承認欲求** で誘発。

### 特定テーマへの興味喚起

```
「あれ、アルゴ変わった？？？」
```
→ シンプルな問いかけだけでテーマへの注目を引ける。

### メルマガ・LINE登録誘導

```
「メルマガ見てる人ならわかるけど〜」
「昨日メルマガで流した極秘案件だけど、もう稼いでる人おるわw」
「LINE登録者だけに今月の極秘情報出した」
```
→ **限定情報の存在を匂わせる** ことで、お願いせずに登録させる。

### スペース参加誘導（事前興味付け）

```
「今日のスペースで、やばいこと話しちゃいます...」
「リアルタイム参加じゃないと聞けないネタあるよ」
「録音残さないので、聞きたい人は来て」
```

### 当事者意識醸成（タイトル/価格/特典募集）

```
「タイトル募集、採用された人に〇〇プレゼント」
「価格決め投票します、A案 / B案 / C案」
「次の特典は何にする？リプで教えて」
```
→ **読者をローンチ準備に巻き込む** ことで購入率を上げる。

## ロジック構築型教育ポストパターン（2型）

> 出典: 教材10「教育とは何か」 + 教材13「ロジック構築」（2026-04-27 取り込み）
> 詳細・原則は `education_framework.md` の「ステップ2: ロジック構築」参照。

ロジックの組み方は無限にあるが、特に使いやすいのが2つ:

| 型 | 使い分け |
|---|---|
| **① 課題解決型** | 悩みが具体的にある人に使いやすい |
| **② 欲求喚起型** | ゼロから需要を生み出したい時に使う |

両方とも `4ステップ` で構成される。

---

### ① 課題解決型ロジック（4ステップ）

```
1. 都合のいいテーマで問題提起
   ↓
2. 都合のいい原因を指摘
   ↓
3. 都合のいい解決策を提示
   ↓
4. 理想の未来を見せる
```

#### 各ステップのテンプレ

##### 1. 都合のいいテーマで問題提起

悩み訴求 or 欲求訴求のいずれかで自分ゴト化させる:

```
[悩み訴求]
✅「ポストしてるけどインプが取れない…」
✅「いいね、リプが増えない…」
✅「フォロワーが減ってきてる…」

[欲求訴求]
✅「ポストした瞬間に50いいね」
✅「1日10分でスマホ1台で集客できちゃう」
✅「ほぼ労力ゼロでフォロワー1000人」
```

→ 自分の売りたい商品で解決できる悩み or 欲求を選ぶ（「都合のいい」原則）。

##### 2. 都合のいい原因を指摘

```
あなたがうまくいってないのは、[原因]ができていないから。
[原因]さえ解決すれば、あの問題もこの問題も解決して、困ることがなくなります。
```

→ 自分の商品に都合のいい原因を選ぶ。
→ 例: アルゴ攻略教材売るなら、原因は「アルゴ理解不足」と指摘。

##### 3. 都合のいい解決策を提示

```
[原因]を解決するのには、[解決策]をしましょう。
```

ポイント: **シンプルに、考えさせない**:

```
✅「〜するだけ！」
✅「〜さえすればOK！」
✅「これだけやっておけば大丈夫」
```

##### 4. 理想の未来を見せる

ベネフィットを **3層** で展開:

```
[直接メリット]
解決策をやって直接得られる結果

[その先のメリット]
収益・成果のスケール拡大

[日常生活レベル]
日々の生活がどう変わるか
```

#### 実例（X運用アルゴ攻略教材販売）

```
[1. 問題提起]
ポストしてもインプ全然伸びないって人、めちゃ多いですよね。
いいねもリプも数件で止まる。フォロワーすら減ってきてる。

[2. 原因指摘]
インプが伸びない原因は、シンプルにアルゴの理解不足です。
最近、Xはアルゴリズムが変わりました。
以前伸びてた人でも、このアルゴ変更に気付けてない人は軒並みインプが落ちてる。

[3. 解決策]
最新アルゴに合わせたポスト戦略に切り替えるだけ。
具体的には〇〇を意識すればOK。

[4. 理想の未来]
直接: ポストの初速で50いいね、インプ1万超え
その先: 月10万、50万、100万のローンチが余裕に
日常: 寝てる間にも集客できて、好きな時に旅行行ける
```

#### 実例（マッチングアプリ写真撮影サービス、教材10由来）

```
[1. 問題提起]
マッチングアプリで全然彼女できないって人、多いですよね。

[2. 原因指摘]
みんな「コミュ」「デートコース」勉強するけど、本当の原因は「マッチしない」ことで、
プロフ画像の1枚目で決まってる。

[3. 解決策]
プロフ画像を「イケメン風」にするだけ。雰囲気イケメンでOK。

[4. 理想の未来]
直接: マッチが激増する
その先: ちょいブスでも美女とデート行きまくれる
日常: 毎週違う女の子と出会える生活
→（撮影サービスのオファー）
```

### ② 欲求喚起型ロジック（4ステップ）

```
1. 理想の未来を見せて欲求喚起
   ↓
2. 要因（=売りたいもの）を提示する
   ↓
3. 都合のいい根拠で固める
   ↓
4. 結論でアクションを提示する
```

#### 各ステップのテンプレ

##### 1. 理想の未来を見せて欲求喚起

```
✅ 自分はこんな実績を持っています
✅ こんな最高な生活を送っています
✅ こんな成果を得ています
```

##### 2. 要因（=売りたいもの）を提示

```
これができるのは [解決策] のおかげです。
```

→ シンプルに「〇〇さえ学んどけばOK」と1点に絞る。

##### 3. 都合のいい根拠で固める

```
✅ 自分の体験談
✅ 第三者の事例
✅ 具体例を挙げる
✅ 有名人の名言を引用
✅ 歴史から引っ張る
✅ それっぽい理論を使う
```

→ 納得感があれば何でもOK。

##### 4. 結論でアクションを提示する

```
[ 解決策 ] ができると 〇〇も××できちゃうので [ 解決策 ] しよう！
```

→ ベネフィットを添えると教育パワーUP。

#### 実例（教材を売るポスト）

```
[1. 欲求喚起]
コンテンツ販売初心者の頃は1回商品売ると月300万円くらいだったが、
今じゃ月1000万円超えは当たり前。

[2. 要因提示]
タネ明かしすると、これも全部教育を学んだおかげ。

[3. 都合のいい根拠]
集客は昔も上手かったから特にビジネス学んでなくても月300出てたけど、
教育能力は皆無だったからここ学んで売上の桁が1桁跳ねた。
去年の9月は3400万売れてるから数字にして約10倍。
ローンチ時に抱えてたフォロワー数はほぼ同じ。教育力の差で売上10倍。

[4. 結論+アクション]
元の額がある程度大きいから分かりにくいけど、
月10万円の人なら教育学べば月100万円いけるということ。
50万円なら500万円。それくらいの破壊力がある。
```

## 「読む・見る教育」型コンテンツ冒頭テンプレ

> 出典: 教材12「6つの強化教育」（2026-04-27 取り込み）
> 記事・動画・スペースを **最後まで見させる** ための冒頭テンプレ。
> フレームワーク本体は `education_framework.md` の「⑦ 読む・見る教育」参照。

### 基本構造

```
①これから話す内容のメリット
   ↓
②何が学べるか（興味付き）
   ↓
③重要性の演出（「メモを促す」「すぐ消す」等）
```

### スペース冒頭テンプレ

```
✅「これ録音残さないんで、メモ取っといてください」
✅「今日話す内容、有益すぎるんで使ってもらえたらやばいことできます」
✅「中盤で〇〇についてポロッと話します。聞き逃さないで」
✅「最後まで聞いた人だけにXXX教えます」
```

### 記事/note冒頭テンプレ

```
✅「この記事を最後まで読むと、〇〇が分かります」
✅「重要なので必ずメモを取りながら読んでください」
✅「公開期間は1週間、その後は消します」
✅「中盤で『公開してない裏情報』をポロッと書きます」
```

### 動画/YouTube冒頭テンプレ

```
✅「最後まで見ると〇〇のテンプレが手に入ります」
✅「概要欄の特典は最後に説明するので飛ばさないで」
✅「途中で答えを言うので、自分で考えながら見てください」
```

### 効果

- 離脱率が下がる
- 集中して見るので教育がかかる
- 動機づけがあるかないかで反応率が **全然変わる**

## ぶっ飛び体験談ストーリーフォーマット

> 出典: 教材12「6つの強化教育」（2026-04-27 取り込み）
> **基準値の教育**（強化教育⑪）と **覚悟の教育**（強化教育⑫）で使う体験談の型。

### 構造

```
①ぶっ飛んだ行動の宣言
   ↓
②周りの反応・苦労・障害
   ↓
③そこを乗り越えた様子
   ↓
④結果としての成果
   ↓
⑤暗黙のメッセージ「これが普通」
```

### テンプレ

```
[① 宣言]
〇〇するために、△△しました。

[② 苦労]
当時は周りに□□と言われ、貯金は◇◇円、状況は最悪でした。

[③ 乗り越え]
でも、〇〇するためには□□は必要だと分かっていたので、
全部捨てて△△しました。

[④ 成果]
結果、××で成果を出せました。

[⑤ 暗黙のメッセージ]
（これくらいやって当然、という顔で語る）
```

### 体験談の種類（教材執筆者の実例）

```
✅ 友達のLINE全員ブロックして貯金ゼロ円で上京、ナンパ師になった話
✅ 貯金0円のニート時代に、コンサル受けるために寝ずに1週間バイトで資金を捻出した話
✅ 親に800万円借金して自己投資した話
✅ ナンパ初心者で、ノウハウゼロで1日100人声かけして初月30即した話
✅ 毎日1万文字書くのは当然という話
✅ 事業拡大のために毎月300〜1000万円使ってる話
✅ 投資チャンスのために2000万円借りた話
✅ 1時間50万円のメンターに払った話
```

### 重要なトーン

```
「これくらいやって当然」という顔で淡々と語る。
ドヤらない。
「普通」のように語るのが最大効果。
```

→ ドヤると胡散臭く見える。淡々と語ると **「これが成功者の普通か…」** と読者の基準値がバグる。

### 占い垢での代替方向

ナンパ系・派手な金額系は占い垢では機能しない。
詳細・代替パターンは `../knowledge_spiritual/teachings.md` 参照。
