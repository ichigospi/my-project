## 興味付けポスト構造

> 出典: 教材1「興味づけポストの基礎」（2026-04-27 取り込み）

```
[フック] → [興味の対象] → [強化要素 ×2〜3]
```

### フックのカテゴリと例

| カテゴリ | 例 |
|---|---|
| 自分の実績/成果報告 | 「俺が〇〇円稼いだ」「昨日仕入れに行って爆利益品みっけたんだけど」 |
| 経歴/背景 | 「20垢運用してわかった」 |
| 他人の権威 | 「メルカリで1億売り上げてる主婦から聞いた」「noteで月2000万売った友達に教えてもらった」「万垢30個分析してわかったんだけど」 |
| 他人の行動＋成果 | 「23歳のフリーターの弟に教えたら速攻で83万円稼いじゃった」 |
| 自身の行動＋期待感 | 「今裏で仕込んでるジャンルだけど」「ワンチャン月500万くらいいけそうw」 |

> ⚠️ フックは **必ずしも成果が出たものでなくていい**。「うまくいきそう」という期待感だけでも人間は興味を持つ。

### 興味の対象カテゴリ

#### 情報系
- 今アツいボロ儲け市場 / 激アツジャンル / バブルなプラットフォーム
- アルゴリズム変更 / 極秘仕入れ先情報
- 知らないとヤバい落とし穴 / 裏情報 / これから来るトレンド

#### 手法系
- コツ・秘訣がある / ずるい戦略 / 独自のスキーム
- とある施策が… / ノウハウ / 着眼点

#### その他（ツール・資産）
- ツール / 専用AI / 型・テンプレ / シート / サイト

> 💡 ジャンルによって何を欲しがるかは違う。「他の人が何を配ってるか」「何系の情報を欲してるか」を常に観察する。

## 強化要素 14カテゴリ語彙集

> 出典: 教材1（2026-04-27）
> ポスト作成時、ここから2〜3個ピックして組み込む。

### ① シンプルで真似しやすい
- やったのはこの2つだけ / 簡単すぎる / コピペでいける

### ② 他の人が知らない
- 誰もやってない / ライバルがいない / 知ってる人マジでいない
- どの教材にも書いてない / 僕が思いついた / 超ニッチ / ブルーオーシャン

### ③ 一部の人だけが独占してる
- ガチで極秘コミュニティだけで言ったけど / 本当は言いたくないけど

### ④ 再現性がある
- 誰でもできる / 高卒ニートの弟でもできたw / 中学生でもいける
- 猿でもいける / 偏差値30でもいける

### ⑤ 労力がかからない
- 15分〇〇しただけ / ガチで労力かからん

### ⑥ 即効性がある
- 2日でこれ / 1週間もあれば

### ⑦ リスクがない
- 仕入れ4円だけ / スマホだけでいける / 元手ゼロでいける
- ローリスク / 顔出し・声出しもいらない

### ⑧ 優越感
- ライバルが消える / 〇〇に困ることがなくなる
- 〇〇できてしかたがない / 無限に〇〇できる / 〇〇を量産できる

### ⑨ 限定性（情報を入手しにくい）
- 極秘ルートで仕入れた情報 / 1億円稼ぐ大学生から聞いた
- 一部のコミュニティだけで共有されてる / 年5億稼ぐ会社のマニュアルに書いてあった

### ⑩ トレンド性
- AIで〜 / XTEP使って〜

### ⑪ 感情
- エグい / ヤバい / 神じゃね？？

### ⑫ 恐怖心（知らなきゃ損）
- これ知らないとヤバい / 知らないと反応が落ちる

### ⑬ 不明性（何をして？どうやって？）
- とある手法で / あるツールを使うことで / あるサイトを見れば
- ちょっとあることをするだけで / 某アカウントで / 某界隈で

### ⑭ 常識破壊
- エロ・恋愛・ビジネスじゃない（SNSで） / 1ポストもしてない・ローンチしてない
- フォロワーいらない / 仕入れ不要 / 鑑定不要

### ⑮ 新規性（情報が新しい）
- 最新〇〇 / 緊急で / ガチで稼いでる人の中で話題になってる / 今トレンドがきてる

### ⑯ 長期的な利益
- 1年以上ずっと入ってくる / 1回やったら300万円確定

### その他
- 得られる金額がデカい / 成果出るのが異常に早い

## 実例分解テンプレ

> 出典: 教材1（2026-04-27）
> 既存ポストを分析するときの3列分解パターン。

```
本文:
　（ポスト本文をそのまま）

フック:
　・（何が「おっ？」を作っているか）

興味対象:
　・（読者は何が知りたくなったか）

強化要素:
　・（どの心理レバーが効いているか × 2〜3）
```

### 分解例1: ぶんじろう垢のTips販売ポスト

```
本文:
  この垢でTips販売開始して3ヶ月目とかやけど地味に売れ続けるコツみたいなんが
  若干掴めつつある。商品数がゴミやから売上しょぼいけどこの容量でガンガン
  増やしていくか。ぶんじろう垢と違って一撃の爆発力はないけど属人生を
  限りなく落としてるから正味誰でも運用できるからかなり楽。
  （売上画像）

フック:
  ・売上画像
  ・商品増やしたら儲かる（雰囲気）

興味対象:
  ・売れ続けるコツ掴めつつある
  ・属人性のない運用方法

強化要素:
  ・属人性落としてる → 再現できそう
  ・誰でも運用できる → 再現できそう
  ・かなり楽
```

### 分解例2: にきち垢の即日現金化せどりポスト

```
本文:
  今日友達にとある即日現金化できる㊙せどり手法教えたんだけど、
  速攻で利益29047円ゲットしてて草
  内訳: 仕入れ61,000 / 売上90,047 / 利益29,047
  もうこの手法知っちゃったら他の手法なんてめんどくさすぎてやってられないね

フック:
  ・他人の行動＋成果（友達が速攻で利益2.9万円ゲット）

興味対象:
  ・即日現金化できる㊙せどり手法

強化要素:
  ・速攻で（即効性）
  ・㊙（秘密性）
  ・めんどくさすぎてやってられない（優越感・手軽さ）
```

### 分解例3: 新商材2日目ポスト（作成例）

```
本文:
  新商材を仕入れ始めて2日目でこれ。仕入れ値やすいからダメージ少ないし
  回転率もいい。こういうのは情報持ってたら誰でもできるから、
  やっぱ情報ゲーやね。（実績画像）

フック:
  ・新商材を仕入れ始めて2日目でこれ（自身の行動＋成果）

興味対象:
  ・仕入れ商材の情報

強化要素:
  ・仕入れ値が安い（リスク低）
  ・ダメージ少ない・回転率いい
  ・誰でもできる（再現性）
```

## 冒頭一文目の興味付け14手法

> 出典: 教材3「反応がとれる良いポストの5つのポイント」 + 教材5「反応が取れるポストのコツ」（2026-04-27 取り込み）

ポストが伸びる流れ:

```
ポストが目に止まる
  ↓
最後まで読む
  ↓
アクション
```

冒頭一文目で「これ気になる」「自分に関係ある」と思わせないとインプは伸びない。
**影響力が小さいアカウントほど冒頭にこだわる**。
冒頭で惹きつけられるかで **ポストの伸びが7割決まる**（教材5）。

> ⭐ = 教材3で特に重要とされた4つ。最初はこの4つから使う。

### ① 不完全情報

何が起きているか曖昧にして「続きを知りたい」と思わせる。

例:
- 「とある案件で儲かっててヤバい…」
- 「ちょっと聞いてほしい話があるんですが…」

### ② 重要性

「これ、超大事です」と前置きで重みを乗せる。

例:
- 「これ結構ヤバい情報ですが...」
- 「超有益なんで言いたくないんですが…」

### ③ 希少性

「めったに公開しない情報」感を出す。

例:
- 「ガチで誰も知らんノウハウなんですが、実は〜」
- 「教材にも書いてないことを言いますが…」

### ④ 権威性

実績ある人物・組織の名前で信頼を借りる。

例:
- 「アフィで月300万円売ってる友人から聞いたんですが...」
- 「年商10億の経営者がこっそり教えてくれたんやが」

### ⑤ 恐怖・損失回避 ⭐

知らないと損する／このままだとヤバい、というフレーミング。
人は「得する話」より「損を避ける話」に強く反応する。

例:
- 「これ知らないと、ガチで損すると思う」
- 「知らなきゃヤバい話」
- 「マジでビジネス理解してなさすぎ。そりゃ稼げない。」

### ⑥ ターゲット刺し ⭐

階段の特定段（特に0〜1段目）の読者を直接呼びかける。

例:
- 「主婦こそ〇〇すべき理由について話します。」
- 「Xで売上でないから萎えた。」（1段目を狙い撃ち）
- 「サラリーマンの副業はこれ一択。」

### ⑦ 強烈な感情

感情を冒頭に置いて引き込む。

例:
- 「エグすぎるwwww」
- 「ちょっとこの前、おもろいことがあったんですが、」
- 「衝撃すぎて寝れなくなった件について」

### ⑧ パワーワード

普通使わない強烈な単語を冒頭に。

例:
- 「放送事故やらかしました...」
- 「情弱狩りが最高な話聞きます？」
- 「稼ぎたければ大◯吸え」
- 「爆益報告させてください」

### ⑨ 簡易性

「こんな簡単に？」感を出す。

例:
- 「たった2分で〇〇できちゃってビビってます...」
- 「コピペ1回するだけで反応3倍になった話」

### ⑩ 矛盾

直感に反する主張で「えっ？」と立ち止まらせる。

例:
- 「自己投資すればするほど稼げなくなる理由」
- 「真面目な人ほどXで稼げない」

### ⑪ ニュース性 ⭐

「最新」「速報」「トレンド」感を出す。

例:
- 「流石にやらかしたかもしれません...」
- 「【速報】〇〇を発見しちゃいました！」
- 「AIが…」（トレンドテーマ）

### ⑫ 暴露・報告

「こっそり教える」感で読者を共犯者にする。

例:
- 「ちょっと深夜なので暴露しちゃうんですが、」
- 「大事な報告をさせてください」
- 「身バレ覚悟で言うんですが、」

### ⑬ 限定性

「今しか読めない」感で焦らせる。

例:
- 「後ですぐ消す」
- 「24時間で消すので保存推奨」

### ⑭ 反社会性

界隈のタブー・グレーに踏み込んで注目を集める。

例:
- 「批判覚悟で言うのですが」
- 「正味、かなりグレーなんだけど」
- 「禁断のノウハウですが…」

### 14手法の組み合わせ

複数手法を1ポストの冒頭で組み合わせると強化される。

```
例: 「【AIが】今すぐ読め。絶対。【知らなきゃヤバい話】」
  - ⑪ ニュース性（AIが）
  - ⑦ 強烈な感情 + ⑧ パワーワード（今すぐ読め。絶対。）
  - ⑤ 損失回避（知らなきゃヤバい話）
```

```
例: 「ちょっと深夜なので暴露しちゃうんですが、ガチで誰も知らんノウハウあって…」
  - ⑫ 暴露・報告
  - ③ 希少性
  - ① 不完全情報
```

### 14手法と強化要素14分類の関係

冒頭14手法は強化要素14分類と一部重なる（同じ心理レバーを冒頭で使うか本文で使うか）:

| 冒頭手法 | 強化要素分類 |
|---|---|
| ③ 希少性 | ② 他の人が知らない / ③ 一部の人だけが独占 |
| ④ 権威性 | ⑨ 限定性（情報を入手しにくい） |
| ⑤ 損失回避 | ⑫ 恐怖心（知らなきゃ損） |
| ⑦ 強烈な感情 | ⑪ 感情 |
| ⑨ 簡易性 | ① シンプルで真似しやすい / ⑤ 労力かからない |
| ⑪ ニュース性 | ⑩ トレンド性 / ⑮ 新規性 |
| ⑫ 暴露・報告 | ⑬ 不明性（とある〜） |
| ⑭ 反社会性 | ⑭ 常識破壊 |

→ 強化要素14分類は **本文の体力**、冒頭14手法は **入口の引っかかり**。役割が違うので両方使う。

### コツ

- 伸びてる人の冒頭をひたすら **真似てストック**
- 「言いたいこと」をそのまま書かず、必ず冒頭にフックを噛ませる
- 小手先と思わずちゃんとやる（伸びてない人は冒頭意識がない）

## Xポストの構成原則: 結論はラスト

> 出典: 教材5「反応が取れるポストのコツ」（2026-04-27）

### PREP法はXでは最悪

ライティング教科書で定番のPREP法（結論→理由→具体例→結論）は **Xでは最悪**。

```
PREP法:
  Point   ← 主張（結論）
  Reason  ← 理由
  Example ← 具体例
  Point   ← 再度主張（まとめ）
```

**理由**: 最初に結論を言うと **続きが気にならなくなって満足される** → 滞在時間が伸びない → アルゴで弱い。

### Xは結論ラスト構造

```
冒頭フック（14手法）
  ↓
背景・興味付け・ストーリー
  ↓
モチベ上げ・自己啓発
  ↓
理由・具体例（最大まで膨らませる）
  ↓
ある程度引き延ばしてから
  ↓
満を持して結論
```

→ 結論を引き延ばすことで滞在時間を稼ぐ。

## 滞在時間を伸ばす（長文化テクニック）

> 出典: 教材5（2026-04-27）

滞在時間が長いほどアルゴ優遇。**たまに長文を挟む**（全部だとウザい）。

### 長文化のアイデア10種

1. **冒頭に強い興味付けを入れる**（14手法）
2. **ストーリーを入れる**（実話・コンサル生事例 等）
3. **感情をリアルに具体的に書く**（「マジでビビった」「鳥肌立った」）
4. **情報の具体度を上げる**（数字・固有名詞・場面描写）
5. **理由を掘り下げる**（なぜそれがダメか / なぜ効くか）
6. **モチベ上げを入れる**（自己啓発要素 — 後述30要素）
7. **メリットを語る**（手段の教育の魅力）
8. **恐怖を煽る**（問題点の教育の放置リスク）
9. **対比を入れる**（よくある人 vs 賢い人）
10. **読者の知能に合わせて噛み砕く**（無駄に親切に書く）

### 例: 短文 → 膨らませる

```
❌ 短文（薄い）:
「ノウハウポストは無意味です」

⭕ 膨らませた版:
「『ノウハウをポストしましょう！🤪』という発信者はパチモンです。
 そもそもノウハウだけ垂れ流しても誰も読まないし、
 教育要素のないポストは反応も来ないから完全に時間の無駄。
 むしろ目的の教育や信用の教育を混ぜないと永遠に伸びません。」
```

→ 倍以上の長さになり、反応したくなる表現に変わる。

### 重要原則: レベル高い人ほど端的に書きがち

レベルが高い人は **本質だけ端的に書く** 癖がある。
→ 初心者には届かない。
→ あえて **無駄にわかりやすく**、読者の知能に合わせて掘り下げる。

## モチベ上げ30要素リスト（自己啓発フレーズ集）

> 出典: 教材5（2026-04-27）
> 文中に2〜3個入れるだけで反応率が大幅に上がる。
> 思いつかない時はChatGPTに「このポストをモチベが上がるように膨らませて書き換えて」と依頼するのも有効。

### 1. 可能性を肯定する
- 読者の成長余地や才能を認める
- 「まだ始めてないだけ」「やればできる」と伝える

### 2. 行動の価値を強調する
- 行動した瞬間に未来が変わる
- 小さな一歩の意味を大きくする

### 3. 継続の力を伝える
- 才能より習慣、継続が勝つ
- 毎日の積み重ねが未来を作る

### 4. 未熟さの肯定
- 最初は誰でも下手
- 失敗は成長の証

### 5. 比較優位を示す
- 行動してるだけで上位層
- 悩んでる時点で進んでる

### 6. 今すぐ始める理由を与える
- 今日の行動が未来を決める
- 後回しにすると損をする

### 7. 成果のタイムラグを意識させる
- 半年後に振り返ってよかったと思える
- 今の努力は未来で花開く

### 8. 常識を逆転させる
- 「やる気が出たらやる」ではなく「やるからやる気が出る」
- リスクは行動しない方が大きい

### 9. 未来の後悔を想像させる
- 挑戦しなかった後悔は一生残る
- 「あの時やっておけば」とならないために動く

### 10. 自分ごと化させる
- 読んでるあなたはすでに特別
- 行動できる人だけがここにいる

### 11. 小さな成功体験を意識させる
- 小さな一歩で十分
- 成功は最初の行動から始まる

### 12. 成長をプロセスとして捉えさせる
- 完璧より未完成の行動
- 過程が実力を作る

### 13. 自己肯定感を高める
- 悩めるのは前進の証拠
- 迷うことは可能性を探してるサイン

### 14. 不安を逆手に取る
- 不安がある＝挑戦してる証
- ドキドキは伸びてるサイン

### 15. 読者を「挑戦者」扱いする
- 行動してるだけで他より前にいる
- あなたはすでに勝負の舞台に立っている

### 16. シンプルな真理を突く
- 動けば変わる
- 習慣が人生を作る

### 17. 背中を押す問いかけ
- 半年後にどうなっていたい？
- 今日の一歩を未来の自分は喜ぶ？

### 18. 勇気の定義を与える
- 恐怖があっても踏み出すのが勇気
- 小さな挑戦こそ勇気の証

### 19. 可能性の拡大を見せる
- 今の一歩で可能性が何倍にも広がる
- 今日動けば未来の選択肢が増える

### 20. 周囲との差を意識させる
- 動いた人だけが景色を変えられる
- 大多数は動かない。だから差がつく

### 21. 成果のシンプル法則
- 習慣＞才能
- 継続＞瞬発力

### 22. 一歩先の未来像を描かせる
- 来月には見える景色が変わる
- 数ヶ月後のあなたは今を感謝する

### 23. 自分の選択を正当化させる
- 行動したあなたは正しい
- 悩んでも動いた人だけが進める

### 24. やめないことの強さ
- 諦めない人が最終的に勝つ
- やめないだけで才能になる

### 25. シンプルな行動促進
- 今すぐやれ

### 26. 逆説的に肯定する
- 「できない」と思う時点で伸びしろがある

### 27. 成功のハードルを下げる

## 起承転結型連投構成（ストーリー型運用）

> 出典: 教材6「ストーリー型運用」 + 教材7「ストーリー型運用の組み方とポイント」（2026-04-27 取り込み）
> 全体像は `story_operation.md` 参照。

ストーリー型運用では **複数のポストを連投で繋げて1つの物語** を作る。
基本骨格は **起承転結**:

```
①起：宣言「〇〇をします！」
   ↓
②承：途中経過・期待感を高める・死亡フラグ
   ↓
③転：ピンチ or チャンス（メイン事件）
   ↓
④結：失敗 or 成功 + オチ
```

### 各段階の詳細

#### ①起 — 物語の始まり / 宣言

エピソードの開始を宣言する。

```
✅ 例: 「浪人界隈の囲われ姫とエンカ（リアルで会う）してホテルに行きます」
✅ 例: 「ネカフェ連れ込み成功！これから童貞卒業するぞ！」
```

→ 「これから物語が進む」感を出す。

#### ②承 — 途中経過 / 死亡フラグ

途中経過をポストすると、読者が自然と応援したくなる。

```
✅ 例: 「ホテル到着、緊張してる」
✅ 例: 「マジでこれから童貞卒業します」
✅ 例: 「絶対いける！俺の勝ちや！」
```

##### 死亡フラグの活用

確信に満ちたポストをあえて入れることで、読者に「あ、こいつヤバいかもな…」と期待感が生まれる（バトルアニメで「俺を残して先に行け！」と言うやつはだいたい死ぬ）。

#### ③転 — ピンチ or チャンス（メイン事件）

物語のメイン事件。状況の上振れ／下振れ。

```
✅ 失敗系: 「シャワー浴びてる間に逃げられた…」
✅ 成功系: 「俺の勝ち！！！！童貞卒業！！！きたぞ！！！！」
```

#### ④結 — オチ

成功談 / 失敗談 / 後日談でオチをつける。

```
✅ 失敗 + 二段落ち: 「ブロックされた」
✅ 成功 + 派手なオチ: 「性病感染の可能性…」
```

##### ハッピーエンドのまま終わらせない

```
❌ ハッピーエンドだけで終わる
⭕ どこかで落としてネタ要素を入れる
⭕ ただ伸ばしたい場合はバッドエンドも有効
```

ただし **商材を売る場合** は成功体験（理想状態）に繋げる必要があるので、最低限の実績にはたどり着かせる。

### 各ポストの切り方の鉄則

ドラマ・アニメと同じく **クライマックス直前で毎回終わる** ように設計する。

```
✅ 「ホテル来たけどやり方わかんねぇ…教えてくれ！これ逃げられたら埋めてくれ…」
   → 続きが死ぬほど気になる

❌ 「ナンパしてホテル行ったけど逃げられました。来週も頑張ります。」
   → 完結してしまって読者は離れる
```

### ストーリーの5要素（構成要素チェックリスト）

連投シリーズを設計する時に必ず含める:

- ✅ **挑戦の対象**（何を達成しようとしているか明確）
- ✅ **障害・困難**（うまくいかない要素）
- ✅ **感情の起伏**（喜び・絶望・焦りをリアル描写）
- ✅ **クライマックス**（成功 / 派手な失敗 / 意外なオチ）
- ✅ **連投で引っ張る切り方**（毎回続きが気になる地点で終える）

### ネタの強さ（事前設計）

連投構成の前に、ネタの強さを確認する3要素（詳細は `story_operation.md`）:

- ✅ **目新しさ** — 他人がやってない / やれない / やりたくない
- ✅ **変化率** — スタートとゴールの差（実績小ならスタートを下げる）
- ✅ **事件性** — 予期しないトラブル

### 実例: ぶんじろう垢の起承転結分析

教材6の3パターンを起承転結に当てはめると:

```
[パターン1] 浪人界隈時代（失敗オチ）
  起: 浪人界隈の姫とエンカでホテル
  承: ホテル到着、緊張・期待
  転: シャワー浴びたら逃げられてる
  結: ブロックされる（二段落ち）

[パターン2] 駆け出しナンパ師時代（失敗オチ）
  起: 女の子と宅飲み！童貞卒業の大チャンス
  承: 期待感ポスト
  転: 途中までいったのに追い出される
  結: 鬼ディスられる

[パターン3] ナンパ成功 → 派手オチ
  起: ネカフェ連れ込み成功
  承: 大チャンス！応援よろしく！
  転: 童貞卒業！俺の勝ち！
  結: 性病感染の可能性…
```

→ 失敗・派手なオチでも十分バズる。

### ⚠️ テンプレ通りに動かそうとしない

教材執筆者自身も「細かいシナリオは事前に決めず、その場その場でやる」と明言。
**起承転結はあくまで事後分析の骨格** であって、リアルタイムは出来事に応じて柔軟に。

100%テンプレ通りのストーリー型運用は **そもそも不可能**。

## ジャブ投稿パターン

> 出典: 教材7「ストーリー型運用の組み方とポイント」（2026-04-27 取り込み）
> ストーリー型運用のリアルタイム感を強化するための補助ポスト。

### ジャブ投稿とは

ストーリーの本流ポスト（起承転結）の **間に挟む短い投稿**。
単体では意味がないが、リアルタイム感を強化し、次のポストへの **布石** になる。

### 例

```
「ちょ、、、ヤバいこと起こった...」
「ショックすぎて気絶してました...夜にまた話します。」
「今から楽しいことしちゃうよーん♪」
「田中まじでこいつ終わりだろ」
「明日大事なポストするので寝ずに待っといてください」
```

### 機能する理由

- **「え、何があったの？！」** と反射的に気にさせる → マインドシェア
- 本流ポストへの **期待感** を高める
- **リアルタイムで何かが進行している感** を演出

### 配置タイミング

```
[起] → [ジャブ] → [承] → [ジャブ] → [転] → [結]
       ↑                ↑
       期待を煽る        次の事件を予感させる
```

### 注意

ジャブを入れるか入れないかで反応が **全然変わる**。
ストーリー型運用時は **意識的に入れる**。
