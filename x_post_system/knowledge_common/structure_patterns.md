## 興味付けポスト構造

> 出典: 教材1「興味づけポストの基礎」（2026-04-27 取り込み）

```
[フック] → [興味の対象] → [強化要素 ×2〜3]
```

### フックのカテゴリと例

| カテゴリ | 例 |
|---|---|
| 自分の実績/成果報告 | 「俺が〇〇円稼いだ」「昨日仕入れに行って爆利益品みっけたんだけど」 |
| 経歴/背景 | 「20垢運用してわかった」 |
| 他人の権威 | 「メルカリで1億売り上げてる主婦から聞いた」「noteで月2000万売った友達に教えてもらった」「万垢30個分析してわかったんだけど」 |
| 他人の行動＋成果 | 「23歳のフリーターの弟に教えたら速攻で83万円稼いじゃった」 |
| 自身の行動＋期待感 | 「今裏で仕込んでるジャンルだけど」「ワンチャン月500万くらいいけそうw」 |

> ⚠️ フックは **必ずしも成果が出たものでなくていい**。「うまくいきそう」という期待感だけでも人間は興味を持つ。

### 興味の対象カテゴリ

#### 情報系
- 今アツいボロ儲け市場 / 激アツジャンル / バブルなプラットフォーム
- アルゴリズム変更 / 極秘仕入れ先情報
- 知らないとヤバい落とし穴 / 裏情報 / これから来るトレンド

#### 手法系
- コツ・秘訣がある / ずるい戦略 / 独自のスキーム
- とある施策が… / ノウハウ / 着眼点

#### その他（ツール・資産）
- ツール / 専用AI / 型・テンプレ / シート / サイト

> 💡 ジャンルによって何を欲しがるかは違う。「他の人が何を配ってるか」「何系の情報を欲してるか」を常に観察する。

## 強化要素 14カテゴリ語彙集

> 出典: 教材1（2026-04-27）
> ポスト作成時、ここから2〜3個ピックして組み込む。

### ① シンプルで真似しやすい
- やったのはこの2つだけ / 簡単すぎる / コピペでいける

### ② 他の人が知らない
- 誰もやってない / ライバルがいない / 知ってる人マジでいない
- どの教材にも書いてない / 僕が思いついた / 超ニッチ / ブルーオーシャン

### ③ 一部の人だけが独占してる
- ガチで極秘コミュニティだけで言ったけど / 本当は言いたくないけど

### ④ 再現性がある
- 誰でもできる / 高卒ニートの弟でもできたw / 中学生でもいける
- 猿でもいける / 偏差値30でもいける

### ⑤ 労力がかからない
- 15分〇〇しただけ / ガチで労力かからん

### ⑥ 即効性がある
- 2日でこれ / 1週間もあれば

### ⑦ リスクがない
- 仕入れ4円だけ / スマホだけでいける / 元手ゼロでいける
- ローリスク / 顔出し・声出しもいらない

### ⑧ 優越感
- ライバルが消える / 〇〇に困ることがなくなる
- 〇〇できてしかたがない / 無限に〇〇できる / 〇〇を量産できる

### ⑨ 限定性（情報を入手しにくい）
- 極秘ルートで仕入れた情報 / 1億円稼ぐ大学生から聞いた
- 一部のコミュニティだけで共有されてる / 年5億稼ぐ会社のマニュアルに書いてあった

### ⑩ トレンド性
- AIで〜 / XTEP使って〜

### ⑪ 感情
- エグい / ヤバい / 神じゃね？？

### ⑫ 恐怖心（知らなきゃ損）
- これ知らないとヤバい / 知らないと反応が落ちる

### ⑬ 不明性（何をして？どうやって？）
- とある手法で / あるツールを使うことで / あるサイトを見れば
- ちょっとあることをするだけで / 某アカウントで / 某界隈で

### ⑭ 常識破壊
- エロ・恋愛・ビジネスじゃない（SNSで） / 1ポストもしてない・ローンチしてない
- フォロワーいらない / 仕入れ不要 / 鑑定不要

### ⑮ 新規性（情報が新しい）
- 最新〇〇 / 緊急で / ガチで稼いでる人の中で話題になってる / 今トレンドがきてる

### ⑯ 長期的な利益
- 1年以上ずっと入ってくる / 1回やったら300万円確定

### その他
- 得られる金額がデカい / 成果出るのが異常に早い

## 実例分解テンプレ

> 出典: 教材1（2026-04-27）
> 既存ポストを分析するときの3列分解パターン。

```
本文:
　（ポスト本文をそのまま）

フック:
　・（何が「おっ？」を作っているか）

興味対象:
　・（読者は何が知りたくなったか）

強化要素:
　・（どの心理レバーが効いているか × 2〜3）
```

### 分解例1: ぶんじろう垢のTips販売ポスト

```
本文:
  この垢でTips販売開始して3ヶ月目とかやけど地味に売れ続けるコツみたいなんが
  若干掴めつつある。商品数がゴミやから売上しょぼいけどこの容量でガンガン
  増やしていくか。ぶんじろう垢と違って一撃の爆発力はないけど属人生を
  限りなく落としてるから正味誰でも運用できるからかなり楽。
  （売上画像）

フック:
  ・売上画像
  ・商品増やしたら儲かる（雰囲気）

興味対象:
  ・売れ続けるコツ掴めつつある
  ・属人性のない運用方法

強化要素:
  ・属人性落としてる → 再現できそう
  ・誰でも運用できる → 再現できそう
  ・かなり楽
```

### 分解例2: にきち垢の即日現金化せどりポスト

```
本文:
  今日友達にとある即日現金化できる㊙せどり手法教えたんだけど、
  速攻で利益29047円ゲットしてて草
  内訳: 仕入れ61,000 / 売上90,047 / 利益29,047
  もうこの手法知っちゃったら他の手法なんてめんどくさすぎてやってられないね

フック:
  ・他人の行動＋成果（友達が速攻で利益2.9万円ゲット）

興味対象:
  ・即日現金化できる㊙せどり手法

強化要素:
  ・速攻で（即効性）
  ・㊙（秘密性）
  ・めんどくさすぎてやってられない（優越感・手軽さ）
```

### 分解例3: 新商材2日目ポスト（作成例）

```
本文:
  新商材を仕入れ始めて2日目でこれ。仕入れ値やすいからダメージ少ないし
  回転率もいい。こういうのは情報持ってたら誰でもできるから、
  やっぱ情報ゲーやね。（実績画像）

フック:
  ・新商材を仕入れ始めて2日目でこれ（自身の行動＋成果）

興味対象:
  ・仕入れ商材の情報

強化要素:
  ・仕入れ値が安い（リスク低）
  ・ダメージ少ない・回転率いい
  ・誰でもできる（再現性）
```

## 冒頭一文目の興味付け4手法

> 出典: 教材3「反応がとれる良いポストの5つのポイント」（2026-04-27 取り込み）

ポストが伸びる流れ:

```
ポストが目に止まる
  ↓
最後まで読む
  ↓
アクション
```

冒頭一文目で「これ気になる」「自分に関係ある」と思わせないとインプは伸びない。
**影響力が小さいアカウントほど冒頭にこだわる**。

### 手法① トレンドテーマ

今ホットなテーマを冒頭に置く。

例: 「**AIが**…知らなきゃヤバい話」

→ 「AIが」だけで「これ最新トレンドかも」と注意を引く。

### 手法② 損失回避

知らないと損する／このままだとヤバいというフレーミング。

例:
- 「**知らなきゃヤバい話**」
- 「マジでビジネス理解してなさすぎ。**そりゃ稼げない。**」

→ 人は「得する話」より「損を避ける話」に強く反応する。

### 手法③ 強調

短く強い命令／断定で注意を奪う。

例: 「**今すぐ読め。絶対。**」

→ 「絶対。」「マジで。」「ガチで。」のような強調語を冒頭に置く。

### 手法④ ターゲット狙い（特定段階の読者を名指し）

階段の特定段（特に0〜1段目）の読者を直接呼びかける。

例: 「**Xで売上でないから萎えた。** マジでビジネス理解してなさすぎ。」

→ 「売上出ないから萎えた」という具体状況で1段目の人が「自分のことだ」と感じる。

### 4手法の組み合わせ

複数手法を1ポストの冒頭で組み合わせると強化される:

```
例文: 「【AIが】今すぐ読め。絶対。【知らなきゃヤバい話】」
  - トレンドテーマ（AIが）
  - 強調（今すぐ読め。絶対。）
  - 損失回避（知らなきゃヤバい話）
```

### 強化要素14分類との関係

冒頭4手法は **強化要素14分類のサブセット**:

| 冒頭4手法 | 14分類との対応 |
|---|---|
| トレンドテーマ | ⑩ トレンド性 |
| 損失回避 | ⑫ 恐怖心（知らなきゃ損） |
| 強調 | ⑪ 感情（エグい/ヤバい） |
| ターゲット狙い | （独立。階段思考に基づく） |

→ 冒頭は **14分類のうち「トレンド/損失回避/感情/ターゲット名指し」** を優先的に使う。
