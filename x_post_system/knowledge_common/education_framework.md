## このフレームワークの位置づけ

ポストにおいては **どんな文章でポストするのか（ライティング）** より、
**何をポストするのか（ネタ）** の方がずっと重要。

ネタを思いつくための **2大思考フレーム**:

1. **6つの教育** — どの教育目的のポストかを意識する
2. **グダ消し** — 買わない理由を先回りで消す

> 厳密にはグダ消しも「6つの教育」のうち手段の教育の一部だが、特に重要なので独立して扱う。

ポストとは **極論ほぼすべて教育**。教育を満遍なく入れることで:
- 「商品を買う読者」
- 「マインドが高く行動力がある読者」
- 「自分の発信に共感してくれる読者」

を作っていくのが情報発信の本質。

## 教育とは何か（定義）

> 出典: 教材10「教育とは何か」（2026-04-27 取り込み）

### 教育の定義

```
相手の価値観や意思決定を変えて、
望むような行動を「自発的に」させること
```

**「自発的に」がポイント**。
- 説得・説教＝「勉強しなさい！」と直接命令
- 教育＝「俺は子供の頃に勉強しなかったせいで苦労した。でも勉強するようになって人生変わった」と話して、自分から机に向かわせる

### なぜ重要か

> 教育ができる ＝ 人を自由自在に動かせる

- 教育ができれば、少ないフォロワーでも商品が馬鹿みたいに売れる
- インプを増やすことも、フォロワーを増やすことも容易になる
- 押し売り（「買ってください」）なしに、お客さんから「買わせてください」と言われる状態を作れる

### 教育の力の証明

教材執筆者の経験（靴磨き / ナンパ / ビジネスで数百人と接客）から:

> 「人間は一定の情報を与えると、何かに突き動かされるかのように、喜んでお金を差し出したり、自分が望む行動をとってくれる」

→ 業種・ジャンルを問わず、教育の原理は同じ。

### 集客 vs 教育

```
集客の上手さ < 教育の上手さ
```

- 集客が得意に見えるのは、実は **教育で人が動くから集客が成立する** だけ
- 教育ができなければ、どれだけ集客しても商品は売れない

## 価値観は「教育の結果」である

> 出典: 教材10（2026-04-27）

人の価値観は、実は **誰かに植え付けられたもの**。

### 例: Xマネタイズに興味を持つまでの価値観遷移

```
会社員しんどい... 手取り少なくて辛い...
        ↓
なんか副業しようかな？
        ↓
物販は仕入れ費が… 動画編集はスキルが…
        ↓
Xマネタイズで稼げるらしい！コストもかからないし青天井！
        ↓
稼ぎたかったら知識つけて自己投資しないとね
        ↓
よし、スクールで学んでみよう！
```

これらの価値観すべて:
- 「会社員＝しんどい」印象
- 「副業しないとやばい」危機感
- 「自己投資しないと稼げない」マインド
- 「行動を起こさないといけない」必要性

→ **誰かの教育の影響** で形成されている。
→ これを **狙って起こせるようになる** のが「教育ができる」状態。

## まだまだ客 → 今すぐ客（= 絶望客 → 理想客）

> 出典: 教材10「教育とは何か」 + 教材11「価値観教育」（2026-04-27 取り込み）

見込み客は2種類に分けられる。教材11では同じ概念を **絶望客 / 理想客** とも呼ぶ。

| まだまだ客 = 絶望客（買えない） | 今すぐ客 = 理想客（買える） |
|---|---|
| 商品購入から遠い | 商品をすぐ買える状態 |
| 「ちょっと怪しいな」 | 「この人から買いたい」 |
| 「無料で済ませたい」 | 「自己投資が当たり前」 |
| 「何からやればいいか分からない」 | 「すぐ始めたい」 |

### 絶望客の詳細価値観リスト（11項目）

> 出典: 教材11（2026-04-27）

絶望客が持っている価値観の典型:

```
□ 手法や商品に興味がない
□ あなたのことを信用してない
□ 現状のままでいいや…と思ってる
□ リスクを取りたくない
□ お金を払いたくない（無料でやりたい）
□ 努力したくない
□ 勉強するのはめんどくさい
□ 自分には目標達成できないと思ってる
□ 手法に再現性を感じない
□ 誰かに依存して稼げるようにしてほしい
□ 行動したくない
```

### 理想客の詳細価値観リスト（10項目）

> 出典: 教材11（2026-04-27）

理想客が持っている価値観の典型:

```
□ 〇〇さんみたいになりたい！
□ 〇〇さんみたいな生活がしたい！
□ 〇〇さんの発信は信頼できる！
□ 自分の問題点に気づいてる
□ 問題を解決する手段もわかっている
□ あなたの手段は再現性があって自分に最適だと思ってる
□ お金を使って知識や時間を買うのは重要！
□ ある程度のリスクを取るのも重要！
□ ビジネスは自責で行動するのが大事！
□ チャンスを掴むには言われたことに素直に即行動するのが大事！
```

### 絶望客に売る危険性

> 出典: 教材11（2026-04-27）

絶望客のまま売ろうとするのは危険:

- ❌ **セールス難易度が激上がり**（成約率も成約単価も大きく下がる）
- ❌ 仮に売れても **行動しないので成果が出ない**
- ❌ リスクが取れない人なので **成果を出させるのが大変**
- ❌ **他責な人が多い** → 炎上やクレームに繋がりやすい

→ 教育で絶望客を理想客に変えてからセールスするのが鉄則。
→ 教育を経ずに面談・セールスに進むのはNG。

### 教育のゴール

```
普段の発信 = 絶望客 → 理想客 への変換工場
```

ローンチで爆売れする人は、**普段から教育で「理想客」を増やしている**。
- 新商品を出した瞬間に即売れる
- ゲリラスペース告知でも数百人が集まる
- 教育を経ない面談・セールスは成約率が激落ちする

### スクールのマインド講座も教育

このスクールが入会直後に **マインド講座を30講義** 見せるのも、
「成果を出しやすい価値観」に全員を変化させるための **教育**。

→ ノウハウ前にマインド整備 = 価値観操作の典型例。

## 教育マスターの2ステップ

> 出典: 教材10（2026-04-27）

教育を体系化すると2ステップ:

```
①価値観操作   — まだまだ客 → 今すぐ客 に根本転換
   ↓
②ロジック構築 — 商品が欲しくなる思考回路を作る
```

### ステップ1: 価値観操作

「商品を買いやすい価値観」へ根本から変える。

例（ビジネス系の価値観操作）:
- 「自己投資して知識つけるのは、稼ぐために大事」
- 「Xマネタイズはリスク低くて自分にもできそう」
- 「学ぶならこの人から学ぶのが一択」

→ 個別商品ではなく **マインド・世界観のレベル** での書き換え。

### ステップ2: ロジック構築

> 出典: 教材10「教育とは何か」 + 教材13「ロジック構築」（2026-04-27 取り込み）
> 教材13で詳細化された正式版。

特定商品が欲しくなる **思考回路** を組む。

#### ロジックの定義

```
「〇〇が必要だ」となるような文章の構成
```

→ 自分が売りたい商品や手法の **必要性** に読者が気づくようなメッセージを作る方法。

#### 社会レベルのロジック実例: 老後2000万円問題 → NISA

ロジック構築は個人の発信だけでなく、社会全体でも機能している:

```
年金だけだと老後資金が2000万円不足する（問題提起）
   ↓
老後に備えるために個人で投資して資産形成しないとやばい（原因+解決策）
   ↓
NISAで投資して老後に備えよう（行動）
```

→ この「老後2000万円問題」のロジックがあったからこそ、ここ数年でNISA利用者が爆発的に増えた。
→ ロジックがなければ「投資で資産形成しよう」という思考にならなかった。

→ **人はロジックで納得して初めて行動する**。

---

### 2つのロジック型

ロジックの組み方は無限にあるが、特に使いやすいのが2つ:

| 型 | 使い分け |
|---|---|
| **① 課題解決型ロジック** | 悩みが具体的にある人に使いやすい |
| **② 欲求喚起型ロジック** | ゼロから需要を生み出したい時に使う |

詳細・テンプレ・実例は `structure_patterns.md` の「課題解決型 / 欲求喚起型ロジック」参照。

---

### 課題解決型ロジック（4ステップ）

```
1. 都合のいいテーマで問題提起
2. 都合のいい原因を指摘
3. 都合のいい解決策を提示
4. 理想の未来を見せる
```

> 教材10の4ステップ（共感→真因→解決策→商品導線）は、教材13でこの版に **置換・統合** された。

---

### 欲求喚起型ロジック（4ステップ）

```
1. 理想の未来を見せて欲求喚起
2. 要因（=売りたいもの）を提示する
3. 都合のいい根拠で固める
4. 結論でアクションを提示する
```

→ ポストでラフに教育したい時に使いやすい。

---

### コア原則:「都合のいい」を3連発

> 出典: 教材13（2026-04-27）

ロジック構築では **「都合のいい」** を3つの段階で意識する:

```
① 都合のいいテーマで問題提起 ← 自分の商品で解決できる悩み・欲求を選ぶ
② 都合のいい原因を指摘       ← 自分の商品に都合よく合う原因を選ぶ
③ 都合のいい解決策を提示     ← 自分の商品が解決策になるよう導く
```

#### なぜ機能するか

人間がうまくいかない原因は **複数** ある。
例: Xでインプが伸びない原因
- 実績がなくて興味付けが弱い
- アカウントパワーが足りない
- アルゴリズムが変わった
- 投稿時間が悪い ...

→ その中から **自分の売りたい商品（例: アルゴ攻略教材）に都合のいい原因（=アルゴ変更）** を取り上げる。
→ 読者は勝手に「自分に足りないのはアルゴの知識だ」と思う。

#### 倫理的な側面

「都合のいい」を選ぶのは **嘘をつくこと** ではない。
- 原因として複数の真実があり、その中の1つを選んでフォーカスするだけ
- 自分の商品で本当に解決できる原因を選ぶ
- 教育で読者を成果に導くのが目的

### コア原則: 読者には考えさせない

> 出典: 教材13（2026-04-27）

```
あなたが稼ごうと思ったら思考力は大事だが、
商品を売りたいなら、読者には考えさせないのが大事。
```

#### 解決策の提示はシンプルに

```
✅「〜するだけ！」
✅「〜さえすればOK！」
✅「これだけやっておけば大丈夫」
```

→ 「いろんな選択肢から選んで…」みたいな複雑な提示はNG。
→ なるべくシンプルに、これ1つで全部解決と感じさせる。

---

### コア原則: 理想の未来は3層で見せる

> 出典: 教材13（2026-04-27）

ベネフィット（理想の未来）は **3層** で展開する:

| 層 | 例（教育を売る場合） |
|---|---|
| **直接メリット** | 成約率が上がる / ローンチが伸びる / LINE登録数が増える |
| **その先のメリット** | 同じ労力で収益が3倍 / 月100万・500万・1000万 / 自由自在にお金を生み出せる |
| **日常生活レベル** | 寝てる間にお金が生まれる / 高級焼肉に行ける / 平日に弾丸で旅行 / 節約不要 / 老後の不安なし |

#### なぜ3層か

- 直接メリットだけだと刺さる人がごく一部
- その先・日常レベルまで描くと **共感と憧れが増える**
- 「未来の理想生活」までイメージさせると行動が引き出せる

---

### 効果の証明

教材執筆者の実測値:

> 「ロジック形成をした読者 vs ロジック形成をしてない読者にセールスかけたら、**売上が2倍以上変わった**」

→ ロジック構築の威力は数値として実証されている。

### 旧マッチングアプリ写真撮影例（教材10、参考保存）

教材10で詳述された具体例。教材13の課題解決型の実装例として参考:

```
[① 都合のいい問題提起]
マッチングアプリで全然彼女できません…っていう人多くて
みんな「コミュニケーション」「デートコース」を勉強するんですけど
```

## 教材10と教材4（6つの教育）の対応マップ

教材10の2ステップは、教材4の6種類を再分類したもの:

| 教材10（メタ・2ステップ） | 教材4（実装・6種類） |
|---|---|
| **価値観操作** | 目的の教育 / 信用の教育 / 行動の教育 / 投資の教育 |
| **ロジック構築** | 問題点の教育 / 手段の教育 |

### 使い分け

→ 両方を並行して回す。

## 12の教育要素フレームワーク

> 出典: 教材11「価値観教育」 + 教材12「6つの強化教育」（2026-04-27 取り込み）

絶望客を理想客に変えるためのメッセージは、パターン化できる。
そのパターンが **12の教育要素**:

```
12の教育要素 = 前半6つ（基礎）+ 後半6つ（強化）

前半6つ: 6つの教育（基礎）
  ① 目的の教育
  ② 信用の教育
  ③ 問題点の教育
  ④ 手段の教育
  ⑤ 投資の教育
  ⑥ 行動の教育

後半6つ: 6つの強化教育（ドーピング剤）
  ⑦ 読む・見る教育        — コンテンツに「見るべき理由」を作る
  ⑧ 変化の教育            — 「変化＝良いこと」思考を植え付け
  ⑨ 素直の教育            — 素直に学ぶ価値観
  ⑩ アウトプットの教育    — アクションを当たり前にする
  ⑪ 基準値の教育          — 極端な行動で基準値をバグらせる
  ⑫ 覚悟の教育            — 覚悟を決めた行動を見せる
```

### 前半 vs 後半の役割

| 種別 | 役割 | 取り込み状況 |
|---|---|---|
| **前半6つ** | 教育の基礎骨格。SNSで稼ぐ全員が知るべき内容。 | ✅ 教材4・教材11で詳述 |
| **後半6つ（強化教育）** | 6つの教育の効きを **何倍にも強化** するドーピング剤。普段のポストや記事・動画に散りばめる。 | ✅ 教材12で詳述 |

### 強化教育の使い方

- 後半6つは独立した教育タイプではなく、**前半6つ + ストーリー型運用 + インセンティブ設計** にスパイスとして散りばめるもの
- 普段のポスト1本ごとに、強化教育を1〜2個埋め込む
- 特に **基準値の教育** と **覚悟の教育** は高単価商品（数十万〜数百万）販売時に必須
- 強化教育は **教材執筆者の独自理論** （ネットビジネス業界標準ではない）

## 6つの教育 — 全体像

ネットビジネス業界で使われている定番フレームワーク。

```
1. 目的の教育   — 「〇〇したい」という欲求を抱かせる
2. 信用の教育   — 読者に信用・共感してもらう
3. 問題点の教育 — うまくいってない原因と放置の危険を伝える
4. 手段の教育   — おすすめの手段とその魅力を教える
5. 行動の教育   — 今すぐ動く重要性を伝える
6. 投資の教育   — 自己投資が当たり前という基準値を作る
```

### 標準的な誘導フロー

```
【目的の教育】
  俺こんな良い状態だよ、いいだろ？
  このやり方知りたいだろ？
        ↓
【信用の教育】
  でも昔はこんなダメでした。
  紆余曲折あって変わりました。あなたにもできます。
        ↓
【問題点の教育】
  あなたがうまくいかない原因はこれです。
  これを放置するとヤバいです。
        ↓
【手段の教育】
  それを解決するにはこの手段がおすすめ。
  再現性もある、誰でもできる、リスクもない。
        ↓
【投資の教育】
  解決するには環境や知識に投資していきましょう。
  僕も自己投資してきたからこそ手に入った。
        ↓
【行動の教育】
  あとは行動しましょう。
  動かない人生は変わりません。今すぐ動きましょう。
```

これを **順番かつ満遍なく** ポストすれば、読者の見込み度合いが徐々に上がる。

## ① 目的の教育

> **「〇〇したい」という欲求を抱かせる教育**

理想の未来や過去からの変化（Before/After）を見せて、欲求喚起する。

### 訴求軸

- お金面の理想の状態
- 時間・環境・ライフスタイルの理想
- 性についての理想（モテ・恋愛）
- 優越感・承認欲求的な理想

### 例フレーズ

- 「満員電車に乗らなくて良くなった」
- 「ウザイ上司より稼いでる優越感」
- 「童貞だったのにセフレに困らなくなった」
- 「月1で旅行に行ける」
- 「毎日のように10万円以上振り込まれる」
- 「ほぼ自動化したので暇すぎる…w」
- 「今日も日給10万円 チャリーン！」

### 3つのポイント

#### ポイント①: 切り口をストックする

ジャンルによって目的の教育の切り口は無限にある。
競合のポストを見て **感情が動いた事例をストック** する。

例:
- フォロワーが伸びる
- インプが増える
- 日給〇〇万円
- 旅しながら稼げる
- 寝ててもお金が入ってくる
- 会社辞めてウザい上司からおさらば
- モテるようになった
- 利益率が高すぎる

#### ポイント②: イメージしやすくして欲求を掻き立てる

目的の教育は **視覚的に見せる方が欲求が動きやすい**。
- 売上画像
- 何か魅力的なことをしてる画像

#### ポイント③: リアルタイム性を出す

| 弱い | 強い |
|---|---|
| 「アフィで10万円稼いだ！」 | 「鮨食ってたら10万円稼げちゃってて黒字！」 |

リアル感があると「なんかいいな」と思われる。

### なぜ重要か

人間は **「こいつ羨ましいな」「なんでこいつだけ良い思いしてんだ」「こいつのやり方知りたいな」** と思うからこそ発信を追ってくれる。

### 追加tips: 小実績でも見せ方で「羨ましい」を作る

> 出典: 教材11（2026-04-27）

「デカい実績がないと目的の教育できない」と思い込んでる人が多いが、**それは半分嘘**。

#### 重要原則
- 情報発信は読者との **「差分」** が価値になる
- 何かの実績や理想状態は必要だが、**デカい実績じゃなくていい**
- 単発1万円の実績でも見せ方次第で「羨ましい状態」を作れる

#### 例: 単発1万円実績の魅せ方

❌ そのまま伝える:
> 「1万円稼ぎました」（インパクト弱）

⭕ 表現を工夫:
> 「寝てたら1万円入ってたわ、こういう収入源いくつか持ってるとめちゃ最高だよねw」
> 「スマホちょっと触ってたら1万円儲かっちゃったわ、これエグすぎるっしょw」

→ 「たくさんの収入源がある」「労力がかからない」と **錯覚させる**。

#### 見せ方の3要素

```
① 簡単そうに見せる
② 労力低そうに見せる
③ 「当たり前」そうに見せる
```

→ 小実績でも誰でも「羨ましい状態」を作れる。

## ② 信用の教育

> **読者に信用・共感してもらう**

目的の教育で欲求を喚起しても、次に「こいつ怪しいな」「詐欺じゃね？」となる。
そこで信用を得る発信が必要。

### 信用構築の要素

- 過去のダメダメさを語る
- サクセスストーリーを語る
- 失敗談を語って共感させる
- 発信の思い・理念を語る
- 証拠を見せる
- 成果を見せつける

### 例フレーズ

- 「自分も2年前までサラリーマンで毎日10時間残業して死にかけてた」
- 「スキルゼロだったし金なかったから、とりあえず始めれるXをやった」
- 「Xはじめて最初に出した商品は700円しか売れなかった」
- 「過去の自分みたいな人を救いたいから全員俺の発信を見ておけ」

### 定番ストーリーパターン

```
今はこんなに稼いでます！
  ↓
でも昔はこんなダメダメでした！
  ↓
でもこの手法に出会って紆余曲折あって今ではこんな生活を手に入れました！
  ↓
同じ人を救いたいと思って発信しています！
```

### ポイント

- どれだけロジックが通っていても **信じてもらえなければ無意味**
- 顔出しも信用の教育になる（信憑性を高める）
- 弱みを見せる / 過去を語る / 理念を語る → どんどんファン化が進む

### 追加tips: 「こいつ俺と同じだったのか！」を作る

> 出典: 教材11（2026-04-27）

信用の教育で1番有効なのは:

```
読者「こいつ、俺と同じだったのか！」
   ↓
「俺と同じレベルだったのに、今はこんなに稼いでるのか…」
「数年前まではただのサラリーマンだったのか…」
   ↓
急に親近感が湧く → 信用が一気に積み上がる
```

#### 重要原則: 共通点を作る

- 読者との **共通点** を意図的に作る
- 「お前と同じ状態から成り上がったよ！」を伝える
- 共通点を作るために **ストーリーや過去話** を使う

#### 共通点の作り方の例

- 「俺も2年前まで会社員で…」
- 「最初は1日3時間しか副業できなかった」
- 「初月の収益は700円だった」
- 「家族に反対されながらやってた」

→ 読者が「あ、これ俺と同じ」と感じる接点を1つでも作る。

## ③ 問題点の教育

> **相手がうまくいってない原因を伝えて、放置の危険を煽る**

人間は **「なんでうまくいってないのか（原因）」** を知ると解決策を知りたくなる。

### 基本パターン

「あなたが（うまくいかない）のは（原因）だからです」
→ 読者「じゃあ、どうやんの？？」
→ 「だからこそ（自分の推す手段）が良い」

### 例フレーズ

- 「『ポスト』ばっか頑張ってもXは伸びません」
- 「初心者が『低単価』で戦うと詰みます」
- 「マチアプでマッチしないのはほぼ『画像』が原因ですねw」
- 「物販で稼げない人は『利益率低すぎ』w」

### 放置する恐怖を煽る

原因が分かっても「まぁ今じゃなくてもいいよね」と思考停止する人が多い。
そこで **放置した時の危険性** を煽って問題意識を増幅する。

### 恐怖煽りの例

- 「一生会社員のまま働き続けて、スキルもつかず40代でリストラされたら…」
- 「人生の大半を労働して生きることになるし、結婚すら厳しいかもしれない…」
- 「太ったままだと30代後半を迎えると、大きな病気のリスクが高まる」
- 「一生非モテのまま生きて、金があればモテると思って40代になり、一切モテないことに気づくも時すでに遅し」
- 「商品ないままX動かしたところで、穴の空いたバケツに水を注いでる状態」

### 2つのポイント

#### ポイント①: 初心者がやりがちなダメ行動を書き出す

稼げない人・モテない人を想像（または観察）して、彼らのダメ行動を **30個書き出す**。

例（ビジ系）:
- 一生ノウハウ集めてる
- いろんな情報に手出ししてる
- インプ取るための方法を知らない
- 導線ができてない
- ...etc

その上で:
- 「（ダメな行動）してる奴は稼げない」
- 「（ダメな行動）は今すぐやめろ」
- 「（ダメな行動）しちゃってる人はマジで危険」
- 「（反対のこと）だけやっときゃOK」

の切り口で理由・具体例を掘り下げれば問題点ポストの完成。

#### ポイント②: 仮想敵を作って責任をなすりつける

「お前が稼げないのはセンスないからだよ」と読者を直接攻撃すると離れる。
代わりに **仮想敵** を作って責任をなすりつける:

- 界隈の風潮・常識
- 大手垢の発信内容
- 社会の常識やシステム
- 別の流派・別の手法

例: 「noteで稼ごうみたいな流れ」「X運用下手な人がXを教えてる現状」

### 追加tips: ダイエット例（汎用テンプレ）

> 出典: 教材11（2026-04-27）

問題点の教育の汎用テンプレ。**「原因 → 放置の恐怖」** がきれいに繋がる例文:

```
あなたが痩せれないのは体質の問題なので、
体質に合ったダイエットをしない限り一生痩せれませんよ？

それどころか合わない方法やっちゃうと
逆にストレスで太ったり体調崩したりするので
まず体質を把握しないと結構ヤバいです。
```

→ この型は **どのジャンルにも応用できる**:
- 「あなたが稼げないのは〇〇が原因なので、これを解決しないと一生稼げません」
- 「あなたが結婚できないのは〇〇が原因なので、これを直さないと一生独身です」
- 「あなたの運気が上がらないのは〇〇が原因なので、これを変えないと一生上向きません」

→ パターン: **「原因の指摘 + 放置時の悪化」** をセットで提示。

## ④ 手段の教育

> **おすすめの手段とその魅力を教える**

自分が売りたい商品と一貫する手段を押し出す。

### 順序の重要性

```
目的の教育 → 信用の教育 → 問題点の教育 → 手段の教育
```

いきなり手段を押すと **ステマ感・業者感がえぐい** ので、必ず上記順序で。

### 訴求ワード

- 「誰でもできる」
- 「簡単すぎる」
- 「リスクゼロ」
- 「〇〇するだけ」
- 「資金ゼロからいける」
- 「顔出しなしで稼げる」
- 「再現性がある」
- 「誰も知らない」

### 効果的な構成要素

- 自分がうまくいった体験談
- 教えた友人・生徒の事例
- 再現性・簡易性の押し出し
- 他の手段に比べて何が良いのか
- 成功までの具体的なロードマップ
- 最初にやるべき「はじめの一歩」

### ポイント①: 自分の商材の理解を深める

手段の教育のためには **自分の手段の魅力** と **競合になる手段** をちゃんと理解する。

理解すべき軸:
- 初期費用がかからない
- リスクが低い
- 初心者でも手が出しやすい
- 短期間で収益化できる
- ...etc

他の手段と比べた強みがわかるからこそ「そっちよりこっちがいいですね！」と言える。

### 連動: 問題点との一貫性

問題点の教育と手段の教育は **一貫させる**:

例（アパレル物販を売りたい場合）:
- 問題点: 「初心者は仕入れで金がかかる商材を選ぶな」「利益率が低い商材はクソ」
- 手段: 「資金10万円以下ならアパレル一択」「仕入れ値の数倍で売れるから利益率最強」

### 追加tips: やんわり他の手段を選択肢から消す

> 出典: 教材11（2026-04-27）

ただ自分の手段をおすすめするだけではなく、**他の手段を選択肢から消す** のも重要:

#### 例
```
「元手が1000万円以上ない人が株やったところでほぼ増えないので無駄」
「物販は仕入れリスクと低利益率で初心者には厳しい」
「動画編集はスキル習得に時間かかるし単価も低い」
```

→ 直接否定ではなく **「あなたには向いてない」「無駄」** という形で柔らかく消す。
→ 結果として「じゃあ何が良いの？」 → 自分の手段への流れが自然に作れる。

#### 詳細は

`incentive_design.md` の「他の選択肢を消すグダ消し」と本質的に同じ。

## ⑤ 行動の教育

> **今、行動する重要性を伝える**

読者は最適解がわかってもガチで行動しない。
ビジネスは **行動が早い読者がいないと面談に集まらない** → 商品が売れない。

### 例フレーズ

- 「フットワーク重いやつは稼げない」
- 「チャンスは即動くべき」
- 「今が〇〇はチャンスです！」
- 「これ逃すと〇〇万円の損失やね」

### 教育内容の柱

- 行動することの重要性
- 今すぐ動く理由づけ
- フットワークを軽くする教育
- 動くのが遅いと起こる地獄の未来（恐怖）

### ゴール

「行動するのが当たり前」「すぐ動くのが当たり前」「フッ軽しか勝たん」を **読者のデフォルト** にする。

### テクニック: 望む行動をした人を褒める

読者にやってもらいたい行動（ポストを遡る / 即反応 / 感想を書く / 学びをアウトプット）について、**その行動をした人を褒めて持ち上げる**:

> 「即アウトプットナイスです！ガチでアウトプット早い人は超伸び早いですねw」

これで他の読者も同じ行動を取りたくなる。

### 追加tips: コンサル生に深夜2時呼び出し事例

> 出典: 教材11（2026-04-27）

教材執筆者のコンサル生では、**深夜の2時にいきなり呼び出してタクシーで飛んできてもらう** こともある。

#### なぜそんなことが可能か

```
普段から「これくらいの行動力が当たり前」という基準値を刷り込んでいる
   ↓
コンサル生が「変なグダ」をしなくなる
   ↓
深夜呼び出しでも即対応する文化ができる
```

#### この事例の意味

「行動の教育」は単発のテクニックではなく、**普段からの基準値の刷り込み**。
- 「即動くのが当たり前」
- 「フッ軽が当たり前」
- 「迷う暇があったら動くのが当たり前」

→ これを継続的にポストすることで、コミュニティ全体の **当たり前の基準** が変わる。

## ⑥ 投資の教育

> **自己投資が当たり前という基準値を作る**

お客さんはお金を使うことにビビる。
冷静に考えれば「稼いでる人にお金払って学んだ方が早い」のに、「ネットのスクールって怪しい」「お金使いたくない」マインドになりがち。

### 教育内容の柱

- 知識に投資するの当たり前だよね
- 知識や環境に投資してショートカットした方がいい
- 自分もお金払って学んだおかげででかいリターンを手に入れた
- 自分が環境や知識に投資してる様子

### 例フレーズ

- 「良い情報は無料で出回らない」
- 「まずは無料でとか言ってるやつは稼げない」
- 「金がないやつほど投資しろ」

### テクニック: レッテル貼り（= ラベリング）

> 出典: 教材4 + 教材11（2026-04-27 取り込み）
> 「レッテル貼り」（教材4）と「ラベリング」（教材11）は同じテクニック。

行動の促進に効く強力テクニック。

| 行動 | 貼るレッテル |
|---|---|
| ⭕ やってほしい行動 | 「伸びる人」「成功する行動」「センスある」などポジティブ |
| ❌ やってほしくない行動 | 「雑魚」「社畜」「負け組」「ゴミ」などネガティブ |

#### 使い方の例

- 「稼いでるやつは〜」
- 「稼げないやつは〜」
- 「ビビってる奴が稼ぐのは無理」
- 「稼いでるやつは全員フッ軽」
- 「数万円のコンテンツ購入すら迷ってるやつに金稼ぎは無理ゲー」

→ 読者は「自分は稼げる側でありたい」と感じて望む行動を取る。
→ 勝手にラベルを貼っておくだけで **読者の「当たり前の基準」を定義できる**。

#### 適用範囲

行動の教育以外でも使える普遍テクニック:
- 投資の教育 → 「自己投資する奴は稼ぐ、無料厨はずっと貧乏」
- 信用の教育 → 「信頼できる発信者を見抜ける人だけが伸びる」
- 問題点の教育 → 「自分の弱点を直視できる人が成長する」

### 追加tips: 子供の学費 vs 大人の自己投資

> 出典: 教材11（2026-04-27）

投資の教育で使える汎用ロジック例:

```
子供の学費であれば、予備校や大学に何百万円払うのに、
大人になって自分のスキルとか学びにお金を使う人ってほぼいませんよね。

ただ大人になってからの学びの方が100倍収入に直結するので、
これやっとくことで差がつきます。
```

→ パターン:
- ① 既に当たり前にお金を使っている分野（学費・医療・保険等）を引き合いに出す
- ② 「実は自己投資の方が効果がある」と論理的に展開
- ③ 自己投資への抵抗を薄める

## ⑦ 読む・見る教育（強化教育1）

> 出典: 教材12「6つの強化教育」（2026-04-27 取り込み）
> **そのコンテンツを見る動機づけ** を作る教育。

### 目的

教育において **1番最悪なのは、そもそも見られずにスルーされること**。
読者は日々大量の情報に触れているので、普通に配信するだけだと「流し見 or 飛ばされる」のが関の山。

→ 記事・動画・スペースで必ず「それを見るべき理由」を作って **最後まで見るように読者を拘束** する。

### 具体的な動機づけ要素

冒頭または事前予告で入れるパターン:

```
□ 記事・動画を読む/見るメリット
□ この記事/動画で何が学べるか（興味付き）
□ 「重要なのでメモを促す」
□ 後半・中盤で"何か"を話すと興味付け
□ 最後まで見たら特典があると訴求
□ 「すぐ消す」などの限定性
```

### スペース冒頭・事前予告の例

```
✅「これ録音残さないんで」
✅「メモ取っといてください」
✅「有益なことポロッというので」
✅「今日話す内容使ってもらえたら、こんなやばいことできます」
```

→ これだけで離脱率と熱量が **全然変わる**。

### 重要原則

- 人間は **自分にメリットがあるとわからないと集中して読まない**
- 「いかにその情報が重要なのか？」を伝えることは、本文の内容以上に大切
- セールスレター・アフィポストで言われる「ベネフィット」と同じことを、記事・スペース・コンテンツでもやる

## ⑧ 変化の教育（強化教育2）

> 出典: 教材12（2026-04-27）
> **「変化＝良いこと」** という思考を植え付け、変化への恐怖をポジティブ変換する教育。

### 前提: 人間は変化を恐れる

人間には生まれながらにして **「変化を恐れる思考」がインストール** されている。

#### ドリームキラー現象

自分が起業・自己投資・新しい挑戦を始めようとすると、家族や友達が:

- 「それやめた方がよくない？」
- 「お前には厳しいよ」
- 「センスがある人しか無理だよ」
- 「まず3年働いてからのほうがよくね？」

と足を引っ張る発言をしてくる。これは **「自分の周りが変わるのが怖い」** から起こる。

→ この変化への恐怖があると、読者は自己投資や新しい挑戦にブレーキがかかる。

### 教育の方向

```
□ 人生を変えるには変化しないといけない
□ 変化するのはいいことだ
□ 変化の恐怖を乗り越えないといけない
□ 成長するとき = 変化の恐怖を克服したとき
□ 成長したきゃ今までの環境を捨てろ
□ ビビったときこそGO
□ 逆にみんながOKということはやめとけ
□ ドリームキラーは無視してOK
```

### 特に効くコア教育: 「変化の恐怖 = 成長痛」

```
変化に伴う恐怖 = 成長痛
ビビる = 成長のチャンス
```

→ この教育を入れておくと、「高額な自己投資怖いな…」と読者がグダりそうになったときに、
**「いや、こういう時こそチャンスだな」** という思考に変換される → マネグダが激減。

### 物語で変化教育を入れる例

教材執筆者のメルマガで使われている事例:

```
「九州出身ですが、ナンパをはじめるときに地元の友達全員のLINEをブロックして、
そのまま上京しました。

ナンパをやると周りには確実に止められるし、ネガティブな言葉を言われます。
『そんなことしてる余裕あるのか』『早く大学に行って働け』『頭おかしくなったのか』

こういう言葉は完全に足枷しかない。
これをとことん排除するために、友達をブロックしたし、環境も大きく変えた。
だからこそ短期間でぶち抜くような成果を出せました。」
```

#### この物語が植え付ける価値観

```
古い人間関係をブロック = 成功者が普通にやること
ネガティブ言葉 = 足枷
環境を変えた → 成功できた
こういうぶっ飛んだ行動 = 成果を出す人の普通
```

→ 「感覚をバグらせる」のが目的。普通の感覚で生きてきて稼げない・モテない現状なのだから、何かを変えないと人生は変わらない。

### この教育の効果

変化の教育がバチっと決まると **強烈にファン化** できる。
情報発信するなら **必須レベルで重要**。

## ⑨ 素直の教育（強化教育3）

> 出典: 教材12（2026-04-27）
> **知識ある人から素直に学んで実践するのが成長の近道** という教育。

### 重要原則

```
素直な人ほど教育がかかりやすい
   = 素直な人ほど言われた通りすぐ行動する
   = 素直な人が爆速で成果を出す
```

→ 「教育がかかりやすい」はネガティブに聞こえるが、裏を返せば **「成果が早い」** ということ。

### 実例: スクール生

爆速で伸びる人は:
- 週末に学んだ内容を、その翌日からすぐに実践
- 細かいこと気にせずマニュアル通りにDM送りまくる
- 「考えて動けない人」より確実に成果を出す

### 素直の教育を入れる効果

```
素直の教育を入れる
   ↓
6つの教育やその他の教育要素が圧倒的にスッと入る
   ↓
読者の成長スピードが上がる
   ↓
コミュニティの成果実績が増える
```

→ Xポストで定期的に出していくのが推奨。

### 具体的手法

```
□ 「素直な人 = 成功する人」というラベリング
□ 素直に言われた通り行動して早く成功できた体験談
□ 逆に自己流でやって失敗した話
□ 実際に超素直に行動してる人をポジティブに紹介
```

### 教育のコアメッセージ

```
初心者はとにかく自己流やめて、
稼げてる人から素直に学べ。
```

→ これをひたすら入れていく。

## ⑩ アウトプットの教育（強化教育4）

> 出典: 教材12（2026-04-27）
> **アウトプットすること＝良いこと** という教育。

### 重要原則

教育において **「相手に何かしらアクションさせる」** のは非常に有効。

```
コメントさせる
いいねさせる
リプさせる
メールを読ませる
何でもいい
```

アクションするのを当たり前にさせておくことで、
**「面談に来る」「商品を購入する」** みたいな大きな行動のハードルも下がる。

### 1番ハードル低い行動 = アウトプット

その中でも **アウトプット** が1番ハードル低い行動なので、まずここから促す。

### 教育の方向

```
□ アウトプットしないと明日には99%忘れる
□ ほとんどの人がアウトプットしないからこそ、やれば上位10%になれる
□ ただ読んで満足してると来年も1ミリも成長せず同じことしてる
□ 熱量ある人がアウトプットしたら認知する（インセンティブも追加）
```

### 重要原則: お願いではなくロジック

```
❌ 「感想書いてください」と直接お願い
⭕ アウトプットすること = 読者にとってのメリット、というロジックを作る
```

#### マーケ目線の流れ

```
感想を書いてくれない
   ↓
口コミが広まらない
   ↓
商品が売れない、認知が広がらない
```

→ アウトプット教育は **事前に必須**。
→ 詳細は `incentive_design.md` の「感想記入のインセンティブ」と相互参照。

### 教育の汎用原則

```
お願いじゃなくてロジックで動かせる。
   ↓
これが教育の基本的な考え方。
   ↓
教育ができると、ネットで稼ぐ・営業・対人コミュ全部得意になる。
```

## ⑪ 基準値の教育（強化教育5）

> 出典: 教材12「6つの強化教育」（2026-04-27 取り込み）
> ⚠️ **この教育はガチでえぐい。悪用厳禁。**

### 目的

```
極端な行動を促して、読者の基準値をバグらせる教育
```

特に **超高単価販売（100万・200万円超え）** に必須のテクニック。
普通の基準値では、凡人は数百万円を払う決断ができない。

### 教育のコア構造

```
普通の基準値 = 凡人、負ける
勝つ人の基準値 = 異常でバグってる
   ↓
これを刷り込む
```

### 教育の方向

```
□ 深夜3時でも即呼び出したらくるのが、成功するやつの基準値
□ 稼ぎたければ最初は100万円借金してでも投資すべき
□ 自分は環境に毎月300万円投資してる
```

### 体験談ストーリー集（教材執筆者の実例）

特に **自分の体験談ストーリー** が効果的:

```
✅ 友達のLINE全員ブロックして貯金ゼロ円で上京、ナンパ師になった話
✅ 貯金0円のニート時代に、コンサル受けるために寝ずに1週間バイトで資金を捻出した話
✅ 親に800万円借金して自己投資した話
✅ ナンパ初心者で、ノウハウゼロで1日100人声かけして初月30即した話
✅ 毎日1万文字書くのは当然という話
```

→ 「え、その基準値やばくね…！？」「これが成果出すやつの普通なのか…！」と思わせる。

### 機能の流れ

```
基準値の教育で読者の「当たり前基準」が引き上げられる
   ↓
読者の自己評価が下がる
「え、1声かけもできない自分へぼくね…？」
   ↓
小さいグダ（5万円商材ためらう / 50万円自己投資ひよる 等）が消える
   ↓
高額商品が売れやすくなる
```

### 重要なポイント

```
「これくらいやって当然」
「当たり前にこれくらいやってる」
   ↓
こういう顔で言うこと
```

→ ドヤらず、淡々と「普通」のように語るのが最大効果。

### グダ消しとの関係

`incentive_design.md` の「グダ消し思考」と相互強化:
- グダ消しは **個別の不安を消す**
- 基準値の教育は **「自分の不安は普通じゃない」と感じさせる**

→ 両方使うと、マネグダが二重に消える。

## ⑫ 覚悟の教育（強化教育6）

> 出典: 教材12（2026-04-27）
> **覚悟を決めて行動してる様子を見せる** 教育。基準値の教育に近いが、感情的な側面が強い。

### 目的

マネグダを消し、強烈にファン化する。

### 具体的な見せ方（教材執筆者の例）

```
✅ 事業拡大のために毎月のように300〜1000万円使ってる様子
✅ 投資チャンスのために2000万円借りて資金を増やしている様子
✅ ビジネス拡大のために先輩経営者に1時間50万円・100万円という金額を払って話を聞いている様子
```

### 機能の流れ

```
覚悟を決めた行動を見せる
   ↓
読者「え、数万円ごときで躊躇してる自分ダセェ…」
   ↓
強烈なファン化（「俺もこんなふうになりてぇ！！」）
   ↓
高額商品の購入グダが消える
```

### 物語の力

魅力的なアニメ主人公は **必ず覚悟を見せるシーン** がある。
人は覚悟を決めた人物に強烈に惹かれる。

→ SNSでこれをやるのが覚悟の教育。

### コアメッセージ

```
ブチ抜いた成果がほしければ覚悟を決めろ
```

→ これを普段のポストに入れていく。

### 基準値の教育との違い

| 教育 | アプローチ |
|---|---|
| 基準値の教育 | 「これが普通」と淡々と語る（読者の基準値をバグらせる） |
| 覚悟の教育 | 「覚悟を決めてやってる」と感情的に見せる（読者を巻き込む） |

→ 両方使うと相乗効果。

---

## 教育のバランス

6つの教育は **どれか1つだけやればOKではない**。
全部をある程度バランスよく入れる。
ただし **アカウントのフェーズによって重点が変わる**。

### フェーズ別の重点

| フェーズ | 重点となる教育 | 目的 |
|---|---|---|
| 初期垢 | **目的 + 信用** | 欲求喚起 / 信用獲得 / 共感 |
| ローンチ前 | **問題点 + 手段** | 自分の売りたい手段への誘導 |
| ポジション確立後 | **行動 + 投資** | 読者の見込み度をさらに上げる |

### 注意: 大手をパクるな

伸びてる大手垢のポストを単純パクリしても、フェーズが違えばハマらない。
例: 大手の「行動しよう」「自己投資しよう」を初期垢が真似しても **ポジションがないから説得力ゼロ** で空振りする。

## 教育ポストの役割

### 教育ポストは伸びにくい

教育ポスト（特に行動・投資の教育）は **必ずしもインプが伸びない**。
だが、それで良い。

### 役割の本質

教育の役割は **読者の見込み度合いを上げる** こと。
- 自己投資の教育 → 商品が売れる土壌を作る
- 行動の教育 → 個別面談に集まる読者を作る
- 問題点の教育 → 解決策（=手段）への需要を作る

→ インプが伸びなくても **必須の役割** がある。

### 初期は別の運用が必要

初期フェーズは興味を持たれていないので、いきなり教育しても刺さらない。

```
まずは興味を持たせる（プレイヤー型運用 / 興味付け中心）
  ↓
教育が刺さるようになる
  ↓
読者の見込み度が上がる
```

→ 初期は **興味付け × プレイヤー型運用**、ポジション確立後に **教育 × 伸ばすテクニック** という運用設計が必要。

## グダ消し思考

### 商品を売る基本原則

> **「買わない理由（グダ）」が消えれば商品は売れる。**

「人間は得より損に敏感」というのはガチで、人間は **買わない理由がちょっとでもあれば** かなり欲しくても買わない決断をする。

### グダ要素の例（汎用）

- 「顔出しが嫌だ」
- 「初期費用かかるのが怖い」
- 「ちょっとめんどくさそう」
- 「信用が傷つくかもしれない」
- 「グレーな感じがする」
- 「自分には才能がない」
- 「時間がない」
- 「家族に反対されそう」

### グダ消しのプロセス

1. **想定するグダを30個以上洗い出す**
   - 自分の商品を売る時、読者が抱きそうな不安・疑念・障害
2. **特に重要なグダ（5〜10個）を特定**
   - 太字でマーキング
3. **各グダを先回りで消すポストを作る**
   - 不安 → ポストで「実は逆」と論破
4. **ローンチ前から継続的に投下**
   - グダ消しは1ポストでは消えない。何度も角度を変えて消す。

### グダ消しは「他の選択肢を消す」も含む

お客さんは常に **あなたの手法と「別の手法」** を天秤にかけている。
もっと楽そうな手法があれば、そっちに流れていく。

ビジ系の例:
- YouTubeが稼げるらしい
- インスタも儲かってるらしい
- 物販は再現性が高いって聞きました
- 株や仮想通貨はどうなの？？
- プログラミングとかは？

→ 先回りして他の選択肢を潰す:
- 「YouTubeやインスタは時間がかかる」
- 「投資はデカい資金がないと無理」
- 「在庫がないビジネスが最強」
- 「労働の切り売りはやめよう」

### 重要な思考

運用が下手な人は **プラス要素ばかり** 考えている:
- 「何を言ったら欲求が喚起されるか？」
- 「相手が行動したくなるか？」

しかし実際は **マイナス要素（グダ）が残ってることで売れない** パターンが多い。
この思考はみんな持っていないので **差がつくポイント**。

### お客さんの旅イメージ

```
お客さんがゴール（=商品購入）に向かって走っている
  ↓
途中にたくさんの障害（グダ）がある
  - 「1日1時間作業しんどい」の障壁
  - 「時間かかることね？」の落とし穴
  - 「物販で稼ぐ」という別ルート（誘惑）
  - 「本当に俺でもいけるの？」という崖
  ↓
グダを放置すると、お客さんは障害にぶち当たって死んでいく
  ↓
個別面談に到達する前に消える、もしくは到達しても成約率が落ちる
```

→ **障害を事前に取り除く** ためにポストする。

### 商品が決まっていない場合

何も売る予定がない初期フェーズでも、**将来売る商品を想定して** グダ消しは仕込んでおくと、いざローンチした時に効果が出る。

## このフレームワークの応用範囲

6つの教育とグダ消しは **ポストだけでなく**:

- 教育記事
- ウェビナー
- セールスの場面（個別面談、LP、メール、LINE）

すべてで使う **モノを売るための基礎**。
ポスト作成にとどまらず、**情報発信全体の設計図** として使う。
