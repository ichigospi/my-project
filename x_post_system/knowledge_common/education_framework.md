## このフレームワークの位置づけ

ポストにおいては **どんな文章でポストするのか（ライティング）** より、
**何をポストするのか（ネタ）** の方がずっと重要。

ネタを思いつくための **2大思考フレーム**:

1. **6つの教育** — どの教育目的のポストかを意識する
2. **グダ消し** — 買わない理由を先回りで消す

> 厳密にはグダ消しも「6つの教育」のうち手段の教育の一部だが、特に重要なので独立して扱う。

ポストとは **極論ほぼすべて教育**。教育を満遍なく入れることで:
- 「商品を買う読者」
- 「マインドが高く行動力がある読者」
- 「自分の発信に共感してくれる読者」

を作っていくのが情報発信の本質。

## 教育とは何か（定義）

> 出典: 教材10「教育とは何か」（2026-04-27 取り込み）

### 教育の定義

```
相手の価値観や意思決定を変えて、
望むような行動を「自発的に」させること
```

**「自発的に」がポイント**。
- 説得・説教＝「勉強しなさい！」と直接命令
- 教育＝「俺は子供の頃に勉強しなかったせいで苦労した。でも勉強するようになって人生変わった」と話して、自分から机に向かわせる

### なぜ重要か

> 教育ができる ＝ 人を自由自在に動かせる

- 教育ができれば、少ないフォロワーでも商品が馬鹿みたいに売れる
- インプを増やすことも、フォロワーを増やすことも容易になる
- 押し売り（「買ってください」）なしに、お客さんから「買わせてください」と言われる状態を作れる

### 教育の力の証明

教材執筆者の経験（靴磨き / ナンパ / ビジネスで数百人と接客）から:

> 「人間は一定の情報を与えると、何かに突き動かされるかのように、喜んでお金を差し出したり、自分が望む行動をとってくれる」

→ 業種・ジャンルを問わず、教育の原理は同じ。

### 集客 vs 教育

```
集客の上手さ < 教育の上手さ
```

- 集客が得意に見えるのは、実は **教育で人が動くから集客が成立する** だけ
- 教育ができなければ、どれだけ集客しても商品は売れない

## 価値観は「教育の結果」である

> 出典: 教材10（2026-04-27）

人の価値観は、実は **誰かに植え付けられたもの**。

### 例: Xマネタイズに興味を持つまでの価値観遷移

```
会社員しんどい... 手取り少なくて辛い...
        ↓
なんか副業しようかな？
        ↓
物販は仕入れ費が… 動画編集はスキルが…
        ↓
Xマネタイズで稼げるらしい！コストもかからないし青天井！
        ↓
稼ぎたかったら知識つけて自己投資しないとね
        ↓
よし、スクールで学んでみよう！
```

これらの価値観すべて:
- 「会社員＝しんどい」印象
- 「副業しないとやばい」危機感
- 「自己投資しないと稼げない」マインド
- 「行動を起こさないといけない」必要性

→ **誰かの教育の影響** で形成されている。
→ これを **狙って起こせるようになる** のが「教育ができる」状態。

## まだまだ客 → 今すぐ客（= 絶望客 → 理想客）

> 出典: 教材10「教育とは何か」 + 教材11「価値観教育」（2026-04-27 取り込み）

見込み客は2種類に分けられる。教材11では同じ概念を **絶望客 / 理想客** とも呼ぶ。

| まだまだ客 = 絶望客（買えない） | 今すぐ客 = 理想客（買える） |
|---|---|
| 商品購入から遠い | 商品をすぐ買える状態 |
| 「ちょっと怪しいな」 | 「この人から買いたい」 |
| 「無料で済ませたい」 | 「自己投資が当たり前」 |
| 「何からやればいいか分からない」 | 「すぐ始めたい」 |

### 絶望客の詳細価値観リスト（11項目）

> 出典: 教材11（2026-04-27）

絶望客が持っている価値観の典型:

```
□ 手法や商品に興味がない
□ あなたのことを信用してない
□ 現状のままでいいや…と思ってる
□ リスクを取りたくない
□ お金を払いたくない（無料でやりたい）
□ 努力したくない
□ 勉強するのはめんどくさい
□ 自分には目標達成できないと思ってる
□ 手法に再現性を感じない
□ 誰かに依存して稼げるようにしてほしい
□ 行動したくない
```

### 理想客の詳細価値観リスト（10項目）

> 出典: 教材11（2026-04-27）

理想客が持っている価値観の典型:

```
□ 〇〇さんみたいになりたい！
□ 〇〇さんみたいな生活がしたい！
□ 〇〇さんの発信は信頼できる！
□ 自分の問題点に気づいてる
□ 問題を解決する手段もわかっている
□ あなたの手段は再現性があって自分に最適だと思ってる
□ お金を使って知識や時間を買うのは重要！
□ ある程度のリスクを取るのも重要！
□ ビジネスは自責で行動するのが大事！
□ チャンスを掴むには言われたことに素直に即行動するのが大事！
```

### 絶望客に売る危険性

> 出典: 教材11（2026-04-27）

絶望客のまま売ろうとするのは危険:

- ❌ **セールス難易度が激上がり**（成約率も成約単価も大きく下がる）
- ❌ 仮に売れても **行動しないので成果が出ない**
- ❌ リスクが取れない人なので **成果を出させるのが大変**
- ❌ **他責な人が多い** → 炎上やクレームに繋がりやすい

→ 教育で絶望客を理想客に変えてからセールスするのが鉄則。
→ 教育を経ずに面談・セールスに進むのはNG。

### 教育のゴール

```
普段の発信 = 絶望客 → 理想客 への変換工場
```

ローンチで爆売れする人は、**普段から教育で「理想客」を増やしている**。
- 新商品を出した瞬間に即売れる
- ゲリラスペース告知でも数百人が集まる
- 教育を経ない面談・セールスは成約率が激落ちする

### スクールのマインド講座も教育

このスクールが入会直後に **マインド講座を30講義** 見せるのも、
「成果を出しやすい価値観」に全員を変化させるための **教育**。

→ ノウハウ前にマインド整備 = 価値観操作の典型例。

## 教育マスターの2ステップ

> 出典: 教材10（2026-04-27）

教育を体系化すると2ステップ:

```
①価値観操作   — まだまだ客 → 今すぐ客 に根本転換
   ↓
②ロジック構築 — 商品が欲しくなる思考回路を作る
```

### ステップ1: 価値観操作

「商品を買いやすい価値観」へ根本から変える。

例（ビジネス系の価値観操作）:
- 「自己投資して知識つけるのは、稼ぐために大事」
- 「Xマネタイズはリスク低くて自分にもできそう」
- 「学ぶならこの人から学ぶのが一択」

→ 個別商品ではなく **マインド・世界観のレベル** での書き換え。

### ステップ2: ロジック構築

特定商品が欲しくなる **思考回路** を組む。

#### 例: マッチングアプリ写真撮影サービス

> 出典: 教材10で詳述された具体例

##### まだまだ客の典型反応
- 「写真って自分で撮っちゃダメなの？」
- 「友達にとってもらえばよくね？」

##### ロジック構築教育の例文
```
マッチングアプリで全然彼女できません…っていう人多くて
みんな「コミュニケーション」「デートコース」を勉強するんですけど
大体の場合「そもそもマッチしない」でつまづいてるんですよね。

で、マッチしない原因をテストしたら、
プロフィール文とかはほぼ関係なくて
「プロフ画像の1枚目」これで決まってる。

プロフ画像を「イケメン風」にしておけばビックリするくらいマッチします。

「俺イケメンじゃないから無理」って思うかもですが、
身だしなみだけパターン通り整えて雰囲気イケメンにすればOK。
ちょいブスでも美女とデート行きまくってる人いますよ。
```

##### このロジックで読者の頭が形成される

```
アプリで遊ぶにはマッチが大事
   ↓
マッチするには画像が大事
   ↓
画像は雰囲気イケメンであればOK
```

→ この状態で「一緒に服買いに行ってイケメン風の画像撮りましょか？」と提案すると **オファーが通る確率が上がる**。

#### ロジック構築の汎用パターン

```
①「みんな〇〇で困ってる」
   ↓
②「実は原因はY」
   ↓
③「Yを解決すれば全部解決」
   ↓
④「Yを解決する方法を提示（自社商品）」
```

### 効果の証明

教材執筆者の実測値:

> 「ロジック形成をした読者 vs ロジック形成をしてない読者にセールスかけたら、**売上が2倍以上変わった**」

→ ロジック構築の威力は数値として実証されている。

## 教材10と教材4（6つの教育）の対応マップ

教材10の2ステップは、教材4の6種類を再分類したもの:

| 教材10（メタ・2ステップ） | 教材4（実装・6種類） |
|---|---|
| **価値観操作** | 目的の教育 / 信用の教育 / 行動の教育 / 投資の教育 |
| **ロジック構築** | 問題点の教育 / 手段の教育 |

### 使い分け

- **価値観操作**: 普段の運用で時間をかけて、世界観・マインドレベルで書き換える
- **ロジック構築**: ローンチ前後で集中的に、特定商品への思考回路を作る

→ 両方を並行して回す。

## 12の教育要素フレームワーク

> 出典: 教材11「価値観教育」（2026-04-27 取り込み）

絶望客を理想客に変えるためのメッセージは、パターン化できる。
そのパターンが **12の教育要素**:

```
12の教育要素 = 前半6つ（基礎）+ 後半6つ（強化）

前半6つ: 6つの教育（基礎）
  ① 目的の教育
  ② 信用の教育
  ③ 問題点の教育
  ④ 手段の教育
  ⑤ 投資の教育
  ⑥ 行動の教育

後半6つ: 6つの強化教育（ドーピング剤）← 教材12（次教材）で詳述予定
  ⑦〜⑫: 未取り込み
```

### 前半 vs 後半の役割

| 種別 | 役割 | 取り込み状況 |
|---|---|---|
| **前半6つ** | 教育の基礎骨格。SNSで稼ぐ全員が知るべき内容。 | ✅ 教材4・教材11で詳述 |
| **後半6つ（強化教育）** | 6つの教育の効きを **何倍にも強化** するドーピング剤。普段のポストや記事・動画に散りばめる。 | ⏳ 次教材で詳述予定（プレースホルダー） |

→ 後半6つの取り込み後、ここを正式版に更新する。

## 6つの教育 — 全体像

ネットビジネス業界で使われている定番フレームワーク。

```
1. 目的の教育   — 「〇〇したい」という欲求を抱かせる
2. 信用の教育   — 読者に信用・共感してもらう
3. 問題点の教育 — うまくいってない原因と放置の危険を伝える
4. 手段の教育   — おすすめの手段とその魅力を教える
5. 行動の教育   — 今すぐ動く重要性を伝える
6. 投資の教育   — 自己投資が当たり前という基準値を作る
```

### 標準的な誘導フロー

```
【目的の教育】
  俺こんな良い状態だよ、いいだろ？
  このやり方知りたいだろ？
        ↓
【信用の教育】
  でも昔はこんなダメでした。
  紆余曲折あって変わりました。あなたにもできます。
        ↓
【問題点の教育】
  あなたがうまくいかない原因はこれです。
  これを放置するとヤバいです。
        ↓
【手段の教育】
  それを解決するにはこの手段がおすすめ。
  再現性もある、誰でもできる、リスクもない。
        ↓
【投資の教育】
  解決するには環境や知識に投資していきましょう。
  僕も自己投資してきたからこそ手に入った。
        ↓
【行動の教育】
  あとは行動しましょう。
  動かない人生は変わりません。今すぐ動きましょう。
```

これを **順番かつ満遍なく** ポストすれば、読者の見込み度合いが徐々に上がる。

## ① 目的の教育

> **「〇〇したい」という欲求を抱かせる教育**

理想の未来や過去からの変化（Before/After）を見せて、欲求喚起する。

### 訴求軸

- お金面の理想の状態
- 時間・環境・ライフスタイルの理想
- 性についての理想（モテ・恋愛）
- 優越感・承認欲求的な理想

### 例フレーズ

- 「満員電車に乗らなくて良くなった」
- 「ウザイ上司より稼いでる優越感」
- 「童貞だったのにセフレに困らなくなった」
- 「月1で旅行に行ける」
- 「毎日のように10万円以上振り込まれる」
- 「ほぼ自動化したので暇すぎる…w」
- 「今日も日給10万円 チャリーン！」

### 3つのポイント

#### ポイント①: 切り口をストックする

ジャンルによって目的の教育の切り口は無限にある。
競合のポストを見て **感情が動いた事例をストック** する。

例:
- フォロワーが伸びる
- インプが増える
- 日給〇〇万円
- 旅しながら稼げる
- 寝ててもお金が入ってくる
- 会社辞めてウザい上司からおさらば
- モテるようになった
- 利益率が高すぎる

#### ポイント②: イメージしやすくして欲求を掻き立てる

目的の教育は **視覚的に見せる方が欲求が動きやすい**。
- 売上画像
- 何か魅力的なことをしてる画像

#### ポイント③: リアルタイム性を出す

| 弱い | 強い |
|---|---|
| 「アフィで10万円稼いだ！」 | 「鮨食ってたら10万円稼げちゃってて黒字！」 |

リアル感があると「なんかいいな」と思われる。

### なぜ重要か

人間は **「こいつ羨ましいな」「なんでこいつだけ良い思いしてんだ」「こいつのやり方知りたいな」** と思うからこそ発信を追ってくれる。

### 追加tips: 小実績でも見せ方で「羨ましい」を作る

> 出典: 教材11（2026-04-27）

「デカい実績がないと目的の教育できない」と思い込んでる人が多いが、**それは半分嘘**。

#### 重要原則
- 情報発信は読者との **「差分」** が価値になる
- 何かの実績や理想状態は必要だが、**デカい実績じゃなくていい**
- 単発1万円の実績でも見せ方次第で「羨ましい状態」を作れる

#### 例: 単発1万円実績の魅せ方

❌ そのまま伝える:
> 「1万円稼ぎました」（インパクト弱）

⭕ 表現を工夫:
> 「寝てたら1万円入ってたわ、こういう収入源いくつか持ってるとめちゃ最高だよねw」
> 「スマホちょっと触ってたら1万円儲かっちゃったわ、これエグすぎるっしょw」

→ 「たくさんの収入源がある」「労力がかからない」と **錯覚させる**。

#### 見せ方の3要素

```
① 簡単そうに見せる
② 労力低そうに見せる
③ 「当たり前」そうに見せる
```

→ 小実績でも誰でも「羨ましい状態」を作れる。

## ② 信用の教育

> **読者に信用・共感してもらう**

目的の教育で欲求を喚起しても、次に「こいつ怪しいな」「詐欺じゃね？」となる。
そこで信用を得る発信が必要。

### 信用構築の要素

- 過去のダメダメさを語る
- サクセスストーリーを語る
- 失敗談を語って共感させる
- 発信の思い・理念を語る
- 証拠を見せる
- 成果を見せつける

### 例フレーズ

- 「自分も2年前までサラリーマンで毎日10時間残業して死にかけてた」
- 「スキルゼロだったし金なかったから、とりあえず始めれるXをやった」
- 「Xはじめて最初に出した商品は700円しか売れなかった」
- 「過去の自分みたいな人を救いたいから全員俺の発信を見ておけ」

### 定番ストーリーパターン

```
今はこんなに稼いでます！
  ↓
でも昔はこんなダメダメでした！
  ↓
でもこの手法に出会って紆余曲折あって今ではこんな生活を手に入れました！
  ↓
同じ人を救いたいと思って発信しています！
```

### ポイント

- どれだけロジックが通っていても **信じてもらえなければ無意味**
- 顔出しも信用の教育になる（信憑性を高める）
- 弱みを見せる / 過去を語る / 理念を語る → どんどんファン化が進む

### 追加tips: 「こいつ俺と同じだったのか！」を作る

> 出典: 教材11（2026-04-27）

信用の教育で1番有効なのは:

```
読者「こいつ、俺と同じだったのか！」
   ↓
「俺と同じレベルだったのに、今はこんなに稼いでるのか…」
「数年前まではただのサラリーマンだったのか…」
   ↓
急に親近感が湧く → 信用が一気に積み上がる
```

#### 重要原則: 共通点を作る

- 読者との **共通点** を意図的に作る
- 「お前と同じ状態から成り上がったよ！」を伝える
- 共通点を作るために **ストーリーや過去話** を使う

#### 共通点の作り方の例

- 「俺も2年前まで会社員で…」
- 「最初は1日3時間しか副業できなかった」
- 「初月の収益は700円だった」
- 「家族に反対されながらやってた」

→ 読者が「あ、これ俺と同じ」と感じる接点を1つでも作る。

## ③ 問題点の教育

> **相手がうまくいってない原因を伝えて、放置の危険を煽る**

人間は **「なんでうまくいってないのか（原因）」** を知ると解決策を知りたくなる。

### 基本パターン

「あなたが（うまくいかない）のは（原因）だからです」
→ 読者「じゃあ、どうやんの？？」
→ 「だからこそ（自分の推す手段）が良い」

### 例フレーズ

- 「『ポスト』ばっか頑張ってもXは伸びません」
- 「初心者が『低単価』で戦うと詰みます」
- 「マチアプでマッチしないのはほぼ『画像』が原因ですねw」
- 「物販で稼げない人は『利益率低すぎ』w」

### 放置する恐怖を煽る

原因が分かっても「まぁ今じゃなくてもいいよね」と思考停止する人が多い。
そこで **放置した時の危険性** を煽って問題意識を増幅する。

### 恐怖煽りの例

- 「一生会社員のまま働き続けて、スキルもつかず40代でリストラされたら…」
- 「人生の大半を労働して生きることになるし、結婚すら厳しいかもしれない…」
- 「太ったままだと30代後半を迎えると、大きな病気のリスクが高まる」
- 「一生非モテのまま生きて、金があればモテると思って40代になり、一切モテないことに気づくも時すでに遅し」
- 「商品ないままX動かしたところで、穴の空いたバケツに水を注いでる状態」

### 2つのポイント

#### ポイント①: 初心者がやりがちなダメ行動を書き出す

稼げない人・モテない人を想像（または観察）して、彼らのダメ行動を **30個書き出す**。

例（ビジ系）:
- 一生ノウハウ集めてる
- いろんな情報に手出ししてる
- インプ取るための方法を知らない
- 導線ができてない
- ...etc

その上で:
- 「（ダメな行動）してる奴は稼げない」
- 「（ダメな行動）は今すぐやめろ」
- 「（ダメな行動）しちゃってる人はマジで危険」
- 「（反対のこと）だけやっときゃOK」

の切り口で理由・具体例を掘り下げれば問題点ポストの完成。

#### ポイント②: 仮想敵を作って責任をなすりつける

「お前が稼げないのはセンスないからだよ」と読者を直接攻撃すると離れる。
代わりに **仮想敵** を作って責任をなすりつける:

- 界隈の風潮・常識
- 大手垢の発信内容
- 社会の常識やシステム
- 別の流派・別の手法

例: 「noteで稼ごうみたいな流れ」「X運用下手な人がXを教えてる現状」

### 追加tips: ダイエット例（汎用テンプレ）

> 出典: 教材11（2026-04-27）

問題点の教育の汎用テンプレ。**「原因 → 放置の恐怖」** がきれいに繋がる例文:

```
あなたが痩せれないのは体質の問題なので、
体質に合ったダイエットをしない限り一生痩せれませんよ？

それどころか合わない方法やっちゃうと
逆にストレスで太ったり体調崩したりするので
まず体質を把握しないと結構ヤバいです。
```

→ この型は **どのジャンルにも応用できる**:
- 「あなたが稼げないのは〇〇が原因なので、これを解決しないと一生稼げません」
- 「あなたが結婚できないのは〇〇が原因なので、これを直さないと一生独身です」
- 「あなたの運気が上がらないのは〇〇が原因なので、これを変えないと一生上向きません」

→ パターン: **「原因の指摘 + 放置時の悪化」** をセットで提示。

## ④ 手段の教育

> **おすすめの手段とその魅力を教える**

自分が売りたい商品と一貫する手段を押し出す。

### 順序の重要性

```
目的の教育 → 信用の教育 → 問題点の教育 → 手段の教育
```

いきなり手段を押すと **ステマ感・業者感がえぐい** ので、必ず上記順序で。

### 訴求ワード

- 「誰でもできる」
- 「簡単すぎる」
- 「リスクゼロ」
- 「〇〇するだけ」
- 「資金ゼロからいける」
- 「顔出しなしで稼げる」
- 「再現性がある」
- 「誰も知らない」

### 効果的な構成要素

- 自分がうまくいった体験談
- 教えた友人・生徒の事例
- 再現性・簡易性の押し出し
- 他の手段に比べて何が良いのか
- 成功までの具体的なロードマップ
- 最初にやるべき「はじめの一歩」

### ポイント①: 自分の商材の理解を深める

手段の教育のためには **自分の手段の魅力** と **競合になる手段** をちゃんと理解する。

理解すべき軸:
- 初期費用がかからない
- リスクが低い
- 初心者でも手が出しやすい
- 短期間で収益化できる
- ...etc

他の手段と比べた強みがわかるからこそ「そっちよりこっちがいいですね！」と言える。

### 連動: 問題点との一貫性

問題点の教育と手段の教育は **一貫させる**:

例（アパレル物販を売りたい場合）:
- 問題点: 「初心者は仕入れで金がかかる商材を選ぶな」「利益率が低い商材はクソ」
- 手段: 「資金10万円以下ならアパレル一択」「仕入れ値の数倍で売れるから利益率最強」

### 追加tips: やんわり他の手段を選択肢から消す

> 出典: 教材11（2026-04-27）

ただ自分の手段をおすすめするだけではなく、**他の手段を選択肢から消す** のも重要:

#### 例
```
「元手が1000万円以上ない人が株やったところでほぼ増えないので無駄」
「物販は仕入れリスクと低利益率で初心者には厳しい」
「動画編集はスキル習得に時間かかるし単価も低い」
```

→ 直接否定ではなく **「あなたには向いてない」「無駄」** という形で柔らかく消す。
→ 結果として「じゃあ何が良いの？」 → 自分の手段への流れが自然に作れる。

#### 詳細は

`incentive_design.md` の「他の選択肢を消すグダ消し」と本質的に同じ。

## ⑤ 行動の教育

> **今、行動する重要性を伝える**

読者は最適解がわかってもガチで行動しない。
ビジネスは **行動が早い読者がいないと面談に集まらない** → 商品が売れない。

### 例フレーズ

- 「フットワーク重いやつは稼げない」
- 「チャンスは即動くべき」
- 「今が〇〇はチャンスです！」
- 「これ逃すと〇〇万円の損失やね」

### 教育内容の柱

- 行動することの重要性
- 今すぐ動く理由づけ
- フットワークを軽くする教育
- 動くのが遅いと起こる地獄の未来（恐怖）

### ゴール

「行動するのが当たり前」「すぐ動くのが当たり前」「フッ軽しか勝たん」を **読者のデフォルト** にする。

### テクニック: 望む行動をした人を褒める

読者にやってもらいたい行動（ポストを遡る / 即反応 / 感想を書く / 学びをアウトプット）について、**その行動をした人を褒めて持ち上げる**:

> 「即アウトプットナイスです！ガチでアウトプット早い人は超伸び早いですねw」

これで他の読者も同じ行動を取りたくなる。

### 追加tips: コンサル生に深夜2時呼び出し事例

> 出典: 教材11（2026-04-27）

教材執筆者のコンサル生では、**深夜の2時にいきなり呼び出してタクシーで飛んできてもらう** こともある。

#### なぜそんなことが可能か

```
普段から「これくらいの行動力が当たり前」という基準値を刷り込んでいる
   ↓
コンサル生が「変なグダ」をしなくなる
   ↓
深夜呼び出しでも即対応する文化ができる
```

#### この事例の意味

「行動の教育」は単発のテクニックではなく、**普段からの基準値の刷り込み**。
- 「即動くのが当たり前」
- 「フッ軽が当たり前」
- 「迷う暇があったら動くのが当たり前」

→ これを継続的にポストすることで、コミュニティ全体の **当たり前の基準** が変わる。

## ⑥ 投資の教育

> **自己投資が当たり前という基準値を作る**

お客さんはお金を使うことにビビる。
冷静に考えれば「稼いでる人にお金払って学んだ方が早い」のに、「ネットのスクールって怪しい」「お金使いたくない」マインドになりがち。

### 教育内容の柱

- 知識に投資するの当たり前だよね
- 知識や環境に投資してショートカットした方がいい
- 自分もお金払って学んだおかげででかいリターンを手に入れた
- 自分が環境や知識に投資してる様子

### 例フレーズ

- 「良い情報は無料で出回らない」
- 「まずは無料でとか言ってるやつは稼げない」
- 「金がないやつほど投資しろ」

### テクニック: レッテル貼り（= ラベリング）

> 出典: 教材4 + 教材11（2026-04-27 取り込み）
> 「レッテル貼り」（教材4）と「ラベリング」（教材11）は同じテクニック。

行動の促進に効く強力テクニック。

| 行動 | 貼るレッテル |
|---|---|
| ⭕ やってほしい行動 | 「伸びる人」「成功する行動」「センスある」などポジティブ |
| ❌ やってほしくない行動 | 「雑魚」「社畜」「負け組」「ゴミ」などネガティブ |

#### 使い方の例

- 「稼いでるやつは〜」
- 「稼げないやつは〜」
- 「ビビってる奴が稼ぐのは無理」
- 「稼いでるやつは全員フッ軽」
- 「数万円のコンテンツ購入すら迷ってるやつに金稼ぎは無理ゲー」

→ 読者は「自分は稼げる側でありたい」と感じて望む行動を取る。
→ 勝手にラベルを貼っておくだけで **読者の「当たり前の基準」を定義できる**。

#### 適用範囲

行動の教育以外でも使える普遍テクニック:
- 投資の教育 → 「自己投資する奴は稼ぐ、無料厨はずっと貧乏」
- 信用の教育 → 「信頼できる発信者を見抜ける人だけが伸びる」
- 問題点の教育 → 「自分の弱点を直視できる人が成長する」

### 追加tips: 子供の学費 vs 大人の自己投資

> 出典: 教材11（2026-04-27）

投資の教育で使える汎用ロジック例:

```
子供の学費であれば、予備校や大学に何百万円払うのに、
大人になって自分のスキルとか学びにお金を使う人ってほぼいませんよね。

ただ大人になってからの学びの方が100倍収入に直結するので、
これやっとくことで差がつきます。
```

→ パターン:
- ① 既に当たり前にお金を使っている分野（学費・医療・保険等）を引き合いに出す
- ② 「実は自己投資の方が効果がある」と論理的に展開
- ③ 自己投資への抵抗を薄める

## 教育のバランス

6つの教育は **どれか1つだけやればOKではない**。
全部をある程度バランスよく入れる。
ただし **アカウントのフェーズによって重点が変わる**。

### フェーズ別の重点

| フェーズ | 重点となる教育 | 目的 |
|---|---|---|
| 初期垢 | **目的 + 信用** | 欲求喚起 / 信用獲得 / 共感 |
| ローンチ前 | **問題点 + 手段** | 自分の売りたい手段への誘導 |
| ポジション確立後 | **行動 + 投資** | 読者の見込み度をさらに上げる |

### 注意: 大手をパクるな

伸びてる大手垢のポストを単純パクリしても、フェーズが違えばハマらない。
例: 大手の「行動しよう」「自己投資しよう」を初期垢が真似しても **ポジションがないから説得力ゼロ** で空振りする。

## 教育ポストの役割

### 教育ポストは伸びにくい

教育ポスト（特に行動・投資の教育）は **必ずしもインプが伸びない**。
だが、それで良い。

### 役割の本質

教育の役割は **読者の見込み度合いを上げる** こと。
- 自己投資の教育 → 商品が売れる土壌を作る
- 行動の教育 → 個別面談に集まる読者を作る
- 問題点の教育 → 解決策（=手段）への需要を作る

→ インプが伸びなくても **必須の役割** がある。

### 初期は別の運用が必要

初期フェーズは興味を持たれていないので、いきなり教育しても刺さらない。

```
まずは興味を持たせる（プレイヤー型運用 / 興味付け中心）
  ↓
教育が刺さるようになる
  ↓
読者の見込み度が上がる
```

→ 初期は **興味付け × プレイヤー型運用**、ポジション確立後に **教育 × 伸ばすテクニック** という運用設計が必要。

## グダ消し思考

### 商品を売る基本原則

> **「買わない理由（グダ）」が消えれば商品は売れる。**

「人間は得より損に敏感」というのはガチで、人間は **買わない理由がちょっとでもあれば** かなり欲しくても買わない決断をする。

### グダ要素の例（汎用）

- 「顔出しが嫌だ」
- 「初期費用かかるのが怖い」
- 「ちょっとめんどくさそう」
- 「信用が傷つくかもしれない」
- 「グレーな感じがする」
- 「自分には才能がない」
- 「時間がない」
- 「家族に反対されそう」

### グダ消しのプロセス

1. **想定するグダを30個以上洗い出す**
   - 自分の商品を売る時、読者が抱きそうな不安・疑念・障害
2. **特に重要なグダ（5〜10個）を特定**
   - 太字でマーキング
3. **各グダを先回りで消すポストを作る**
   - 不安 → ポストで「実は逆」と論破
4. **ローンチ前から継続的に投下**
   - グダ消しは1ポストでは消えない。何度も角度を変えて消す。

### グダ消しは「他の選択肢を消す」も含む

お客さんは常に **あなたの手法と「別の手法」** を天秤にかけている。
もっと楽そうな手法があれば、そっちに流れていく。

ビジ系の例:
- YouTubeが稼げるらしい
- インスタも儲かってるらしい
- 物販は再現性が高いって聞きました
- 株や仮想通貨はどうなの？？
- プログラミングとかは？

→ 先回りして他の選択肢を潰す:
- 「YouTubeやインスタは時間がかかる」
- 「投資はデカい資金がないと無理」
- 「在庫がないビジネスが最強」
- 「労働の切り売りはやめよう」

### 重要な思考

運用が下手な人は **プラス要素ばかり** 考えている:
- 「何を言ったら欲求が喚起されるか？」
- 「相手が行動したくなるか？」

しかし実際は **マイナス要素（グダ）が残ってることで売れない** パターンが多い。
この思考はみんな持っていないので **差がつくポイント**。

### お客さんの旅イメージ

```
お客さんがゴール（=商品購入）に向かって走っている
  ↓
途中にたくさんの障害（グダ）がある
  - 「1日1時間作業しんどい」の障壁
  - 「時間かかることね？」の落とし穴
  - 「物販で稼ぐ」という別ルート（誘惑）
  - 「本当に俺でもいけるの？」という崖
  ↓
グダを放置すると、お客さんは障害にぶち当たって死んでいく
  ↓
個別面談に到達する前に消える、もしくは到達しても成約率が落ちる
```

→ **障害を事前に取り除く** ためにポストする。

### 商品が決まっていない場合

何も売る予定がない初期フェーズでも、**将来売る商品を想定して** グダ消しは仕込んでおくと、いざローンチした時に効果が出る。

## このフレームワークの応用範囲

6つの教育とグダ消しは **ポストだけでなく**:

- 教育記事
- ウェビナー
- セールスの場面（個別面談、LP、メール、LINE）

すべてで使う **モノを売るための基礎**。
ポスト作成にとどまらず、**情報発信全体の設計図** として使う。
